## 1. TODO/FIXME 标记

### 查找命令

PowerShell 命令：
```powershell
# 查找所有 TODO
Select-String -Pattern "TODO" -Path backend/src,frontend/src -Recurse

# 查找所有 FIXME
Select-String -Pattern "FIXME" -Path backend/src,frontend/src -Recurse
```

### 标记分类

| 标记 | 含义 | 处理方式 |
|------|------|---------|
| **TODO** | 待实现功能 | 规划实现 |
| **FIXME** | 已知 Bug | 优先修复 |
| **XXX** | 警告 | 需要注意 |
| **HACK** | 临时方案 | 后续重构 |

## 2. 硬编码值清理

### 应该提取为常量的值

错误示例（硬编码数字）：
```typescript
const FREE_STORAGE_DAYS = 7
if (days > 3)
```

正确示例（使用常量）：
```typescript
const ALERT_THRESHOLD_DAYS = 3
if (days > ALERT_THRESHOLD_DAYS)

const DEFAULT_COUNTRY_CODE = 'GB'
```

### 硬编码颜色清理

错误示例：
```scss
background-color: #409EFF
.status-warning { color: #FFA500 }
```

正确示例：
```scss
background-color: $primary-color
.status-warning { color: $warning-color }
```

## 3. 性能优化技巧

### 缓存策略

Vue computed 缓存：
```typescript
const filteredContainers = computed(() => {
  return containers.value.filter(c => c.status === 'active')
})
```

Redis 缓存：
```typescript
const cacheKey = `container:${containerNumber}`
const cached = await redis.get(cacheKey)
if (cached) return JSON.parse(cached)
```

### 懒加载

路由懒加载：
```typescript
const ContainerDetail = () => import('@/views/ContainerDetail.vue')
```

组件懒加载：
```typescript
const HeavyChart = defineAsyncComponent(() => import('./HeavyChart.vue'))
```

### 防抖节流

搜索框防抖：
```typescript
const debouncedSearch = debounce((query) => {
  fetchContainers({ search: query })
}, 300)
```

按钮防抖：
```typescript
const handleClick = throttle(() => {
  submitForm()
}, 1000)
```

## 4. 错误处理模式

### Try-Catch 模式

```typescript
async function fetchContainer(id: string) {
  try {
    const container = await containerService.getById(id)
    return container
  } catch (error) {
    logger.error('获取货柜失败', { id, error })
    throw new BusinessException('货柜不存在')
  }
}
```

### 前端错误拦截

Axios 拦截器：
```typescript
axios.interceptors.response.use(
  response => response,
  error => {
    if (error.response?.status === 401) {
      router.push('/login')
    }
    ElMessage.error(error.message)
    return Promise.reject(error)
  }
)
```

## 5. 设计模式应用

### Adapter Pattern（适配器模式）

外部数据适配器接口：
```typescript
interface ExternalDataAdapter {
  fetchData(): Promise<RawData[]>
  parseData(raw: RawData): ParsedData
}

class FeiTuoAdapter implements ExternalDataAdapter {
  async fetchData() { /* ... */ }
  parseData(raw) { /* ... */ }
}
```

### Strategy Pattern（策略模式）

预警规则策略：
```typescript
interface AlertRule {
  condition: (container: Container) => boolean
  message: (container: Container) => string
}

const ALERT_RULES: AlertRule[] = [
  { condition: isImminent, message: msgImminent },
  { condition: isOverdue, message: msgOverdue }
]
```

### Singleton Pattern（单例模式）

数据库连接池单例：
```typescript
let pool: pg.Pool | null = null

export function getPool() {
  if (!pool) {
    pool = new Pool({ /* config */ })
  }
  return pool
}
```
