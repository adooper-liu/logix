# 组件规范

LogiX 前端组件开发规范。

## 组件结构

### 目录结构

```
components/
├── ContainerCard/
│   ├── ContainerCard.vue    # 主组件
│   ├── index.ts             # 导出
│   └── types.ts             # 类型定义
└── FilterBar/
    ├── FilterBar.vue
    ├── index.ts

```

| 目录 | 说明 |
|------|------|
| common/ | 通用基础组件 |

## 类型定义

### Props 定义

```typescript
// types.ts
export interface ContainerCardProps {
  container: Container
  showActions?: boolean
}

export interface FilterBarProps {
  modelValue: FilterParams
  loading?: boolean
}
```

### 事件定义

```typescript
// 方式一：函数重载
const emit = defineEmits<{
  (e: 'update:modelValue', value: FilterParams): void
  (e: 'refresh'): void
}>()

// 方式二：对象语法
const emit = defineEmits({
  'update:modelValue': (value: FilterParams) => true,
  refresh: () => true
})
```

## 响应式数据

### 使用 ref/reactive

```typescript
// 简单类型用 ref
const loading = ref(false)
const keyword = ref('')

// 对象用 reactive
const form = reactive({
  name: '',
  status: ''
})

// 复杂对象用 ref
const containers = ref<Container[]>([])
```

### 计算属性

```typescript
const filteredContainers = computed(() => {
  if (!props.search) return props.containers
  return props.containers.filter(c =>
    c.containerNumber.includes(props.search)
  )
})
```

## 生命周期

### 常用钩子

```typescript
// 组件挂载
onMounted(() => {
  fetchData()
})

// 监听属性变化
watch(() => props.container, (newVal) => {
  // 处理变化
})

// 组件卸载
onUnmounted(() => {
  cleanup()
```

## 最佳实践

### Props 默认值

```typescript
// 使用 withDefaults
const props = withDefaults(defineProps<Props>(), {
  showActions: true,
  data: () => []
})
```

### 事件透传

```vue
<!-- 父组件 -->
<ChildComponent
  v-model="search"
  @refresh="handleRefresh"
/>

<!-- 子组件 -->
<script setup>
const props = defineProps<{ modelValue: string }>()
const emit = defineEmits(['update:modelValue', 'refresh'])

function onInput(value: string) {
  emit('update:modelValue', value)
}
</script>
```

### 加载状态

```typescript
const loading = ref(false)

async function fetchData() {
  loading.value = true
  try {
    // 获取数据
  } catch (error) {
    ElMessage.error('加载失败')
  } finally {
    loading.value = false
  }
}
```

### 错误处理

```typescript
try {
  await containerService.delete(id)
  ElMessage.success('删除成功')
} catch (error: any) {
  ElMessage.error(error.message || '操作失败')
}
```
