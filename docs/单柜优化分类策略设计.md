## 一、核心洞察

### 1.1 用户建议

**问题**: 单柜优化是否也应该应用分类逻辑？

**建议**:

1. ✅ **判断自己属于哪个子集**（免费期内 vs 已超期）
2. ✅ **采用相同策略**：
   - 已超期：尽量往前排（日期越早越好）

3. ✅ **资源预留**：免费期内的要让出能力给超期的
4. ✅ **跳过机制**：没有能力时保持原日期

### K - Keep It Logical（逻辑清晰）

✅ **遵循**:

- 单柜优化：判断分类 → 选择策略 → 搜索 → 优化
- 批量优化：分类 → 排序 → 逐个优化 → 统计

### I - Incremental（渐进式）

✅ **遵循**:

- 先修改单柜优化支持分类（已完成一部分）
- 再实现批量优化（进行中）
- 渐进式开发，每一步都可独立测试

### L - Leverage（复用现有）

✅ **遵循**:

- 单柜优化复用批量优化的分类逻辑
- 成本计算组件被所有场景复用
- 不重复造轮子

✅ **遵循**:

- 只计算必要的货柜

- 预计算公共数据（仓库能力）

#### 4.1.2 分类算法

```typescript
private categorizeSingleContainer(
  containerNumber: string,
  basePickupDate: Date,
  lastFreeDate?: Date
): ContainerCategory {
  // 1. 获取免费期
  const demurrageResult = await this.demurrageService.calculateForContainer(containerNumber);
  const effectiveLastFreeDate = lastFreeDate || demurrageResult.result?.items?.[0]?.lastFreeDate;

  // 2. 计算剩余天数
  const today = new Date();
  const remainingDays = dateTimeUtils.daysBetween(today, effectiveLastFreeDate);

  // 3. 分类
  const category: ContainerCategory = {
    containerNumber,
    category: remainingDays >= 0 ? 'within_free_period' : 'overdue',
    plannedPickupDate: basePickupDate,
    lastFreeDate: effectiveLastFreeDate,
    remainingDays,
    originalCost: 0,
    warehouseCode: '',
    truckingCompanyId: ''
  };

  return category;
}
```

### 4.2 优化策略选择

#### 4.2.1 策略配置

```typescript
interface OptimizationStrategy {
  searchDirection: "forward" | "backward" | "both"; // 搜索方向
  searchStartOffset: number; // 搜索起始偏移
  searchEndOffset: number; // 搜索结束偏移
  prioritizeZeroCost: boolean; // 是否优先零成本
  allowSkipIfNoCapacity: boolean; // 是否允许无能力时跳过
}

const STRATEGIES = {
  // 已超期：尽量往前排（日期越早越好）
  OVERDUE: {
    searchDirection: "forward", // 从今天开始往后找
    searchStartOffset: 0, // 今天
    searchEndOffset: 7, // 往后 7 天
    prioritizeZeroCost: false, // 已经不可能的，尽量减少损失
    allowSkipIfNoCapacity: false, // 超期的必须处理
  },

  // 免费期内：从免费日往前优化（日期越大越好）
  WITHIN_FREE_PERIOD: {
    searchDirection: "backward", // 从免费期截止日往前找
    searchStartOffset: 0, // 从免费期截止日开始
    searchEndOffset: -7, // 往前 7 天
    prioritizeZeroCost: true, // 必须成本为 0
    allowSkipIfNoCapacity: true, // 无能力时保持原日期
  },
};
```

---

#### 4.2.2 策略选择器

```typescript
private selectOptimizationStrategy(category: ContainerCategory): OptimizationStrategy {
  if (category.category === 'overdue') {
    return STRATEGIES.OVERDUE;
  } else {
    return STRATEGIES.WITHIN_FREE_PERIOD;
  }
}
```

### 4.3 智能搜索算法

#### 4.3.1 智能搜索范围

```typescript
private generateSearchRange(
  basePickupDate: Date,
  lastFreeDate: Date,
  strategy: OptimizationStrategy
): Date[] {
  const dates: Date[] = [];

  if (strategy.searchDirection === 'forward') {
    // 已超期：从今天开始往后找
    for (let offset = strategy.searchStartOffset; offset <= strategy.searchEndOffset; offset++) {
      const date = dateTimeUtils.addDays(new Date(), offset);
      if (date >= basePickupDate) { // 不能早于原计划
        dates.push(date);
      }
    }
  } else if (strategy.searchDirection === 'backward') {
    // 免费期内：从免费日往前找
    for (let offset = 0; offset >= strategy.searchEndOffset; offset--) {
      const date = dateTimeUtils.addDays(lastFreeDate, offset);
      if (date >= new Date()) { // 不能是过去
        dates.push(date);
      }
    }
  }

  return dates;
}
```

#### 4.3.2 带能力检查的搜索

```typescript
private async searchWithCapacityCheck(
  containerNumber: string,
  warehouse: Warehouse,
  truckingCompany: TruckingCompany,
  searchDates: Date[],
  strategy: OptimizationStrategy,
  warehouseCapacities?: Map<string, number> // 可选：仓库能力预计算
): Promise<{
  pickupDate: Date;
  strategy: 'Direct' | 'Drop off' | 'Expedited';
  totalCost: number;
  breakdown: CostBreakdown;
}[]> {
  const candidates = [];

  for (const candidateDate of searchDates) {
    // 跳过周末
    if (this.isWeekend(candidateDate) && await this.shouldSkipWeekends()) {
      continue;
    }

    // 检查仓库档期
    const availableCapacity = warehouseCapacities?.get(candidateDate.toISOString()) ??
                               await this.getWarehouseCapacity(warehouse, candidateDate);

    if (availableCapacity <= 0) {
      // ✅ 无能力时的处理
      if (strategy.allowSkipIfNoCapacity) {
        log.debug(`[CostOptimizer] No capacity on ${candidateDate.toISOString()}, skipping...`);
        continue; // 免费期内的可以跳过
      } else {
        log.warn(`[CostOptimizer] No capacity on ${candidateDate.toISOString()}, but must process...`);
        // 超期的必须处理，即使没有能力也要继续找
      }
    }

    // 为每个候选日期评估不同策略
    const strategies: Array<'Direct' | 'Drop off' | 'Expedited'> = [
      'Direct',
      ...(truckingCompany.hasYard ? ['Drop off'] : []),
      ...(candidateDate <= candidateDate ? ['Expedited'] : [])
    ];

    for (const strat of strategies) {
      const option: UnloadOption = {
        containerNumber,
        warehouse,
        plannedPickupDate: candidateDate,
        strategy: strat,
        truckingCompany,
        isWithinFreePeriod: candidateDate <= candidateDate
      };

      const breakdown = await this.evaluateTotalCost(option);

      // ✅ 优先零成本
      if (strategy.prioritizeZeroCost && breakdown.totalCost > 0) {
        continue; // 跳过非零成本的选项
      }

      candidates.push({
        pickupDate: candidateDate,
        strategy: strat,
        totalCost: breakdown.totalCost,
        breakdown
      });

      // ✅ 找到第一个满足条件的就停止（减少计算）
      if (strategy.prioritizeZeroCost && breakdown.totalCost === 0) {
        return candidates; // 返回最优解
      }
    }
  }

  return candidates;
}
```

### 4.4 主流程重构

#### 4.4.1 重构后的 `suggestOptimalUnloadDate`

```typescript
async suggestOptimalUnloadDate(
  containerNumber: string,
  warehouse: Warehouse,
  truckingCompany: TruckingCompany,
  basePickupDate: Date,
  lastFreeDate?: Date
): Promise<{
  suggestedPickupDate: Date;
  suggestedStrategy: 'Direct' | 'Drop off' | 'Expedited';
  originalCost: number;
  optimizedCost: number;
  savings: number;
  savingsPercent: number;
  alternatives: Array<{
    pickupDate: Date;
    strategy: 'Direct' | 'Drop off' | 'Expedited';
    totalCost: number;
    savings: number;
    breakdown: CostBreakdown;
  }>;
}> {
  try {
    log.info(`[CostOptimizer] Starting cost optimization for ${containerNumber}`);

    // 1. 获取货柜信息
    const container = await this.containerRepo.findOne({
      where: { containerNumber },
      relations: ['portOperations']
    });

    if (!container) {
      throw new Error(`Container ${containerNumber} not found`);
    }

    // 2. ✅ 新增：分类判断
    const category = this.categorizeSingleContainer(
      containerNumber,
      basePickupDate,
      lastFreeDate
    );

    log.info(
      `[CostOptimizer] Categorized ${containerNumber}: ${category.category} ` +
      `(remaining days: ${category.remainingDays})`
    );

    // 3. 获取免费期
    const effectiveLastFreeDate = category.lastFreeDate;

    // 4. 计算当前方案成本
    const currentOption: UnloadOption = {
      containerNumber,
      warehouse,
      plannedPickupDate: basePickupDate,
      strategy: truckingCompany.hasYard ? 'Drop off' : 'Direct',
      truckingCompany,
      isWithinFreePeriod: basePickupDate <= effectiveLastFreeDate
    };

    const currentBreakdown = await this.evaluateTotalCost(currentOption);
    const originalCost = currentBreakdown.totalCost;

    log.info(`[CostOptimizer] Current cost: $${originalCost.toFixed(2)}`);

    // 5. ✅ 新增：选择优化策略
    const optimizationStrategy = this.selectOptimizationStrategy(category);

    log.info(
      `[CostOptimizer] Using strategy: ${optimizationStrategy.searchDirection} ` +
      `(offset: ${optimizationStrategy.searchStartOffset} ~ ${optimizationStrategy.searchEndOffset})`
    );

    // 6. ✅ 新增：生成智能搜索范围
    const searchDates = this.generateSearchRange(
      basePickupDate,
      effectiveLastFreeDate,
      optimizationStrategy
    );

    log.info(`[CostOptimizer] Search range: ${searchDates.length} dates`);

    // 7. ✅ 新增：带能力检查的智能搜索
    const candidates = await this.searchWithCapacityCheck(
      containerNumber,
      warehouse,
      truckingCompany,
      searchDates,
      optimizationStrategy
    );

    log.info(`[CostOptimizer] Found ${candidates.length} candidates`);

    // 8. 找到成本最低的方案
    if (candidates.length === 0) {
      log.warn(`[CostOptimizer] No candidates found for ${containerNumber}`);

      // ✅ 如果没有候选方案，且允许跳过
      if (optimizationStrategy.allowSkipIfNoCapacity) {
        log.info(`[CostOptimizer] Skipping optimization, using original date`);
        return {
          suggestedPickupDate: basePickupDate,
          suggestedStrategy: truckingCompany.hasYard ? 'Drop off' : 'Direct',
          originalCost,
          optimizedCost: originalCost,
          savings: 0,
          savingsPercent: 0,
          alternatives: [
            {
              pickupDate: basePickupDate,
              strategy: truckingCompany.hasYard ? 'Drop off' : 'Direct',
              totalCost: originalCost,
              savings: 0,
              breakdown: currentBreakdown
            }
          ]
        };
      }

      // 不允许跳过，使用当前方案
      return {
        suggestedPickupDate: basePickupDate,
        suggestedStrategy: truckingCompany.hasYard ? 'Drop off' : 'Direct',
        originalCost,
        optimizedCost: originalCost,
        savings: 0,
        savingsPercent: 0,
        alternatives: [
          {
            pickupDate: basePickupDate,
            strategy: truckingCompany.hasYard ? 'Drop off' : 'Direct',
            totalCost: originalCost,
            savings: 0,
            breakdown: currentBreakdown
          }
        ]
      };
    }

    // 9. 选择最优方案
    const optimalCandidate = candidates.reduce((min, curr) =>
      curr.totalCost < min.totalCost ? curr : min
    );

    log.info(`[CostOptimizer] Found optimal:`);
    log.info(`  - Date: ${optimalCandidate.pickupDate.toISOString().split('T')[0]}`);
    log.info(`  - Strategy: ${optimalCandidate.strategy}`);
    log.info(`  - Cost: $${optimalCandidate.totalCost.toFixed(2)}`);

    const optimizedCost = optimalCandidate.totalCost;
    const savings = originalCost - optimizedCost;
    const savingsPercent = originalCost > 0 ? (savings / originalCost) * 100 : 0;

    // 10. 返回前 3 个最优方案
    const sortedCandidates = candidates
      .sort((a, b) => a.totalCost - b.totalCost)
      .slice(0, 3)
      .map((candidate) => ({
        pickupDate: candidate.pickupDate,
        strategy: candidate.strategy,
        totalCost: candidate.totalCost,
        savings: originalCost - candidate.totalCost,
        breakdown: candidate.breakdown
      }));

    return {
      suggestedPickupDate: optimalCandidate.pickupDate,
      suggestedStrategy: optimalCandidate.strategy,
      originalCost,
      optimizedCost,
      savings,
      savingsPercent,
      alternatives: sortedCandidates
    };
  } catch (error) {
    log.error(`[CostOptimizer] Failed to suggest optimal date:`, error);
    throw error;
  }
}
```
