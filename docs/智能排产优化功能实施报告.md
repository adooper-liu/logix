### ③ 费用明细树形结构

#### 改动内容：

- ✅ **根节点**：总费用
- ✅ **叶节点**：5 项分项费用（滞港费、滞箱费、港口存储费、运输费、堆场堆存费）
- ✅ **可折叠**：支持展开/收起
- ✅ **样式分级**：根据费用等级显示不同颜色

#### 实现代码：

**树形组件：**

```vue
<el-tree
  v-if="row.estimatedCosts && row.estimatedCosts.totalCost"
  :data="buildCostTree(row.estimatedCosts, row.plannedData?.warehouseCountry || 'US')"
  :props="costTreeProps"
  :expand-on-click-node="false"
  default-expand-all
  size="small"
  class="cost-tree"
>
  <template #default="{ node, data }">
    <span class="cost-tree-node">
      <span class="cost-tree-label">{{ node.label }}</span>
      <span 
        class="cost-tree-value" 
        :class="data.level === 0 ? 'cost-value-total' : getAmountClass(data.value)"
      >
        {{ formatCurrency(data.value, row.plannedData?.warehouseCountry || 'US') }}
      </span>
    </span>
  </template>
</el-tree>
```

**树构建方法：**

```typescript
const costTreeProps = {
  children: "children",
  label: "label",
  value: "value",
};

const buildCostTree = (costs: any, _country: string) => {
  const tree: any[] = [];
  const children: any[] = [];

  // 添加分项费用（叶节点）
  if (costs.demurrageCost) {
    children.push({
      label: "滞港费",
      value: costs.demurrageCost,
      level: 1,
    });
  }

  if (costs.detentionCost) {
    children.push({
      label: "滞箱费",
      value: costs.detentionCost,
      level: 1,
    });
  }

  // ... 其他费用

  // 添加总费用（根节点）
  tree.push({
    label: "总费用",
    value: costs.totalCost,
    level: 0,
    children: children,
  });

  return tree;
};
```

#### 树形结构示例：

```
▶ 总费用 $1,250.00
  ├─ 滞港费 $500.00
  ├─ 滞箱费 $300.00
  ├─ 港口存储费 $200.00
  ├─ 运输费 $150.00
  └─ 堆场堆存费 $100.00
```
