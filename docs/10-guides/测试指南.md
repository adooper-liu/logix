# LogiX 测试指南

> 掌握 LogiX 项目的测试编写与运行方法

## 测试技术栈

| 层级 | 工具 | 配置位置 |
|------|------|----------|
| 后端单元测试 | Jest + ts-jest | backend/jest.config.js |
| 前端单元测试 | Vitest | frontend/vitest.config.ts |
| 前端 E2E 测试 | Playwright | frontend/playwright.config.ts |

## 后端测试

### 运行测试

```bash
cd backend

# 运行所有测试
npm test

# 运行指定测试文件
npm test -- container.service.test.ts

# 监听模式（文件变化自动重跑）
npm test -- --watch
```

### 测试结构

```
backend/
├── src/
│   └── services/
│       └── __tests__/
│           ├── container.service.test.ts
│           └── demurrage.service.test.ts
└── jest.config.js
```

### 测试示例

```typescript
// backend/src/services/__tests__/container.service.test.ts
import { ContainerService } from '../container.service';
import { AppDataSource } from '../../database';

describe('ContainerService', () => {
  let service: ContainerService;

  beforeAll(async () => {
    // 初始化测试数据库连接
    await AppDataSource.initialize();
    service = new ContainerService();
  });

  afterAll(async () => {
    // 关闭数据库连接
    await AppDataSource.destroy();
  });

  describe('findAll', () => {
    it('should return container list', async () => {
      const result = await service.findAll();
      expect(Array.isArray(result)).toBe(true);
    });
  });

  describe('findOne', () => {
    it('should return container by id', async () => {
      const result = await service.findOne('TCNU1234567');
      expect(result).toBeDefined();
    });

    it('should return null for non-existent container', async () => {
      const result = await service.findOne('INVALID');
      expect(result).toBeNull();
    });
  });
});
```

```vue
<!-- frontend/src/components/__tests__/ContainerCard.test.vue -->
<script setup lang="ts">
import { mount } from '@vue/test-utils';
import ContainerCard from '../ContainerCard.vue';

describe('ContainerCard', () => {
  it('should render container info', () => {
    const wrapper = mount(ContainerCard, {
      props: {
        container: {
          containerNumber: 'TCNU1234567',
          logisticsStatus: 'in_transit',
        },
      },
    });

    expect(wrapper.text()).toContain('TCNU1234567');
  });

  it('should emit click event', async () => {
    const wrapper = mount(ContainerCard, {
      props: {
        container: { containerNumber: 'TCNU1234567' },
      },
    });

    await wrapper.trigger('click');
    expect(wrapper.emitted('click')).toBeTruthy();
  });
});
```

### Composables 测试

```typescript
// frontend/src/composables/__tests__/useContainerData.test.ts
```

### E2E 测试示例

```typescript
// frontend/e2e/shipments.spec.ts
import { test, expect } from '@playwright/test';

test.describe('货柜列表', () => {
  test.beforeEach(async ({ page }) => {
    await page.goto('http://localhost:5173/shipments');
  });

  test('should display container list', async ({ page }) => {
    await expect(page.locator('.container-table')).toBeVisible();
  });

  test('should filter by status', async ({ page }) => {
    await page.click('.filter-status');
    await page.click('.status-option[in_transit]');
    await expect(page.locator('.container-row').first()).toContainText('in_transit');
  });

  test('should navigate to detail', async ({ page }) => {
    await page.click('.container-row:first-child');
    await expect(page).toHaveURL(/.*\/shipments\/.*/);
  });
});
```

### 2. Arrange-Act-Assert 模式

```typescript
it('should calculate demurrage correctly', () => {
  // Arrange - 准备数据
  const container = { lastFreeDate: '2026-03-20' };
  const charge = { freeDays: 7, dailyRate: 50 };

});
```

### 常见问题

| 问题 | 解决方案 |
|------|----------|
| 测试超时 | 增加 timeout 或检查异步操作 |
| Mock 不生效 | 检查 mock 路径是否正确 |
| 数据库连接失败 | 确保测试数据库可用 |
| E2E 测试失败 | 检查服务是否启动 |
