# LogiX 项目结构速查

> 快速了解 LogiX 项目代码组织方式

## 整体目录结构

```
logix/
├── backend/                 # 后端服务（Express + TypeORM）
│   ├── src/
│   │   ├── entities/       # TypeORM 实体定义
│   │   ├── routes/         # API 路由
│   │   ├── controllers/    # 控制器
│   │   ├── services/       # 业务逻辑服务
│   │   ├── services/statistics/  # 统计服务
│   │   ├── middleware/     # 中间件
│   │   ├── constants/      # 常量定义
│   │   ├── utils/          # 工具函数
│   │   ├── database/       # 数据库初始化
│   │   └── server.ts       # 服务入口
│   ├── scripts/            # 脚本（迁移、诊断）
│   └── package.json
│
├── frontend/               # 前端服务（Vue 3 + TypeScript）
│   ├── src/
│   │   ├── api/            # API 调用封装
│   │   ├── assets/         # 静态资源
│   │   ├── components/     # 公共组件
│   │   ├── composables/    # 组合式函数
│   │   ├── constants/      # 常量定义
│   │   ├── router/         # 路由配置
│   │   ├── stores/         # Pinia 状态管理
│   │   ├── styles/         # 样式文件
│   │   ├── types/          # TypeScript 类型
│   │   ├── utils/          # 工具函数
│   │   ├── views/          # 页面组件
│   │   └── App.vue         # 根组件
│   ├── public/
│   │   └── docs/           # 文档（静态资源方式访问）
│   └── package.json
│
├── migrations/             # 数据库迁移脚本
├── scripts/                # 项目级脚本
├── logistics-path-system/ # 物流路径微服务
└── docker-compose.*.yml    # Docker 编排配置
```

## 数据库表结构

### 表前缀含义

| 前缀 | 含义 | 示例 |
|------|------|------|
| `dict_` | 字典表 | dict_ports, dict_countries |
| `biz_` | 业务表 | biz_containers, biz_replenishment_orders |
| `process_` | 流程表 | process_sea_freight, process_port_operations |
| `ext_` | 扩展表 | ext_container_status_events |

### 核心业务表

```
biz_replenishment_orders (备货单)
    │
    └── biz_containers (货柜)
            │
            ├── process_sea_freight (海运)
            ├── process_port_operations (港口操作)
            ├── process_trucking_transport (拖卡运输)
            ├── process_warehouse_operations (仓库操作)
            └── process_empty_return (还空箱)
```

### 物流状态流转

```
not_shipped → shipped → in_transit → at_port → picked_up → unloaded → returned_empty
```

## 后端代码组织

### 实体定义 (entities/)

```typescript
// backend/src/entities/
Container.ts              // 货柜
ReplenishmentOrder.ts    // 备货单
SeaFreight.ts            // 海运
PortOperation.ts         // 港口操作
TruckingTransport.ts     // 拖卡运输
WarehouseOperation.ts   // 仓库操作
EmptyReturn.ts           // 还空箱
```

### 服务层 (services/)

```typescript
// backend/src/services/
container.service.ts           // 货柜 CRUD
replenishment-order.service.ts // 备货单 CRUD
demurrage.service.ts          // 滞港费计算
statistics/
├── containerStatistics.service.ts     // 统计入口
├── ArrivalStatistics.service.ts       // 按到港统计
├── EtaStatistics.service.ts           // 按 ETA 统计
└── PlannedPickupStatistics.service.ts  // 按计划提柜统计
```

### 路由层 (routes/)

```
routes/
├── containers/           // 货柜路由
├── replenishment-orders/ // 备货单路由
├── dict-manage/          // 字典管理路由
└── external-data/         // 外部数据同步
```

## API 路由速查

前缀：`/api/v1`

### 货柜管理

```
GET    /containers              # 列表（分页+筛选）
GET    /containers/:id          # 详情
POST   /containers              # 创建
PATCH  /containers/:id          # 更新
DELETE /containers/:id          # 删除
```

### 统计相关

```
GET    /containers/statistics/arrival   # 按到港统计
GET    /containers/statistics/eta       # 按 ETA 统计
GET    /containers/statistics/planned   # 按计划提柜统计
GET    /containers/statistics/last-pickup # 按最晚提柜统计
GET    /containers/statistics/last-return # 按最晚还箱统计
```

### 备货单管理

```
GET    /replenishment-orders
GET    /replenishment-orders/:id
POST   /replenishment-orders
PATCH  /replenishment-orders/:id
DELETE /replenishment-orders/:id
```

### 字典管理

```
GET    /dict-manage/types           # 所有字典类型
GET    /dict-manage/:type           # 字典数据列表
POST   /dict-manage/:type           # 新增字典项
PUT    /dict-manage/:type/:id       # 更新字典项
DELETE /dict-manage/:type/:id       # 删除字典项
```

## 命名规则

| 层级 | 规则 | 示例 |
|------|------|------|
| 数据库表名 | snake_case + 前缀 | biz_containers |
| 数据库字段 | snake_case | container_number |
| 实体属性 | camelCase | containerNumber |
| 前端 API 请求 | snake_case | { container_number: '...' } |
| 前端组件 | PascalCase + .vue | ContainerDetail.vue |
| CSS 类名 | kebab-case | .container-card |
