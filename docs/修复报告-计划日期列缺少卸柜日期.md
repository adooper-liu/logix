# 修复报告 - 计划日期列缺少卸柜日期

## 🐛 问题描述

**用户反馈：** 计划日期列没有显示"卸柜"日期

**问题现象：**

```
计划日期列显示：
  ● 提柜：2026-03-28
  ● 送仓：2026-03-28
  ● 还箱：2026-03-29

缺少：
  ❌ 卸柜日期
```

---

## 🔍 原因分析

**代码检查发现：**

1. 计划日期列的模板代码仍然是旧版本
2. 只包含 3 个日期（提柜、送仓、还箱）
3. 缺少第 4 个日期（卸柜）

**旧代码：**

```vue
<el-table-column label="计划日期" min-width="200">
  <template #default="{ row }">
    <span v-if="row.plannedData">
      <span class="date-item">提柜：{{ row.plannedData.plannedPickupDate || '-' }}</span>
      <span class="date-item">送仓：{{ row.plannedData.plannedDeliveryDate || '-' }}</span>
      <span class="date-item">还箱：{{ row.plannedData.plannedReturnDate || '-' }}</span>
    </span>
  </template>
</el-table-column>
```

- **图标：** 🏢 OfficeBuilding

- **字段：** `row.plannedData.plannedReturnDate`

## 📊 业务逻辑

### 为什么需要卸柜日期？

**业务场景：**

1. **提柜** → 从堆场提取重箱
2. **送仓** → 将重箱送到仓库
3. **卸柜** → 在仓库卸货（关键节点！）
4. **还箱** → 将空箱还回堆场

**卸柜日期的重要性：**

- 📅 **免费期起算点**：滞港费、滞箱费的免费期从卸柜日开始计算
- 💰 **费用计算**：影响堆场堆存费、滞箱费等
- ⏰ **时效管理**：卸柜效率影响后续还箱时间
- 📊 **KPI 统计**：仓库作业效率指标
