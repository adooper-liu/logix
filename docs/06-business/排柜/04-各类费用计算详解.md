## 1. 费用类型识别逻辑

LogiX 系统通过费用名称自动识别费用类型：

```typescript
// 费用类型识别函数
function getChargeType(std: { chargeTypeCode?: string; chargeName?: string }): string {
  const code = (std.chargeTypeCode ?? '').toUpperCase();
  const name = (std.chargeName ?? '').toLowerCase();
  
  // 1. 合并 D&D (同时包含 Demurrage 和 Detention)
  if (code.includes('DEMURRAGE') && code.includes('DETENTION')) return 'D&D';
  if (name.includes('demurrage') && name.includes('detention')) return 'D&D';
  
  // 2. 滞箱费 (Detention)
  if (code.includes('DETENTION') || name.includes('detention') || name.includes('滞箱')) 
    return 'Detention';
  
  // 3. 港口堆存费 (Storage)
  if (code.includes('STORAGE') || name.includes('storage') || name.includes('堆存')) 
    return 'Storage';
  
  // 4. 滞港费 (Demurrage) - 默认
  if (code.includes('DEMURRAGE') || name.includes('demurrage') || name.includes('滞港')) 
    return 'Demurrage';
  
  return 'Unknown';
}
```

## 2. 滞港费 (Demurrage) 计算

### 定义
- **区间**: 到港/卸船 → 实际提柜日（无则当天）
- **起算日**: ATA 或卸船日 (dischargeDate)
- **截止日**: 实际提柜日 或 今天 (取小值)
- **免费日**: lastFreeDate = 起算日 + freeDays - 1

### 计算公式
```
chargeDays = max(0, endDate - lastFreeDate - 1)
费用 = chargeDays × ratePerDay
```

### 场景模拟

**参数**:
- ATA: 2026-02-01
- 卸船日: 2026-02-03
- 计划提柜日: 2026-02-15
- 实际提柜日: 2026-02-20
- 免费天数: 7天
- 日费率: $100/天
- 计算日期: 2026-02-25

**计算过程**:

| 步骤 | 计算 | 结果 |
|------|------|------|
| 1. 起算日 | 取ATA | 2026-02-01 |
| 2. lastFreeDate | 02-01 + 7 - 1 | 2026-02-07 |
| 3. 截止日 | min(实际提柜日, today) | 2026-02-20 |
| 4. chargeStart | lastFreeDate + 1 | 2026-02-08 |
| 5. chargeDays | 02-20 - 02-08 | 13天 |
| 6. 费用 | 13 × $100 | $1,300 |

**结果**: $1,300

## 3. 滞箱费 (Detention) 计算

### 定义
- **区间**: 实际提柜日 → 实际还箱日（无则当天）
- **起算日**: 实际提柜日 或 计划提柜日 (forecast模式)
- **截止日**: 实际还箱日 或 今天 (取大值)
- **免费日**: lastReturnDate = 提柜日起算 + freeDays - 1

### 核心逻辑
```typescript
// 起算日确定
if (actual模式) {
  detentionStartDate = 实际提柜日;
} else {
  // forecast模式
  if (有计划提柜日) {
    detentionStartDate = 计划提柜日;
  } else if (有lastFreeDate) {
    detentionStartDate = lastFreeDate; // fallback
  }
}

// 截止日确定
if (actual模式) {
  detentionEndDate = 实际还箱日 ?? 今天;
} else {
  // forecast模式：取大值（未还箱则按今天计算）
  detentionEndDate = max(实际还箱日, 今天);
}
```

## 4. 堆存费 (Storage) 计算

### 定义
- **区间**: 港口堆存期间
- **起算日**: ATA 或 卸船日
- **截止日**: 还箱日 或 今天

### 计算公式
```
storageDays = max(0, endDate - startDate - freeStorageDays)
storageCost = storageDays × storageRate
```

## 5. 费用类型与状态关联

| 费用类型 | 计算条件 | 状态要求 |
|----------|----------|----------|
| 滞港费 | ATA 发生后 | at_port 或之后 |
| 滞箱费 | 实际提柜后 | picked_up 或之后 |
| 堆存费 | 港口堆存期间 | at_port |

| 堆存费 | ATA/卸船日 | 还箱日/today | freeStorageDays |
