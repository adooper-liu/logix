## 🔗 后端对接需求

### 数据库表结构

```sql
-- 手动设置表
CREATE TABLE IF NOT EXISTS ext_capacity_manual_settings (
  id SERIAL PRIMARY KEY,
  date DATE NOT NULL UNIQUE,
  capacity INTEGER NOT NULL DEFAULT 0,
  reason TEXT,
  created_at TIMESTAMP DEFAULT NOW(),
  updated_at TIMESTAMP DEFAULT NOW()
);

-- 索引
CREATE INDEX idx_capacity_manual_date ON ext_capacity_manual_settings(date);

-- 视图（查询每日能力）
CREATE OR REPLACE VIEW v_daily_capacity AS
SELECT 
  d.date,
  CASE 
    WHEN m.capacity IS NOT NULL THEN m.capacity
    WHEN EXTRACT(DOW FROM d.date) IN (0, 6) THEN 0
    ELSE c.base_capacity * c.weekday_multiplier
  END as final_capacity,
  COALESCE(o.occupied_count, 0) as occupied,
  -- 计算剩余能力
  ...
FROM generate_date_range(...) d
LEFT JOIN ext_capacity_manual_settings m ON d.date = m.date
LEFT JOIN dict_scheduling_config c ON c.config_key = '...'
LEFT JOIN ext_warehouse_daily_occupancy o ON d.date = o.date
```

### API 端点实现

**Controller 示例**:
```typescript
// backend/src/controllers/capacity.controller.ts
@Controller('/api/capacity')
export class CapacityController {
  @Get('/range')
  async getCapacityRange(
    @Query('start') start: string,
    @Query('end') end: string
  ) {
    const service = new CapacityService()
    const data = await service.getCapacityRange(start, end)
    return { success: true, data }
  }
  
  @Post('/manual')
  async setManualCapacity(@Body() body: any) {
    const { date, capacity, reason } = body
    await service.setManualCapacity(date, capacity, reason)
    return { success: true }
  }
}
```
