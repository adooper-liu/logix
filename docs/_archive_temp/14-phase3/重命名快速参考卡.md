## 📦 **接口变更**

### **修改前**

```typescript
export interface UnloadOption {
  unloadDate: Date; // ❌ 语义不明
}
```

### **修改后**

```typescript
export interface UnloadOption {
  plannedPickupDate: Date; // ✅ 计划提柜日（核心输入）
  plannedUnloadDate?: Date; // ✅ 计划卸柜日（可选）
}
```

---

## 🎯 **三种策略的日期推导**

| 策略                   | 提柜日 | 送仓日 | 卸柜日 | 还箱日 | 推导规则            |
| ---------------------- | ------ | ------ | ------ | ------ | ------------------- |
| **Direct / Live load** | D1     | D1     | D1     | D1     | 提=送=卸=还         |
| **Drop off**           | D1     | D3     | D3     | D6     | 提<送=卸，还=卸+3   |
| **Expedited**          | D(-1)  | D(-1)  | D(-1)  | D(-1)  | 免费期内提=送=卸=还 |

## ⚠️ **常见错误**

### **❌ 错误 1：混淆提柜日和卸柜日**

```typescript
// 错误写法
const pickupDate = option.unloadDate; // unloadDate 不存在！
const yardDays = daysBetween(option.unloadDate, deliveryDate); // 错误！
```

### **✅ 正确写法**

```typescript
const pickupDate = option.plannedPickupDate;
const unloadDate = option.plannedUnloadDate || addDays(option.plannedPickupDate, 2);
const yardDays = daysBetween(pickupDate, unloadDate);
```

---

### **❌ 错误 2：Drop off 模式不计算卸柜日**

```typescript
// 错误写法
const option = {
  plannedPickupDate: D1,
  strategy: "Drop off",
  // ❌ 没有 plannedUnloadDate，会导致推导错误
};
```

### **✅ 正确写法**

```typescript
// 系统会自动推导
const option = {
  plannedPickupDate: D1,
  strategy: "Drop off",
  // ✅ evaluateTotalCost 会自动推导 plannedUnloadDate = D1 + 2
};
```

## 📊 **费用计算逻辑**

### **Drop off 模式外部堆场堆存费**

```typescript
if (strategy === "Drop off" && truckingCompany.hasYard) {
  const pickupDayStr = plannedPickupDate.toISOString().split("T")[0];
  const deliveryDayStr = plannedUnloadDate.toISOString().split("T")[0];

  if (pickupDayStr !== deliveryDayStr) {
    // ✅ 提 < 送，收费
    const yardStorageDays = daysBetween(plannedPickupDate, plannedUnloadDate);
    const yardStorageCost = standardRate * yardStorageDays + yardOperationFee;
  } else {
    // ❌ 提 = 送，不收费（直接送仓）
    const yardStorageCost = 0;
  }
}
```
