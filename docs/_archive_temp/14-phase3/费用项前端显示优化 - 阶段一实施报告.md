### **3. 平均费用统计**

#### **实现代码**

```typescript
// 平均费用
const averageCost = computed(() => {
  const successWithCost = props.previewResults.filter((r) => r.success && r.estimatedCosts?.totalCost);

  if (successWithCost.length === 0) return 0;
  return totalEstimatedCost.value / successWithCost.length;
});

// 平均费用颜色类型
const averageCostType = computed(() => {
  const avg = averageCost.value;
  if (avg === 0) return "success"; // 绿色
  if (avg <= 100) return ""; // 默认色
  if (avg <= 500) return "warning"; // 橙色
  return "danger"; // 红色
});
```

#### **展示效果**

```

│ 平均费用     │ 费用区间     │
├──────────────┼──────────────┤
│ [橙色] $450  │ $0 ~ $1,200  │
└──────────────┴──────────────┘
```

**作用**:

- 📊 快速了解整体成本水平
- 🎯 判断是否有异常高费用拉高平均值
- 💡 辅助决策：如果平均费用过高，可能需要重新排产

### **4. 费用区间统计**

#### **实现代码**

```typescript
// 最小费用
const minCost = computed(() => {
  const costs = props.previewResults.filter((r) => r.success && r.estimatedCosts?.totalCost).map((r) => r.estimatedCosts!.totalCost!);
  return costs.length > 0 ? Math.min(...costs) : 0;
});

// 最大费用
const maxCost = computed(() => {
  const costs = props.previewResults.filter((r) => r.success && r.estimatedCosts?.totalCost).map((r) => r.estimatedCosts!.totalCost!);
  return costs.length > 0 ? Math.max(...costs) : 0;
});
```

**业务价值**:

- 📈 **离散程度分析**: 区间越大，费用分布越分散
- 🎯 **异常识别**: 如果 max >> min，说明有货柜费用异常高
- 💰 **成本控制**: 重点关注接近 max 的货柜，优化空间大
