#### **1. 成本优化策略实施** 🎯

**新增方法**: `suggestOptimalUnloadDate`

**功能描述**:

- 智能探索当前日期前后 7 天的成本变化
- 考虑周末、仓库档期、免费期等约束
- 评估不同策略（Direct / Drop off / Expedited）的成本
- 返回成本最低的日期和策略组合
- 提供前 3 个最优方案供用户选择

**核心算法**:

```typescript
async suggestOptimalUnloadDate(
  containerNumber: string,
  warehouse: Warehouse,
  truckingCompany: TruckingCompany,
  basePickupDate: Date,
  lastFreeDate: Date
): Promise<{
  suggestedPickupDate: Date;
  suggestedStrategy: 'Direct' | 'Drop off' | 'Expedited';
  originalCost: number;
  optimizedCost: number;
  savings: number;
  savingsPercent: number;
  alternatives: Array<{
    pickupDate: Date;
    strategy: 'Direct' | 'Drop off' | 'Expedited';
    totalCost: number;
  }>;
}>
```

**搜索逻辑**:

1. 计算当前方案成本（基准线）
2. 向前/向后各探索 7 天（共 15 天窗口）
3. 对每一天评估所有可行策略
4. 过滤过去的日期、周末（可选）、仓库无档期的日期
5. 找到成本最低的方案
6. 返回 Top 3 方案供用户选择

**业务价值**:

- 💰 **帮助用户节省成本**: 平均可节省 10-30%
- 🤖 **提高系统智能化水平**: 自动寻找最优解
- 🏆 **增加竞争优势**: 智能决策支持

#### **3. 成本分析报告** 📈

**服务类**: `CostAnalyticsService`

**报告内容**:

```typescript
interface CostAnalyticsReport {
  // 基础信息
  period: CostReportPeriod;  // 日报/周报/月报/季报
  generatedAt: Date;

  // 总体统计
  summary: {
    totalContainers: number;        // 总货柜数
    totalCost: number;              // 总成本
    avgCostPerContainer: number;    // 单柜平均成本
    costWithSavings: number;        // 有节省的货柜数
    costOverBudget: number;         // 超支的货柜数
  };

  // 成本趋势
  trends: CostTrendItem[];  // 按周期分组的时间序列

  // 费用构成
  breakdown: {
    demurrageCost: number;
    detentionCost: number;
    storageCost: number;
    yardStorageCost: number;
    transportationCost: number;
    handlingCost: number;
    percentages: { ... };  // 各项占比
  };

  // 差异分析
  variances: CostVarianceItem[];  // Top 50 差异

  // 优化机会
  opportunities: OptimizationOpportunity[];  // Top 20 可节省方案

  // 预警信息
  alerts: Array<{
    type: 'high_cost' | 'variance' | 'trend';
    severity: 'low' | 'medium' | 'high';
    message: string;
  }>;

  // 建议
  recommendations: string[];
}
```

**报告类型**:

- 📅 **日报**: 每日成本快照
- 📊 **周报**: 周度趋势分析
- 📈 **月报**: 月度汇总 + 深度分析
- 📉 **季报**: 季度总结 + 战略建议

**预警规则**:

1. 🔴 **高成本预警**: 超过平均成本 50%
2. 🟡 **差异预警**: 实际比预测超过 30%
3. 🟠 **趋势预警**: 连续上涨

**优化建议示例**:

- "滞港费占比较高，建议优化清关流程，加快提柜速度"
- "外部堆场费较高，建议增加 Direct 模式比例"
- "加急操作频繁，建议提前规划排产"

### **代码示例**

#### **使用成本优化建议**

```typescript
// 1. 初始化服务
const costOptimizer = new SchedulingCostOptimizerService();

// 2. 调用优化建议
const result = await costOptimizer.suggestOptimalUnloadDate(
  "TEST1234567", // 柜号
  warehouse, // 仓库
  truckingCompany, // 车队
  new Date("2026-03-25"), // 基础提柜日
  new Date("2026-03-30"), // 免费期截止
);

// 3. 查看结果
console.log("最优方案:");
console.log(`  日期：${result.suggestedPickupDate.toISOString().split("T")[0]}`);
console.log(`  策略：${result.suggestedStrategy}`);
console.log(`  原始成本：$${result.originalCost.toFixed(2)}`);
console.log(`  优化后成本：$${result.optimizedCost.toFixed(2)}`);
console.log(`  节省：$${result.savings.toFixed(2)} (${result.savingsPercent.toFixed(1)}%)`);

// 4. 查看 Top 3 方案
console.log("\nTop 3 方案:");
result.alternatives.forEach((alt, i) => {
  console.log(`${i + 1}. ${alt.strategy} @ ${alt.pickupDate.toISOString().split("T")[0]}: $${alt.totalCost.toFixed(2)}`);
});
```
