## 概述

### 功能目标

为货柜管理提供**智能预警**功能，帮助用户及时识别和处理异常状态的货柜。

### 预警类型

| 级别       | 名称     | 触发条件                   | 视觉标识                  |
| ---------- | -------- | -------------------------- | ------------------------- |
| ⚠️ Warning | 即将超期 | lastFreeDate ≤ 今天 + 3 天 | 橙色边框 (2px)            |
| 🔴 Danger  | 逾期未提 | lastFreeDate < 今天        | 红色边框 (3px) + 脉冲动画 |

### 核心价值

- ✅ **提前预警**: 在问题发生前提醒用户
- ✅ **分级处理**: 不同级别采用不同视觉强度
- ✅ **直观展示**: 甘特图中直接标识，无需切换视图
- ✅ **实时统计**: 统计面板实时显示预警数量

## 预警规则

### 规则定义

#### Rule 1: 即将超期预警 (Warning)

**触发条件**:

```typescript
lastFreeDate ≤ today + 3 days && lastFreeDate ≥ today
```

**含义**:

- 最晚提柜日期在未来 3 天内
- 但还未逾期

**视觉标识**:

- 橙色边框 (#FFA500)
- 边框宽度：2px
- 无动画

**示例**:

```
今天：2026-03-21
lastFreeDate: 2026-03-23 → ⚠️ Warning (还有 2 天)
```

#### Rule 2: 逾期未提预警 (Danger)

**触发条件**:

```typescript
lastFreeDate < today;
```

**含义**:

- 最晚提柜日期已过
- 货柜尚未提取

**视觉标识**:

- 红色边框 (#DC3545)
- 边框宽度：3px
- 脉冲动画（呼吸灯效果）

**示例**:

```
今天：2026-03-21
lastFreeDate: 2026-03-20 → 🔴 Danger (已逾期 1 天)
```

### 规则配置

```typescript
// useGanttLogic.ts
interface AlertRule {
  id: string; // 规则 ID
  name: string; // 规则名称
  level: "warning" | "danger"; // 预警级别
  condition: (container: Container) => boolean; // 触发条件
  message: (container: Container) => string; // 预警消息
}

const ALERT_RULES: AlertRule[] = [
  {
    id: "overdue_not_pickup",
    name: "逾期未提",
    level: "danger",
    condition: (container) => {
      const lastFreeDate = getLastFreeDateFromContainer(container);
      if (!lastFreeDate) return false;

      const today = new Date();
      today.setHours(0, 0, 0, 0);

      return lastFreeDate < today;
    },
    message: (container) => {
      const lastFreeDate = getLastFreeDateFromContainer(container);
      const today = new Date();
      today.setHours(0, 0, 0, 0);

      const diffDays = Math.floor((today.getTime() - lastFreeDate.getTime()) / (1000 * 60 * 60 * 24));
      return `最晚提柜日期为 ${formatDate(lastFreeDate)}，已逾期 ${diffDays} 天`;
    },
  },
  {
    id: "about_to_expire",
    name: "即将超期",
    level: "warning",
    condition: (container) => {
      const lastFreeDate = getLastFreeDateFromContainer(container);
      if (!lastFreeDate) return false;

      const today = new Date();
      today.setHours(0, 0, 0, 0);

      const threeDaysLater = new Date(today);
      threeDaysLater.setDate(threeDaysLater.getDate() + 3);

      return lastFreeDate <= threeDaysLater && lastFreeDate >= today;
    },
    message: (container) => {
      const lastFreeDate = getLastFreeDateFromContainer(container);
      const today = new Date();
      today.setHours(0, 0, 0, 0);

      const diffDays = Math.floor((lastFreeDate.getTime() - today.getTime()) / (1000 * 60 * 60 * 24));
      return `最晚提柜日期为 ${formatDate(lastFreeDate)}，还剩 ${diffDays} 天`;
    },
  },
];
```

## 核心 API

### 函数清单

#### 1. getContainerAlerts(container)

**功能**: 获取容器的所有预警信息

**签名**:

```typescript
function getContainerAlerts(container: Container): AlertInfo[];
```

**返回值**:

```typescript
interface AlertInfo {
  ruleId: string; // 规则 ID
  ruleName: string; // 规则名称
  level: "warning" | "danger"; // 预警级别
  message: string; // 预警消息
}
```

**示例**:

```typescript
const alerts = getContainerAlerts(container);
// 返回: [
//   {
//     ruleId: 'about_to_expire',
//     ruleName: '即将超期',
//     level: 'warning',
//     message: '最晚提柜日期为 2026-03-23，还剩 2 天'
//   }
// ]
```

#### 2. hasAlert(container)

**功能**: 快速判断容器是否有预警

**签名**:

```typescript
function hasAlert(container: Container): boolean;
```

**返回值**: `true` - 有预警，`false` - 无预警

**用途**: 条件渲染、过滤等场景

**示例**:

```vue
<div v-if="hasAlert(container)" class="alert-badge">
  !
</div>
```

#### 3. getAlertLevel(container)

**功能**: 获取容器的预警级别（取最高级别）

**签名**:

```typescript
function getAlertLevel(container: Container): "warning" | "danger" | null;
```

**返回值**:

- `'danger'` - 最高优先级（只要有 danger 级预警）
- `'warning'` - 次高优先级（只有 warning 级预警）
- `null` - 无预警

**示例**:

```typescript
const level = getAlertLevel(container);
// 返回: 'danger' | 'warning' | null
```

#### 4. getContainerBorderStyle(container)

**功能**: 生成容器的边框样式（用于甘特图渲染）

**签名**:

```typescript
function getContainerBorderStyle(container: Container): CSSProperties;
```

**返回值**:

```typescript
{
  border: '2px solid #FFA500',  // Warning
  animation: 'none'
}
```

或

```typescript
{
  border: '3px solid #DC3545',  // Danger
  animation: 'pulse 2s infinite'
}
```

**用途**:

```vue
<div :style="getContainerBorderStyle(container)">
  {{ container.container_number }}
</div>
```

#### 5. isCriticalDate(date)

**功能**: 判断某个日期是否为关键日期（未来 3 天内）

**签名**:

```typescript
function isCriticalDate(date: Date): boolean;
```

**返回值**: `true` - 是关键日期，`false` - 不是

**用途**: 高亮显示、特殊标记等场景

#### 6. getLastFreeDateFromContainer(container)

**功能**: 从容器对象中提取最晚提柜日期

**签名**:

```typescript
function getLastFreeDateFromContainer(container: Container): Date | null;
```

**返回值**: `Date` 对象或 `null`

**数据来源优先级**:

1. `container.last_free_date` - 主字段
2. `container.portOperations?.[0]?.last_free_date` - 港口操作表
3. `null` - 都没有则返回 null

### SQL 验证查询

```sql
-- 查询即将超期的货柜（未来 3 天）
SELECT
  container_number,
  last_free_date,
  DATEDIFF(day, GETDATE(), last_free_date) AS days_remaining
FROM biz_containers
WHERE last_free_date BETWEEN GETDATE() AND DATEADD(day, 3, GETDATE())
ORDER BY last_free_date ASC;

-- 查询逾期未提的货柜
SELECT
  container_number,
  last_free_date,
  DATEDIFF(day, last_free_date, GETDATE()) AS overdue_days
FROM biz_containers
WHERE last_free_date < GETDATE()
ORDER BY last_free_date ASC;

-- 查询所有有预警的货柜
SELECT
  container_number,
  last_free_date,
  CASE
    WHEN last_free_date < GETDATE() THEN '逾期未提'
    WHEN last_free_date <= DATEADD(day, 3, GETDATE()) THEN '即将超期'
    ELSE '正常'
  END AS alert_status
FROM biz_containers
WHERE last_free_date <= DATEADD(day, 3, GETDATE())
ORDER BY
  CASE
    WHEN last_free_date < GETDATE() THEN 1
    WHEN last_free_date <= DATEADD(day, 3, GETDATE()) THEN 2
    ELSE 3
  END;
```

### 常见问题排查

#### Q1: 为什么没有显示预警？

**检查点**:

1. 容器数据是否包含 `last_free_date` 字段
2. `useGanttLogic` 是否正确导入和初始化
3. 计算属性是否正确处理数据
4. 视图是否正确绑定样式

**调试代码**:

```typescript
console.log("[Debug] Container data:", container);
console.log("[Debug] Last free date:", getLastFreeDateFromContainer(container));
console.log("[Debug] Alerts:", getContainerAlerts(container));
```

#### Q2: 为什么预警级别不对？

**可能原因**:

- 日期格式不正确
- 时区问题导致日期比较错误
- 多个预警时没有取最高级别

**解决方案**:

```typescript
// 确保日期标准化
const today = new Date();
today.setHours(0, 0, 0, 0);

// 确保取最高级别
function getAlertLevel(container): string | null {
  const alerts = getContainerAlerts(container);
  if (alerts.length === 0) return null;

  // Danger 优先级最高
  if (alerts.some((a) => a.level === "danger")) return "danger";

  // 其次 Warning
  if (alerts.some((a) => a.level === "warning")) return "warning";

  return null;
}
```

#### Q3: 脉冲动画不生效？

**检查点**:

1. CSS 动画定义是否正确
2. 是否应用了正确的类名
3. 浏览器兼容性

**调试代码**:
