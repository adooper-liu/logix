## 🎯 使用指南

### 新手入门路径

1. **第一步**: 阅读 [成本优化计算逻辑场景模拟](./专题%20-%20成本优化计算逻辑场景模拟.md)
   - 了解费用类型和计算模式
   - 掌握基本公式和参数

2. **第二步**: 阅读 [智能排产计算逻辑场景模拟](./专题%20-%20智能排产计算逻辑场景模拟.md)
   - 理解五个计划日期推导
   - 掌握卸柜日锚点概念

3. **第三步**: 对照实际代码验证
   - 查看 `demurrage.service.ts` 实现
   - 查看 `intelligentScheduling.service.ts` 实现

### 问题排查流程

```
遇到计算问题
    ↓
1. 确定问题类型
    ├─ 滞港费 → 查看场景 1-5
    ├─ 滞箱费 → 查看场景 6-7
    ├─ D&D → 查看场景 8
    ├─ 排产日期 → 查看智能排产场景 1-13
    └─ 成本优化 → 查看场景 11-12
    ↓
2. 找到对应场景
    ↓
3. 对比参数和计算过程
    ↓
4. 检查代码位置 (文档中有标注)
    ↓
5. 运行验证 SQL
```
