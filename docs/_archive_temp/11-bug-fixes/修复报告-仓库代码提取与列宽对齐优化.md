## ✅ 修复方案

### 1️⃣ 从 warehouseName 提取仓库代码

**修复前：**

```typescript
const warehouseCode = row.plannedData?.warehouseCode || row.warehouseCode || row.destinationWarehouse || row.warehouseId;
// ❌ 只有 4 个获取源，都不存在
```

**修复后：**

```typescript
const warehouseCode =
  row.plannedData?.warehouseCode || // 优先从 plannedData 获取
  row.warehouseCode || // 从根对象获取
  row.destinationWarehouse || // 备用字段 1
  row.warehouseId || // 备用字段 2
  row.warehouseName?.split(" ")[0] || // ✅ 新增：从仓库名称提取第 1 个单词
  row.plannedData?.warehouseName?.split(" ")[0]; // ✅ 新增：从 plannedData 提取

// 示例：
// warehouseName = "Bedford Warehouse" → warehouseCode = "Bedford"
// warehouseName = "GBFXT Distribution Center" → warehouseCode = "GBFXT"
```

**工作原理：**

```javascript
"Bedford Warehouse".split(" ")[0]; // → "Bedford"
"GBFXT Distribution Center".split(" ")[0]; // → "GBFXT"
```

## 🚀 后续优化建议

### 建议 1：更智能的仓库代码提取

```typescript
// 支持多种分隔符
const extractWarehouseCode = (name: string) => {
  if (!name) return null;

  // 尝试提取第一个单词、第一个缩写等
  const patterns = [
    /^([A-Z]+)/, // 提取开头大写字母：GBFXT Warehouse
    /^([^-\s]+)/, // 提取第一个连字符前的内容
    /^([^_\s]+)/, // 提取第一个下划线前的内容
    /^(\S+)/, // 提取第一个单词
  ];

  for (const pattern of patterns) {
    const match = name.match(pattern);
    if (match) return match[1];
  }

  return name.split(" ")[0];
};
```
