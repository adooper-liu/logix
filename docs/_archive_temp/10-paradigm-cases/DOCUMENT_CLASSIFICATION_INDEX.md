## 🎯 使用指南

### 新人入门路径

```markdown
Day 1: 阅读快速指南.md
Day 2: 学习开发范式案例库.md
Day 3-5: 选择一个模块，阅读相关架构文档
Week 2: 在实际项目中应用，参考最佳实践
```

### Code Review 参考

```markdown
1. 打开对应类别的文档
2. 查找类似案例
3. 对照检查清单
4. 确认符合最佳实践
```

### 技术方案设计

```markdown
1. 阅读架构设计文档
2. 查找类似案例
3. 应用 SKILL 原则
4. 组织方案评审
5. 编写实施清单
```
