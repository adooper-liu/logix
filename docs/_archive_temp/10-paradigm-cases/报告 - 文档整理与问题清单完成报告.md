#### 典型问题 Top 5

1. **ts-node 模块加载错误** (最复杂)
   - 耗时：6h
   - 根因：TypeScript ESM 模块问题
   - 方案：创建专用 tsconfig.json + JS 包装器

2. **档期字段名错误** (最常见)
   - 影响：3 个文件需要修改
   - 根因：未验证实体定义
   - 教训：数据库优先开发原则

3. **资源状态显示混乱** (用户体验影响最大)
   - 根因：缓存数据结构不一致
   - 方案：添加类型检查
   - 改进：分离产能负荷和日期紧急度

4. **Vue 组件类型冲突** (最具启发性)
   - 根因：过度追求类型安全
   - 方案：适当使用 any
   - 平衡：功能优先于类型完美

5. **search_replace 匹配失败** (最高频)
   - 出现：5 次
   - 根因：上下文不够唯一
   - 方案：扩大上下文范围
