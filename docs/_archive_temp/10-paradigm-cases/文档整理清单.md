## 🏷️ 命名规范说明

### 命名格式

```
分类 - 名称【中文名称】.md

组成部分:
- 分类：索引 | 指南 | 专题 | 核心 | 报告 | 案例 | 方案 | 清单
- 名称：简洁描述（英文或拼音）
- 中文名称：可选，便于理解
```

### 分类标识

| 分类前缀 | 用途 | 示例 |
|---------|------|------|
| **索引** | 各类索引文档 | 索引 - 案例库总索引 |
| **指南** | 使用指南、快速入门 | 指南 - 案例库使用指南 |
| **专题** | 专题案例集 | 专题 - 成本优化案例集 |
| **核心** | 核心范式文档 | 核心 - 开发范式案例库 |
| **报告** | 完成报告、总结 | 报告 - 文档整理完成报告 |
| **案例** | 单个案例文档 | 案例 - 排产确认保存 |
| **方案** | 技术方案设计 | 方案 - 智能排产优化 |
| **清单** | 检查清单 | 清单 - 架构分析检查项 |

### 示例

```markdown
✅ 正确命名:
- 索引 - 案例库总索引 [CASE_STUDY_INDEX].md
- 指南 - 案例库使用指南.md
- 专题 - 成本优化完整案例集.md
- 核心 - 开发范式案例库.md
- 案例 - 排产确认保存功能完善.md
- 方案 - 智能排产成本优化设计.md
- 清单 - 架构分析五维检查项.md

❌ 错误命名:
- CASE_STUDY_INDEX.md (无分类前缀)
- 成本优化案例.md (不明确是专题还是单个案例)
- 报告.md (名称太模糊)
```
