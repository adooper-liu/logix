## 🛠️ 使用技巧

### 技巧 1: 高效搜索

```bash
# 按关键字搜索
grep -r "架构分析" docs/ --include="*.md" -l

# 按标签搜索
grep -r "标签.*SKILL" docs/ --include="*.md" -l

# 组合搜索
grep -r "成本优化.*性能提升" docs/ --include="*.md" -A 5 -B 5
```

### 技巧 2: 对比学习

```markdown
选择两个相关案例:
1. 成功案例 vs 失败案例
2. 不同解决方案对比
3. 前后版本差异

对比维度:
- 问题定义
- 解决方案
- 实施过程
- 最终结果
- 经验教训
```

### 技巧 3: 模板化应用

```markdown
遇到新问题时:
1. 打开案例模板
2. 填写问题描述
3. 查找类似案例
4. 借鉴解决方案
5. 记录解决过程
6. 沉淀为新的案例
```

## 📞 获取帮助

### 常见问题

**Q: 如何快速找到需要的文档？**
A: 使用 `CASE_STUDY_INDEX.md` 或 `DOCUMENT_CLASSIFICATION_INDEX.md` 的索引，或直接用 grep 搜索。

**Q: 案例太多看不完怎么办？**
A: 先看"快速入门"，再根据当前工作选择相关专题，不必一次性看完所有案例。

**Q: 如何贡献案例？**
A: 参考案例模板编写，然后提交 PR 到 docs/ 目录。
