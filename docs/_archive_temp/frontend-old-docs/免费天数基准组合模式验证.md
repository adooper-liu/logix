### 核心函数

#### 1. freePeriodUsesWorkingDays (L215-218)

```typescript
/** 免费期是否按工作日（周六、周日不计入） */
function freePeriodUsesWorkingDays(basis: string | null | undefined): boolean {
  const b = (basis ?? '').toLowerCase();
  return b.includes('工作 + 自然') || b.includes('natural+working') || b === '工作日' || b === 'working';
}
```

**判断逻辑**:
```
返回 true (按工作日计算免费期) 的情况:
1. basis 包含 "工作 + 自然" → 工作日 + 自然日模式
2. basis 包含 "natural+working" → 自然日 + 工作日模式的英文
3. basis === "工作日" → 纯工作日模式
4. basis === "working" → 纯工作日模式英文

返回 false (按自然日计算免费期) 的情况:
1. basis 包含 "自然 + 工作" → 自然日 + 工作日模式
2. basis 包含 "working+natural" → 工作日 + 自然日模式的英文
3. basis === "自然日" → 纯自然日模式
4. basis === "natural" → 纯自然日模式英文
5. 其他或未设置 → 默认自然日
```

**验证结果**: ✅ 
- "工作 + 自然" → 返回 `true` (免费期按工作日) ✅
- "自然 + 工作" → 返回 `false` (免费期按自然日) ✅

### 模式 3: 自然日 + 工作日 (Natural + Working) ⭐

```typescript
basis = "自然 + 工作" 或 "working+natural"

freePeriodUsesWorkingDays("自然 + 工作") = false  // 免费期按自然日 ✓
chargePeriodUsesWorkingDays("自然 + 工作") = true  // 计费期按工作日 ✓

结果: 
- 免费天数按自然日计算 (连续计算，包括周末)
- 超过免费期后，计费天数只计工作日 (排除周末)
```

**业务场景示例**:
```
场景：free_days = 7, pickup_date = 2026-03-20 (周五), return_date = 2026-03-30 (周一)

步骤 1: 计算免费期截止日 (按自然日)
n = 7 - 1 = 6
lastFreeDate = 03-20 + 6 天 = 03-26 (周四)
  (03-20, 03-21, 03-22, 03-23, 03-24, 03-25, 03-26 = 7 个自然日)

步骤 2: 计算计费天数 (按工作日)
chargeStart = 03-27 (周五)
实际还箱 = 03-30 (周一)
逐日检查:
  03-27 (周五) ✓ → Day 1
  03-28 (周六) ✗ → 跳过
  03-29 (周日) ✗ → 跳过
  03-30 (周一) ✓ → Day 2
chargeDays = 2 个工作日

结果: 免费期 7 个自然日，计费 2 个工作日
```

结果: 免费期 5 个工作日，计费 1 个自然日

```typescript

// 步骤 2: 计费期按工作日计算 (L259-261)

const chargeDays = workingDaysBetween(03-27, 03-30);
// 03-27 (周五) ✓
// 03-28 (周六) ✗ 跳过
// 03-29 (周日) ✗ 跳过

// chargeDays = 2 个工作日

totalAmount = 2 × $100 = $200
```

**业务含义**:
- 客户享有 7 个自然日的免费用箱期 (包括周末)
- 超过免费期后，只在工作日用箱才计费 (周末免费)
- 这种模式对客户更有利
