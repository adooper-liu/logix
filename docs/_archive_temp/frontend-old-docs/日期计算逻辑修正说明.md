## ❌ 原始错误理解

### 错误表述

```
提柜日 = 清关日 + 1
送柜日 = 卸柜日（Drop off）或 提柜日（Live load）
卸柜日 = 根据仓库档期确定
还箱日 = 卸柜日（Live load）或 卸柜日 +1（Drop off）
```

### 问题分析

这种表述给人的印象是：
1. **提柜日是简单公式计算** → 清关日 +1
2. **卸柜日是独立确定的** → 根据仓库档期
3. **送柜日是二选一** → 卸柜日 或 提柜日

**实际逻辑并非如此简单！**

### 核心流程（正向推导）

                 ▼

                 │

## 🔑 核心要点

### 1. 卸柜日是瓶颈

**不是** 先确定卸柜日再推导其他日期，而是：

```typescript
// ❌ 错误理解
unloadDate = someFormula()  // 某个公式计算
deliveryDate = unloadDate   // 然后推导送柜日

// ✅ 正确逻辑
pickupDate = clearanceDate + 1              // 先确定提柜日
unloadDate = findEarliestAvailableWarehouse() // 再查仓库产能
deliveryDate = unloadMode === 'Live load' ? pickupDate : unloadDate
```

### 2. 送柜日不是独立计算的

**不是** 在提柜日和卸柜日之间做选择，而是：

```typescript
// ❌ 错误理解
deliveryDate = (mode === 'Live load') ? pickupDate : unloadDate

// ✅ 正确逻辑
if (mode === 'Live load') {
  // 提柜后直接送仓，所以送柜日 = 提柜日 = 卸柜日
  deliveryDate = pickupDate;
} else {
  // 先拉堆场再送仓，所以送柜日 = 卸柜日
  deliveryDate = unloadDate;
}
```

### 3. 还箱日取决于模式和最晚还箱日

**不是** 固定公式，需要考虑约束：

```typescript
// 基础逻辑
if (mode === 'Live load') {
  returnDate = unloadDate;    // 当天还
} else {
  returnDate = unloadDate + 1;  // 次日还
}

// 但要受 lastReturnDate 约束
if (lastReturnDate && returnDate > lastReturnDate) {
  returnDate = lastReturnDate;  // 不能超过最晚还箱日
}

// 兜底：还箱日必须 >= 卸柜日
if (returnDate < unloadDate) {
  returnDate = unloadDate;
  if (mode === 'Drop off') {
    returnDate += 1;
  }
}
```

#### 计划卸柜日（关键步骤！）

```typescript
// ⚠️ 这不是简单公式！需要先确定提柜日，再查仓库产能
unloadDate = findEarliestAvailableWarehouse(warehouse, pickupDate)

// 算法步骤:
// 1. 从 pickupDate 开始检查仓库每天的剩余产能
// 2. 找到第一个 remaining > 0 的日期
// 3. 如果 pickupDate 当天无产能，继续往后找
// 4. 返回最早可用日期

// 伪代码:
function findEarliestAvailableWarehouse(warehouse, startDate) {
  currentDate = startDate
  while (true) {
    occupancy = getWarehouseOccupancy(warehouse, currentDate)
    if (occupancy.remaining > 0) {
      return currentDate  // 找到可用日期
    }
    currentDate = currentDate + 1 day  // 检查下一天
  }
}
```

**核心要点**:
- ✅ **卸柜日由仓库产能决定**：不是固定公式计算
- ✅ **需要查询 ext_warehouse_daily_occupancy 表**
- ✅ **从提柜日起查找最早可用日期**
- ✅ **可能等于提柜日，也可能晚于提柜日**

## 🎯 关键代码位置

### 后端实现

**文件**: `backend/src/services/intelligentScheduling.service.ts`

**关键方法**:

1. **计算提柜日** (Line 489-505):
```typescript
private async calculatePlannedPickupDate(customsDate: Date, lastFreeDate?: Date): Promise<Date> {
  const pickupDate = new Date(customsDate);
  pickupDate.setDate(pickupDate.getDate() + 1); // 清关后次日提柜

  if (lastFreeDate) {
    const lastFree = new Date(lastFreeDate);
    lastFree.setHours(0, 0, 0, 0);
    if (pickupDate > lastFree) {
      pickupDate.setTime(lastFree.getTime());
    }
  }

  await this.skipWeekendsIfNeeded(pickupDate);
  return pickupDate;
}
```

2. **查找最早可用卸柜日** (Line 650-680):
```typescript
private async findEarliestAvailableWarehouse(
  warehouses: Warehouse[],
  startDate: Date
): Promise<{ warehouse: Warehouse; plannedUnloadDate: Date }> {
  let currentDate = new Date(startDate);
  const maxDate = new Date(startDate);
  maxDate.setDate(maxDate.getDate() + 30); // 最多查找 30 天

  while (currentDate <= maxDate) {
    for (const warehouse of warehouses) {
      const occupancy = await this.warehouseOccupancyRepo.findOne({
        where: { warehouseCode: warehouse.warehouseCode, date: currentDate }
      });

      const capacity = occupancy?.capacity ?? warehouse.dailyCapacity ?? 20;
      const remaining = occupancy?.remaining ?? capacity;

      if (remaining > 0) {
        return { warehouse, plannedUnloadDate: currentDate };
      }
    }
    currentDate.setDate(currentDate.getDate() + 1);
  }

  return { warehouse: warehouses[0], plannedUnloadDate: null as any };
}
```

3. **计算送柜日** (Line 701-711):
```typescript
private calculatePlannedDeliveryDate(
  pickupDate: Date,
  unloadMode: string,
  unloadDate: Date
): Date {
  if (unloadMode === 'Live load') {
    return new Date(pickupDate); // 提 = 送（同日）
  }
  // Drop off：送 = 卸（送仓日即卸柜日）
  return new Date(unloadDate);
}
```

4. **计算还箱日** (Line 718-745):
```typescript
private calculatePlannedReturnDate(
  unloadDate: Date,
  unloadMode: string,
  lastReturnDate?: Date
): Date {
  const returnDate = new Date(unloadDate);
  if (unloadMode === 'Drop off') {
    returnDate.setDate(returnDate.getDate() + 1); // Drop off 卸柜后次日还箱
  }
  
  // 受 lastReturnDate 约束
  if (lastReturnDate && returnDate > lastReturnDate) {
    returnDate.setTime(lastReturnDate.getTime());
  }
  
  return returnDate;
}
```
