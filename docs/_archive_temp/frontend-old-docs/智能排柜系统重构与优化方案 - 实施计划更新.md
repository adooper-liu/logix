### 2.2 技术要点

#### SchedulingCostOptimizerService
- **职责**: 成本优化服务，负责生成所有可行方案并评估成本
- **核心方法**:
  - `generateAllFeasibleOptions()` - 生成所有可行的卸柜方案
  - `evaluateTotalCost()` - 评估单个方案的总成本
  - `selectBestOption()` - 选择成本最低的方案

#### 方案生成逻辑
- **Direct 方案**: 直接卸柜，评估不同日期的成本
- **Drop off 方案**: 先送到外部堆场，再预约卸柜
- **Expedited 方案**: 协调加急，安排在免费期内

#### 成本评估逻辑
- **滞港费**: 调用 `predictDemurrageForUnloadDate()`
- **滞箱费**: 调用 `predictDetentionForReturnDate()`
- **堆存费**: 从配置读取外部堆场日费率
- **运输费**: Phase 2 暂置为 0，Phase 3 实现
- **操作费**: 从配置读取加急操作费

#### 前端展示
- **成本明细**: 展示各项成本和总成本
- **优化建议**: 显示成本优化建议
- **多方案对比**: 允许用户查看和选择方案
