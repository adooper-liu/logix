# 开发环境

LogiX 开发环境搭建与配置。

## 环境要求

| 软件 | 版本 | 说明 |
|------|------|------|
| Node.js | 18+ | 前端/后端运行时 |
| PostgreSQL | 14+ | 主数据库 |
| TimescaleDB | 2.13+ | 时序数据库（可选） |
| Docker | 24+ | 容器化（推荐） |
| Git | 2.0+ | 版本控制 |

## 本地开发环境

### 1. 安装 Node.js

从 [nodejs.org](https://nodejs.org/) 下载 LTS 版本。

### 2. 安装 PostgreSQL

```bash
# macOS
brew install postgresql@14
brew services start postgresql@14

# Ubuntu
sudo apt install postgresql-14

# Windows
# 下载安装包 https://www.postgresql.org/download/windows/
```

### 3. 创建数据库

```bash
psql -U postgres

CREATE USER logix_user WITH PASSWORD 'LogiX@2024!Secure';
CREATE DATABASE logix_db OWNER logix_user;
GRANT ALL PRIVILEGES ON DATABASE logix_db TO logix_user;
```

### 4. 配置环境变量

```bash
# backend/.env
DB_HOST=localhost
DB_PORT=5432
DB_USERNAME=logix_user
DB_PASSWORD=LogiX@2024!Secure
DB_DATABASE=logix_db
PORT=3001
```

## Docker 开发环境

### 启动数据库

```bash
# 使用 docker-compose
docker-compose -f docker-compose.timescaledb.yml up -d

# 验证
docker ps
```

### 常用命令

```bash
# 启动
docker-compose -f docker-compose.timescaledb.yml up -d

# 停止
docker-compose -f docker-compose.timescaledb.yml down

# 查看日志
docker-compose -f docker-compose.timescaledb.yml logs -f

# 进入容器
docker exec -it logix-db psql -U logix_user -d logix_db
```
