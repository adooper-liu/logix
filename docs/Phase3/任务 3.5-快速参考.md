## 🐛 常见问题

### Q1: 方案数量为 0？
**A**: 这是预期行为，如果数据库中没有实际数据。测试时可以使用 mock 数据。

### Q2: 成本计算为 0？
**A**: 检查 `dict_scheduling_config` 表中是否有配置项：
```sql
SELECT * FROM dict_scheduling_config 
WHERE config_key IN (
  'transport_base_rate_per_mile',
  'transport_direct_multiplier',
  'transport_dropoff_multiplier',
  'transport_expedited_multiplier'
);
```

### Q3: ECharts 不显示？
**A**: 确保已安装 echarts：
```bash
npm install echarts --save
```
