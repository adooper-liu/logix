## 💡 常见问题

### Q1: 日历不显示？

**A**: 检查以下几点：
1. 依赖是否安装成功：`npm list @fullcalendar/vue3`
2. 导入语句是否正确
3. 浏览器控制台是否有错误
4. Network 请求是否成功

### Q2: 样式错乱？

**A**: 添加样式覆盖：
```scss
.calendar-capacity-view {
  :deep(.fc) {
    all: initial;
    font-family: inherit;
  }
}
```

### Q3: API 请求失败 404？

**A**: 这是正常的，因为后端还没实现。可以先用 Mock 数据测试前端功能。
