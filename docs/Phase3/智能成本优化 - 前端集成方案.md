## 💻 **代码实现**

### **核心数据结构**

```typescript
interface OptimizationResult {
  optimizedCount: number      // 可优化的货柜数量
  totalSavings: number        // 总节省金额
  alternatives?: any[]        // Top 3 方案（待实现）
}
```

---

### **响应式变量**

```typescript
const optimizing = ref(false)  // 优化加载中
const optimizationResult = ref<OptimizationResult | null>(null)
```

---

### **优化函数**

```typescript
const handleSmartOptimization = async () => {
  optimizing.value = true
  optimizationResult.value = null
  
  try {
    // TODO: 调用后端智能优化 API
    // const result = await costOptimizerService.suggestOptimalUnloadDate(...)
    
    // 模拟优化结果（演示用）
    await new Promise(resolve => setTimeout(resolve, 1500))
    
    optimizationResult.value = {
      optimizedCount: Math.floor(props.previewResults.length * 0.3), // 假设 30% 可优化
      totalSavings: Math.random() * 500 + 100, // 随机节省 100-600
      alternatives: [] // Top 3 方案
    }
    
    ElMessage.success(
      `发现 ${optimizationResult.value.optimizedCount} 个货柜可优化，预计节省 $${optimizationResult.value.totalSavings.toFixed(2)}`
    )
  } catch (error) {
    ElMessage.error('智能优化失败，请稍后重试')
  } finally {
    optimizing.value = false
  }
}
```

## 🔌 **后端 API 接口**

### **API 端点**

```typescript
POST /api/scheduling/optimize-cost

Request:
{
  containers: string[],           // 柜号列表
  warehouseCode: string,          // 仓库代码
  truckingCompanyId: string,      // 车队 ID
  basePickupDate: string,         // 基础提柜日
  lastFreeDate: string            // 免费期截止
}

Response:
{
  code: 200,
  data: {
    originalCost: number,         // 原始总成本
    optimizedCost: number,        // 优化后总成本
    savings: number,              // 节省金额
    savingsPercent: number,       // 节省百分比
    alternatives: [{              // Top 3 方案
      containerNumber: string,
      pickupDate: string,
      strategy: string,
      totalCost: number,
      savings: number
    }]
  }
}
```

const optimizationCache = new Map<string, OptimizationResult>()

  if (optimizationCache.has(key)) {
    return optimizationCache.get(key)
  }
  const result = await callApi()
  optimizationCache.set(key, result)
