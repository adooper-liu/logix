#### ① **概念澄清**

**核心成果**：明确区分了两种完全不同的堆存相关费用

| 维度 | 港口存储费（Storage Charge） | 外部堆场堆存费（Yard Storage Fee） |
|------|---------------------------|----------------------------------|
| **英文名称** | Storage Charge | Yard Storage Fee |
| **收取方** | 港口/码头 | 拖车车队 |
| **数据源** | `ext_demurrage_standards` | `dict_trucking_port_mapping` |
| **字段映射** | `chargeTypeCode = 'STORAGE'` | `standard_rate` + `yard_operation_fee` |
| **判断条件** | 匹配到 STORAGE 标准 | `yard_capacity > 0` + Drop off |
| **费用范畴** | D&D 费用类型 | 运输环节附加费 |
| **计费区间** | 到港日 → 提柜日 | 卸柜日 → 还箱日（堆场期间） |
| **免费期** | 有（根据 freeDays） | 无 |
| **阶梯费率** | 支持 | 不支持 |
| **计算复杂度** | 复杂（状态机 + 多行累加） | 简单（`rate * days + fee`） |

### 📊 **完整费用类型体系**

#### **5 种核心费用类型**

| 费用类型 | 英文 | 数据源 | 计算逻辑 |
|---------|------|--------|---------|
| **滞港费** | Demurrage | `ext_demurrage_standards` | `calculateForContainer()` + `isDemurrageCharge()` |
| **滞箱费** | Detention | `ext_demurrage_standards` | `calculateForContainer()` + `isDetentionCharge()` |
| **港口存储费** | Storage Charge | `ext_demurrage_standards` | `calculateForContainer()` + `isStorageCharge()` |
| **外部堆场费** | Yard Storage Fee | `dict_trucking_port_mapping` | `standardRate * days + yardOperationFee` |
| **运输费** | Transportation Cost | `dict_trucking_port_mapping` | `transportFee`（Drop off 模式翻倍） |

### 🎯 **关键澄清点**

#### ❌ **误解 1：所有堆存费是一种费用**

**✅ 正确理解**：
- 这是**两种完全不同**的费用
- 就像"停车费"和"过路费"的区别
- 收取方不同、计算逻辑不同、数据源不同

---

#### ❌ **误解 2：TruckingPortMapping.standardRate 是港口存储费率**

**✅ 正确理解**：
- `standardRate` 用于**外部堆场堆存费**
- **不是**港口的 Storage Charge
- 港口存储费应从 `ext_demurrage_standards` 读取

---

#### ❌ **误解 3：Drop off 模式只计算一种堆存费**

**✅ 正确理解**：
Drop off 模式可能产生**两笔**堆存相关费用：

```
场景：货柜延迟提柜 + Drop off 模式

1. 港口存储费（Storage Charge）
   - 原因：货柜在港口堆场存放超过免费期
   - 计算：$50/天 × 2 天 = $100

2. 外部堆场堆存费（Yard Storage Fee）
   - 原因：货柜在车队堆场临时存放
   - 计算：$80/天 × 3 天 + $50 = $290

总计：$390（两笔堆存费同时存在）
```

### 🚀 **技术实现细节**

#### **1. 外部堆场堆存费计算逻辑**

```typescript
// 仅在以下条件同时满足时计算：
// 1. unloadMode === 'Drop off'
// 2. truckingCompany.hasYard === true

// 计算公式：
yardStorageCost = (truckingPortMapping.standardRate || 0) * yardStorageDays + 
                 (truckingPortMapping.yardOperationFee || 0);

// 其中：
// - yardStorageDays = daysBetween(plannedUnloadDate, plannedReturnDate)
// - standardRate: 从 TruckingPortMapping 读取（每日堆场费率）
// - yardOperationFee: 从 TruckingPortMapping 读取（堆场操作费）
```

#### **2. 数据来源说明**

| 字段 | 表名 | 说明 |
|------|------|------|
| `standardRate` | `dict_trucking_port_mapping` | 外部堆场每日费率 |
| `yardOperationFee` | `dict_trucking_port_mapping` | 堆场操作费（一次性） |
| `yardCapacity` | `dict_trucking_port_mapping` | 堆场容量（>0 表示有堆场） |
| `hasYard` | `dict_trucking_companies` | 车队是否有堆场 |
