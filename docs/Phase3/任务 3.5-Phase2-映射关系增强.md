## 📋 需求概述

### 用户需求

1. **严格遵循映射关系**: 成本优化功能应像智能排柜一样，从 `dict_warehouse_trucking_mapping` 和 `dict_trucking_port_mapping` 两个映射表中获取仓库和车队

2. **费用结构增强**:
   - ① **运输费** → `trucking-port-mapping.transport_fee`（拖卡费，为每一次的总运输费用）
   - ② **外部堆场费用**:
     - `trucking-port-mapping.standard_rate`（收费标准，每天费用）
     - `trucking-port-mapping.yard_operation_fee`（堆场操作费，一个货柜收取一次）
   - ③ **堆场容量** → `trucking-port-mapping.yard_capacity`（每天最多可接受的 Drop 模式货柜量，用来缓解仓库压力）

3. **确认现有表结构**:
   - 仓库每天的卸柜能力：✅ 已有 `dict_warehouses.daily_unload_capacity`
   - 车队每天的提送柜能力：✅ 已有 `dict_trucking_companies.daily_capacity`

```typescript
async getCandidateWarehouses(
  countryCode: string,
  portCode: string
): Promise<Warehouse[]> {
  if (!portCode || !countryCode) {
    return [];  // ✅ 无参数时返回空
  }

  // 1. 港口→车队（dict_trucking_port_mapping）
  const portMappings = await this.truckingPortMappingRepo.find({
    where: { portCode, country: countryCode, isActive: true }
  });
  if (portMappings.length === 0) return [];  // ✅ 无映射时返回空

  const truckingCompanyIds = [...new Set(portMappings.map(m => m.truckingCompanyId))];

  // 2. 车队→仓库（dict_warehouse_trucking_mapping）
  const warehouseMappings = await this.warehouseTruckingMappingRepo.find({
    where: {
      country: countryCode,
      truckingCompanyId: truckingCompanyIds,
      isActive: true
    }
  });
  if (warehouseMappings.length === 0) return [];  // ✅ 无映射时返回空

  const warehouseCodes = [...new Set(warehouseMappings.map(m => m.warehouseCode))];

  // 3. 获取仓库详细信息
  const warehouses = await this.warehouseRepo.find({
    where: {
      warehouseCode: warehouseCodes,
      country: countryCode,
      status: 'ACTIVE'
    }
  });
  
  return warehouses;  // ✅ 只返回映射链中的仓库
}
```

```typescript
private async calculateTransportationCost(
  containerNumber: string,
  warehouse: Warehouse,
  strategy: 'Direct' | 'Drop off' | 'Expedited'
): Promise<number> {
  // 1. 获取目的港
  const container = await this.containerRepo.findOne({
    where: { containerNumber },
    relations: ['portOperations']
  });
  
  const destPo = container?.portOperations?.find(po => po.portType === 'destination');
  const portCode = destPo?.portCode || 'USLAX';
  const countryCode = warehouse.country || 'US';
  
  // 2. 获取仓库关联的车队
  const warehouseMappings = await this.warehouseTruckingMappingRepo.find({
    where: {
      warehouseCode: warehouse.warehouseCode,
      country: countryCode,
      isActive: true
    }
  });
  
  if (warehouseMappings.length === 0) {
    log.warn(`No trucking mapping for warehouse ${warehouse.warehouseCode}`);
    return 0;
  }
  
  // 3. 从 TruckingPortMapping 获取运输费
  let transportFee = 0;
  let storageRate = 0;
  let yardOpFee = 0;
  
  for (const mapping of warehouseMappings) {
    const truckingPortMapping = await this.truckingPortMappingRepo.findOne({
      where: {
        country: countryCode,
        portCode: portCode,
        truckingCompanyId: mapping.truckingCompanyId,
        isActive: true
      }
    });
    
    if (truckingPortMapping) {
      transportFee = truckingPortMapping.transportFee || 0;
      storageRate = truckingPortMapping.standardRate || 0;
      yardOpFee = truckingPortMapping.yardOperationFee || 0;
      break;
    }
  }
  
  // 4. 根据策略计算费用
  switch (strategy) {
    case 'Direct':
      return transportFee;  // 只收拖卡费
      
    case 'Drop off':
      const storageDays = this.calculateStorageDays(option);
      return storageRate * storageDays + yardOpFee;  // 堆场费
      
    case 'Expedited':
      return transportFee * 1.5;  // 加急 50%
      
    default:
      return transportFee;
  }
}
```

#### 3.4 新增 calculateStorageDays 方法 ⏳

```typescript
private calculateStorageDays(option: UnloadOption): number {
  // Drop off 模式：从卸柜日到还箱日
  const unloadDate = option.unloadDate;
  const returnDate = option.returnDate || unloadDate;
  
  const diffTime = Math.abs(returnDate.getTime() - unloadDate.getTime());
  const diffDays = Math.ceil(diffTime / (1000 * 60 * 60 * 24));
  
  return Math.max(1, diffDays);  // 至少 1 天
}
```
