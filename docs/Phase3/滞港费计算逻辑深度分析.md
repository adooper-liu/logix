## 一、核心服务架构

### 1.1 DemurrageService 定位


### 4.2 判断是否到达目的港

```typescript
function isArrivedAtDestinationPortForDemurrage(ls: LogisticsStatusResult): boolean {
  const { status, currentPortType } = ls;
  
  // 已提柜/已卸柜/已还箱 → 视为已到目的港
  if (status === SimplifiedStatus.PICKED_UP ||
      status === SimplifiedStatus.UNLOADED ||
      status === SimplifiedStatus.RETURNED_EMPTY) {
    return true;
  }
  
  // AT_PORT 且 currentPortType=destination → 已到目的港
  if (status === SimplifiedStatus.AT_PORT && currentPortType === 'destination') {
    return true;
  }
  
  // 其他情况（中转港到港、在途、未出运）→ 未到目的港
  return false;
}
```

### 4.3 计算模式映射

| 物流状态 | 计算模式 | 使用日期类型 |
|---------|---------|-------------|
| `PICKED_UP` (已提柜) | **actual** | 实际提柜日、实际还箱日 |
| `UNLOADED` (已卸柜) | **actual** | 实际卸柜日、实际提柜日 |
| `RETURNED_EMPTY` (已还箱) | **actual** | 实际提柜日、实际还箱日 |
| `AT_PORT` (目的港) | **actual** | ATA、实际提柜日 |
| `AT_PORT` (中转港) | **forecast** | ETA、计划提柜日 |
| `IN_TRANSIT` (在途) | **forecast** | ETA、计划提柜日 |
| `NOT_SHIPPED` (未出运) | **forecast** | ETA、计划提柜日 |

**核心逻辑**:
```typescript
// 已到达目的港 → actual 模式
if (isArrivedAtDestinationPortForDemurrage(logisticsSnapshot)) {
  calculationMode = 'actual';
  // 使用实际日期：pickupDateActual, returnTime
} else {
  calculationMode = 'forecast';
  // 使用计划日期：plannedPickupDate, plannedReturnDate
}
```

## 五、费用类型识别

### 5.1 四种费用类型判断函数

```typescript
/**
 * 判断是否为「Demurrage & Detention」合并费用项
 * chargeTypeCode/chargeName 同时包含 Demurrage 与 Detention
 */
function isCombinedDemurrageDetention(std): boolean {
  const code = (std.chargeTypeCode ?? '').toUpperCase();
  const name = (std.chargeName ?? '').toLowerCase();
  const hasDem = code.includes('DEMURRAGE') || name.includes('demurrage') || name.includes('滞港');
  const hasDet = code.includes('DETENTION') || name.includes('detention') || name.includes('滞箱');
  return hasDem && hasDet;
}

/**
 * 判断是否为纯滞箱费（Detention）标准
 * 排除合并类型
 */
function isDetentionCharge(std): boolean {
  if (isCombinedDemurrageDetention(std)) return false;
  const code = (std.chargeTypeCode ?? '').toUpperCase();
  const name = (std.chargeName ?? '').toLowerCase();
  return code.includes('DETENTION') || name.includes('detention') || name.includes('滞箱');
}

/**
 * 判断是否为堆存费（Storage）
 * 港区堆存，区间为「到港侧起算日 → 提柜侧截止日」
 */
function isStorageCharge(std): boolean {
  if (isCombinedDemurrageDetention(std)) return false;
  if (isDetentionCharge(std)) return false;
  const code = (std.chargeTypeCode ?? '').toUpperCase();
  const name = (std.chargeName ?? '').toLowerCase();
  return code.includes('STORAGE') || name.includes('storage') || name.includes('堆存');
}

/**
 * 判断是否为纯滞港费（Demurrage）标准
 * 到港侧起算，提柜截止
 */
function isDemurrageCharge(std): boolean {
  if (isCombinedDemurrageDetention(std)) return false;
  if (isDetentionCharge(std)) return false;
  if (isStorageCharge(std)) return false;
  const code = (std.chargeTypeCode ?? '').toUpperCase();
  const name = (std.chargeName ?? '').toLowerCase();
  return code.includes('DEMURRAGE') || name.includes('demurrage') || name.includes('滞港');
}
```

### 5.2 费用类型对比表

| 费用类型 | 起算日 | 截止日 | 免费期基准 |
|---------|-------|-------|-----------|
| **Demurrage (滞港费)** | ATA / ETA / 卸柜日 | 提柜日 / 今日 | 自然日或工作日 |
| **Detention (滞箱费)** | 提柜日 | 还箱日 / 今日 | 自然日或工作日 |
| **Storage (堆存费)** | ATA / 卸柜日 | 提柜日 / 今日 | 通常按自然日 |
| **D&D Combined** | ATA / ETA / 卸柜日 | 还箱日 / 今日 | 一个免费期覆盖全程 |

## 七、滞港费标准匹配逻辑

### 7.1 四字段匹配

```typescript
async matchStandards(containerNumber: string): Promise<ExtDemurrageStandard[]> {
  const params = await this.getContainerMatchParams(containerNumber);
  const today = toDateOnly(new Date());
  
  // 1. 获取所有收费项（is_chargeable='N'）
  const allChargeable = await this.standardRepo
    .createQueryBuilder('s')
    .where('s.is_chargeable = :chargeable', { chargeable: 'N' })
    .orderBy('s.sequence_number', 'ASC')
    .getMany();
  
  // 2. 四字段匹配（目的港、船公司、货代、客户）
  const fourFieldMatched = allChargeable.filter((s) => {
    if (params.destinationPortCode) {
      if (!matchCodeOrName(params.destinationPortCode, s.destinationPortCode, s.destinationPortName))
        return false;
    }
    if (params.shippingCompanyCode) {
      if (!matchCodeOrName(params.shippingCompanyCode, s.shippingCompanyCode, s.shippingCompanyName))
        return false;
    }
    if (params.originForwarderCode) {
      if (!matchCodeOrName(params.originForwarderCode, s.originForwarderCode, s.originForwarderName))
        return false;
    }
    if (params.foreignCompanyCode) {
      if (!matchCodeOrName(params.foreignCompanyCode, s.foreignCompanyCode, s.foreignCompanyName))
        return false;
    }
    return true;
  });
  
  // 3. 有效期匹配
  const validityMatched = fourFieldMatched.filter((s) => {
    if (s.effectiveDate) {
      const eff = toDateOnly(s.effectiveDate);
      if (eff > today) return false;
    }
    if (s.expiryDate) {
      const exp = toDateOnly(s.expiryDate);
      if (exp < today) return false;
    }
    return true;
  });
  
  if (validityMatched.length > 0) return validityMatched;
  
  // 4. 有效期无一匹配但四字段有匹配 → 取最新标准
  const latest = fourFieldMatched
    .sort((a, b) => b.effectiveDate.getTime() - a.effectiveDate.getTime())
    .filter((s, i, arr) => i === 0 || s.effectiveDate.getTime() === arr[0].effectiveDate.getTime());
  
  return latest;
}
```

## 八、阶梯费率计算

### 8.1 支持多种存储格式

```typescript
function normalizeTiers(raw: unknown): DemurrageTierDto[] | null {
  // 格式 1: 数组 [{ fromDay, toDay, ratePerDay }]
  if (Array.isArray(raw)) {
    return raw.map(t => ({
      fromDay: Number(t.fromDay ?? t.minDays ?? 0),
      toDay: t.toDay != null ? Number(t.toDay) : t.maxDays != null ? Number(t.maxDays) : null,
      ratePerDay: Number(t.ratePerDay ?? t.rate ?? 0)
    }));
  }
  
  // 格式 2: 对象 { "1": 50, "2-5": 60, "6+": 70 }
  if (typeof raw === 'object' && raw !== null) {
    const entries = Object.entries(raw)
      .filter(([k]) => /^\d+$|^\d+\+$/.test(String(k)))
      .map(([k, v]) => ({
        day: parseInt(String(k).replace(/\D/g, ''), 10),
        rate: Number(v),
        isOpenEnded: String(k).includes('+')
      }))
      .sort((a, b) => a.day - b.day);
    
    // 合并连续相同费率的阶梯
    const tiers: DemurrageTierDto[] = [];
    let i = 0;
    while (i < entries.length) {
      const fromDay = entries[i].day;
      const rate = entries[i].rate;
      let toDay = fromDay;
      
      while (
        i + 1 < entries.length &&
        entries[i + 1].rate === rate &&
        !entries[i + 1].isOpenEnded &&
        entries[i + 1].day === toDay + 1
      ) {
        i++;
        toDay = entries[i].day;
      }
      
      tiers.push({
        fromDay,
        toDay: entries[i].isOpenEnded ? null : toDay,
        ratePerDay: rate
      });
      i++;
    }
    
    return tiers;
  }
  
  return null;
}
```

### 10.2 成本优化搜索空间

```typescript
// 生成所有可行方案
async generateAllFeasibleOptions(
  container: Container,
  basePickupDate: Date,
  lastFreeDate: Date,
  searchDays: number = 15
): Promise<UnloadOption[]> {
  const options: UnloadOption[] = [];
  
  // Direct 策略：探索 15 天内的每一天
  for (let day = 0; day < searchDays; day++) {
    const candidateDate = addDays(basePickupDate, day);
    options.push({
      containerNumber: container.containerNumber,
      warehouse,
      unloadDate: candidateDate,
      strategy: 'Direct',
      isWithinFreePeriod: candidateDate <= lastFreeDate
    });
  }
  
  // Drop off 策略：提前提柜，存放堆场
  for (let day = 0; day < searchDays; day++) {
    const pickupDate = addDays(basePickupDate, day);
    const deliveryDate = addDays(pickupDate, 1); // 次日送仓
    
    if (deliveryDate <= lastFreeDate) {
      options.push({
        containerNumber: container.containerNumber,
        warehouse,
        unloadDate: deliveryDate,
        strategy: 'Drop off',
        isWithinFreePeriod: true
      });
    }
  }
  
  // Expedited 策略：加急提柜
  for (let day = -5; day < 0; day++) {
    const expeditedDate = addDays(basePickupDate, day);
    options.push({
      containerNumber: container.containerNumber,
      warehouse,
      unloadDate: expeditedDate,
      strategy: 'Expedited',
      isWithinFreePeriod: expeditedDate <= lastFreeDate
    });
  }
  
  return options;
}
```

```typescript
  };
}
```

```typescript
function freePeriodUsesWorkingDays(basis: string | null | undefined): boolean {
  const b = (basis ?? '').toLowerCase();
  return (
    b.includes('工作 + 自然') || 
    b.includes('natural+working') ||

function chargePeriodUsesWorkingDays(basis: string | null | undefined): boolean {
  const b = (basis ?? '').toLowerCase();
  return (

    b.includes('working+natural') || 
    b === '工作日' || 
    b === 'working'
  );
}
```

**支持的基准类型**:
- `自然日` / `Natural`: 包含周末
- `工作日` / `Working`: 排除周末
- `工作 + 自然日`: 免费期按工作日，计费期按自然日
- `自然 + 工作日`: 免费期按自然日，计费期按工作日

### 11.2 日期计算辅助函数

```typescript
/** 提取日期部分（使用 UTC 避免时区导致 ±1 天偏移） */
function toDateOnly(d: Date | string): Date {
  return new Date(Date.UTC(
    d.getUTCFullYear(),
    d.getUTCMonth(),
    d.getUTCDate()
  ));
}

/** 取较晚一日 */
function maxDate(a: Date, b: Date): Date {
  return a.getTime() >= b.getTime() ? a : b;
}

/** 加 N 天 */
function addDays(d: Date, days: number): Date {
  const result = new Date(d);
  result.setUTCDate(result.getUTCDate() + days);
  return result;
}

/** 加 N 个工作日 */
function addWorkingDays(start: Date, n: number): Date {
  if (n <= 0) return new Date(start.getTime());
  let result = new Date(start.getTime());
  let count = 0;
  while (count < n) {
    if (!isWeekend(result)) count++;
    if (count < n) result.setUTCDate(result.getUTCDate() + 1);
  }
  return result;
}

/** 两日期间的工作日数 */
function workingDaysBetween(start: Date, end: Date): number {
  let count = 0;
  const cur = new Date(start.getTime());
  while (cur.getTime() <= end.getTime()) {
    if (!isWeekend(cur)) count++;
    cur.setUTCDate(cur.getUTCDate() + 1);
  }
  return count;
}

/** 周六 (6)、周日 (0) 为非工作日 */
function isWeekend(d: Date): boolean {
  const dow = d.getUTCDay();
  return dow === 0 || dow === 6;
}
```

### 11.3 匹配兜底逻辑

```typescript
function matchCodeOrName(
  containerVal: string | null,
  stdCode: string | null,
  stdName: string | null
): boolean {
  if (!containerVal) return true;  // 货柜侧无值，视为匹配
  
  const cv = String(containerVal).trim().toLowerCase();
  const sc = stdCode ? String(stdCode).trim().toLowerCase() : '';
  const sn = stdName ? String(stdName).trim().toLowerCase() : '';
  
  if (!sc && !sn) return true;  // 标准侧无要求，视为匹配

  if (sn === cv) return true;   // 名称匹配
  
  return false;
}
```

## 十三、总结

### 13.1 系统已完整支持的功能

✅ **两种计算模式**:
- `forecast` (预测模式): 使用 ETA、计划提柜日、计划还箱日
- `actual` (实际模式): 使用 ATA、实际提柜日、实际还箱日

✅ **四种费用类型**:
- Demurrage (滞港费): 到港→提柜
- Detention (滞箱费): 提柜→还箱
- Storage (堆存费): 到港→提柜（港区堆存）
- D&D Combined (合并费用): 到港→还箱（一口价）

✅ **三种场景支持**:
1. forecast + 无计划提柜 → 使用 LFD 作为 fallback
2. forecast + 有计划提柜 → 使用计划提柜日
3. actual + 有实际提柜 → 使用实际提柜日

✅ **灵活的免费期计算**:
- 自然日 / 工作日 / 混合模式
- 免费期和计费期可独立设置基准

✅ **阶梯费率**:
- 支持多种存储格式（数组、对象）
- 自动合并连续相同费率

✅ **智能匹配**:
- 四字段匹配（目的港、船公司、货代、客户）
- 有效期验证
- 字典解析和映射兜底

✅ **写回机制**:
- 单条更新（前端手动触发）
- 批量更新（定时任务）
- 计算记录持久化

### 13.2 lastFreeDate 的实际用途

**在系统中的真实含义**:

```typescript
// lastFreeDate 是「滞港费的最后免费日」
const lastPickupDate = destPort?.lastFreeDate && destPort.lastFreeDateSource === 'manual'
  ? toDateOnly(destPort.lastFreeDate)
  : null;

// 用途 1: 作为滞港费计算的截止日参考
const _pickupDate = lastPickupDate ?? plannedPickupDate;
const endDate = pickupDateActual ?? today;

// 用途 2: 作为滞箱费计算的 fallback 起算日（当无计划提柜日时）
if (!params.calculationDates.plannedPickupDate && computedLastFreeDate) {
  pickupBasisForDetention = computedLastFreeDate;
}

// 用途 3: 智能成本优化的约束条件
const isWithinFreePeriod = candidateDate <= lastFreeDate;
```

**重要澄清**:

❌ **错误理解**: "系统缺少滞箱费免费期支持"

✅ **正确理解**: 
- 系统已经有完整的滞箱费计算逻辑
- 滞箱费使用独立的 `detentionStartDate` 和 `detentionEndDate`
- 滞箱费的免费期通过 `free_detention_days` 字段支持（如果存在）
- `lastFreeDate` 专用于滞港费，不作为滞箱费的唯一依据

### 13.3 智能成本优化的数据流

```
用户请求
  ↓
SchedulingCostOptimizerService
  ↓
1. 查询基础数据（柜号、仓库、车队、basePickupDate、lastFreeDate）
  ↓
2. 生成可行方案（15 天 × 3 策略 = 45 个候选）
  ↓

  └─ 计算操作费（Expedited 专属）
  ↓
4. 选择总成本最低的方案
  ↓
返回最优方案及成本明细
```
