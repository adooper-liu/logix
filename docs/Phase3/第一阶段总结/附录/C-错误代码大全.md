## 1. 后端错误代码

### 1.1 通用错误

| 错误码 | 说明 | 常见原因 | 解决方案 |
|--------|------|---------|---------|
| ERR_001 | 参数缺失 | 请求缺少必要参数 | 检查请求参数 |
| ERR_002 | 参数无效 | 参数格式或值不正确 | 校验参数格式 |
| ERR_003 | 资源不存在 | 查找的资源ID不存在 | 确认资源ID |
| ERR_004 | 资源已存在 | 重复创建唯一键资源 | 检查唯一约束 |
| ERR_005 | 权限不足 | 用户没有操作权限 | 检查权限配置 |

### 1.2 业务错误

| 错误码 | 说明 | 常见原因 | 解决方案 |
|--------|------|---------|---------|
| BIZ_001 | 货柜不存在 | containerNumber 无效 | 确认货柜号 |
| BIZ_002 | 备货单不存在 | orderNumber 无效 | 确认备货单号 |
| BIZ_003 | 滞港费标准未找到 | 没有匹配的费用标准 | 配置滞港费标准 |
| BIZ_004 | 仓库容量不足 | 仓库已达最大容量 | 调整排产计划 |
| BIZ_005 | 车队不可用 | 车队无可用槽位 | 选择其他车队 |
| BIZ_006 | 日期冲突 | 计划日期已被占用 | 调整计划日期 |
| BIZ_007 | 数据已锁定 | 数据正在被其他操作使用 | 稍后重试 |

### 1.3 数据导入错误

| 错误码 | 说明 | 常见原因 | 解决方案 |
|--------|------|---------|---------|
| IMP_001 | 文件格式错误 | 不是有效的 Excel 文件 | 检查文件格式 |
| IMP_002 | 缺少必填列 | Excel 缺少必要列 | 补充必填列 |
| IMP_003 | 数据类型错误 | 列值类型不符合 | 检查数据类型 |
| IMP_004 | 重复键冲突 | 导入数据与现有冲突 | 检查唯一键 |
| IMP_005 | 外键约束失败 | 关联数据不存在 | 先创建关联数据 |
| IMP_006 | 数据校验失败 | 数据不符合业务规则 | 检查数据有效性 |

### 1.4 API 错误示例

```typescript
// 400 - 参数错误
{
  "success": false,
  "message": "缺少必要参数：containerNumber",
  "error": "ERR_001"
}

// 404 - 资源不存在
{
  "success": false,
  "message": "货柜 CXDU1234567 不存在",
  "error": "BIZ_001"
}

// 500 - 服务器错误
{
  "success": false,
  "message": "服务器内部错误",
  "error": "SYSTEM_001",
  "detail": "数据库连接失败"
}
```

## 2. 前端错误代码

### 2.1 网络错误

| 错误码 | 说明 | 常见原因 | 解决方案 |
|--------|------|---------|---------|
| NET_001 | 请求超时 | 服务器响应超时 | 检查网络/服务器 |
| NET_002 | 网络断开 | 客户端网络不可用 | 检查网络连接 |
| NET_003 | 服务不可用 | 服务器未启动 | 启动后端服务 |
| NET_004 | CORS 错误 | 跨域请求被阻止 | 配置 CORS |

### 2.2 状态码映射

| HTTP 状态码 | 前端处理 |
|-------------|---------|
| 400 | 显示错误信息，允许用户修改 |
| 401 | 跳转登录页 |
| 403 | 显示"无权限"提示 |
| 404 | 显示"资源不存在" |
| 500 | 显示"服务器错误"，记录日志 |

### 2.3 Vue 组件错误

| 错误码 | 说明 | 常见原因 | 解决方案 |
|--------|------|---------|---------|
| VUE_001 | 组件加载失败 | 组件路径错误 | 检查组件路径 |
| VUE_002 | 响应式数据错误 | 数据未定义 | 检查数据定义 |
| VUE_003 | 路由跳转失败 | 路由未注册 | 检查路由配置 |
| VUE_004 | Pinia 状态错误 | Store 未初始化 | 检查 Store |

## 3. 数据库错误

### 3.1 PostgreSQL 错误

| 错误码 | 说明 | 常见原因 | 解决方案 |
|--------|------|---------|---------|
| 23505 | 唯一约束冲突 | 重复插入相同键值 | 检查唯一键 |
| 23503 | 外键约束失败 | 引用不存在的父记录 | 先创建父记录 |
| 22001 | 字符串过长 | 插入值超过字段长度 | 缩短字符串 |
| 22012 | 除零错误 | 表达式包含除以零 | 检查除数 |
| 42804 | 类型不匹配 | 数据类型不兼容 | 转换数据类型 |
| 42501 | 权限不足 | 用户无表操作权限 | 授权 |

### 3.2 TypeORM 错误

| 错误码 | 说明 | 常见原因 | 解决方案 |
|--------|------|---------|---------|
| EntityPropertyNotFound | 实体属性不存在 | 字段名拼写错误 | 检查实体定义 |
| CannotGetEntityMetadataNotDefined | 实体未注册 | 实体未添加到 DataSource | 检查实体导入 |
| QueryFailedError | 查询执行失败 | SQL 语法错误 | 检查 SQL |

### 3.3 错误示例

```sql
-- 23505: 唯一约束冲突
ERROR: duplicate key value violates unique constraint "biz_containers_pkey"
DETAIL: Key (container_number)=(CXDU1234567) already exists.

-- 23503: 外键约束失败
ERROR: insert or update on table "biz_containers" violates foreign key constraint "fk_order"
DETAIL: Key (order_number)=(ORD001) is not present in table "biz_replenishment_orders".
```

## 5. 排查步骤指南

### 5.1 通用排查流程

```
遇到错误
  │
  ▼
1. 查看错误信息
  │
  ├── 有错误码 → 对照本表查找解决方案
  │
  └── 无错误码 → 查看 detail/stack
      │
      ▼
2. 检查常见原因
  │
  ├── 参数问题 → 检查请求参数
  │
  ├── 权限问题 → 检查用户权限
  │
  ├── 数据问题 → 检查数据完整性
  │
      ▼
3. 查看日志
  │
  ├── 后端日志: backend/logs/
  │
  └── 前端日志: 浏览器控制台
      │
      ▼
4. 定位修复
  │
  ├── 自己修复 → 修复后测试
  │
  └── 无法修复 → 记录并上报
```

### 5.2 快速诊断命令

```bash
# 检查后端服务状态
curl http://localhost:3000/health

# 检查数据库连接
psql -h localhost -U logix_user -d logix_db -c "SELECT 1"

# 检查 Redis 连接
redis-cli ping

# 查看错误日志
tail -f backend/logs/error.log
```
