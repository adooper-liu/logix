## 概述

### 功能定位

简单甘特图是 LogiX 智能排柜系统的**核心可视化组件**，用于展示货柜在不同港口的操作时间线。

### 核心价值

- ✅ **直观展示**: 时间轴方式呈现货柜流转过程
- ✅ **多港经停**: 支持起运港 → 中转港 → 目的港的完整链路
- ✅ **预警集成**: 实时显示货柜预警状态
- ✅ **筛选过滤**: 支持多维度数据筛选

### 技术栈

- **前端框架**: Vue 3 + TypeScript
- **UI 组件库**: Element Plus
- **图表库**: FullCalendar / 自研甘特图
- **状态管理**: Composition API (Composables)

### 2. 智能预警

**功能**: 实时识别并标识异常状态的货柜

**预警规则**:

- ⚠️ Warning: lastFreeDate ≤ today + 3 days
- 🔴 Danger: lastFreeDate < today

**视觉呈现**:

- Warning: 橙色边框 (2px)
- Danger: 红色边框 (3px) + 脉冲动画

**实现代码**:

```typescript
function getContainerAlerts(container: Container): AlertInfo[] {
  const alerts: AlertInfo[] = [];
  const lastFreeDate = getLastFreeDateFromContainer(container);

  if (!lastFreeDate) return alerts;

  const today = new Date();
  today.setHours(0, 0, 0, 0);

  // 检查逾期
  if (lastFreeDate < today) {
    alerts.push({
      ruleId: "overdue_not_pickup",
      ruleName: "逾期未提",
      level: "danger",
      message: `最晚提柜日期为 ${formatDate(lastFreeDate)}，已逾期`,
    });
  }

  // 检查即将超期
  const threeDaysLater = new Date(today);
  threeDaysLater.setDate(threeDaysLater.getDate() + 3);

  if (lastFreeDate <= threeDaysLater && lastFreeDate >= today) {
    alerts.push({
      ruleId: "about_to_expire",
      ruleName: "即将超期",
      level: "warning",
      message: `最晚提柜日期为 ${formatDate(lastFreeDate)}，还剩 3 天`,
    });
  }

  return alerts;
}
```

### 3. 多维度筛选

**支持的筛选维度**:

- 仓库 (warehouse)
- 车队 (trucking company)
- 状态 (status: 全部/正常/预警/逾期)
- 时间范围 (date range)

**筛选逻辑**:

```typescript
const filteredContainers = computed(() => {
  return processedContainers.value.filter((container) => {
    // 仓库筛选
    if (selectedWarehouse.value && container.warehouse !== selectedWarehouse.value) {
      return false;
    }

    // 车队筛选
    if (selectedTrucking.value && container.truckingCompany !== selectedTrucking.value) {
      return false;
    }

    // 状态筛选
    if (selectedStatus.value === "warning") {
      const level = getAlertLevel(container);
      return level === "warning" || level === "danger";
    }

    if (selectedStatus.value === "overdue") {
      return getAlertLevel(container) === "danger";
    }

    return true;
  });
});
```

## 技术难点与解决方案

### 难点 1: 大数据量性能优化

**问题**: 当货柜数量超过 1000 个时，渲染缓慢

**解决方案**:

1. **虚拟滚动**: 只渲染可见区域的 DOM
2. **分批渲染**: 使用 `requestAnimationFrame` 分批插入数据
3. **计算缓存**: 使用 `computed` 避免重复计算

**代码示例**:

```typescript
// 分批渲染
function renderInBatches(containers: Container[]) {
  const batchSize = 100;
  let index = 0;

  function renderBatch() {
    const batch = containers.slice(index, index + batchSize);
    visibleContainers.value.push(...batch);

    index += batchSize;
    if (index < containers.length) {
      requestAnimationFrame(renderBatch);
    }
  }

  renderBatch();
}
```

### 难点 2: 多港经停数据组织

**问题**: 一个货柜出现在多个港口，如何正确分组？

**解决方案**:

```typescript
// 按港口分组
const containersByPort = computed(() => {
  const groups: Record<string, ProcessedContainer[]> = {};

  processedContainers.value.forEach((container) => {
    container.portOperations?.forEach((portOp) => {
      const key = portOp.port_code;

      if (!groups[key]) {
        groups[key] = [];
      }

      // 注意：同一个货柜可能出现在多个港口组中
      groups[key].push({
        ...container,
        currentPort: portOp,
      });
    });
  });

  return groups;
});
```

### 难点 3: 时区处理

**问题**: 服务器返回 UTC 时间，前端需要本地化显示

**解决方案**:

```typescript
// 统一的日期格式化函数
function formatDate(date: Date | string | null): string {
  if (!date) return "";

  const d = typeof date === "string" ? new Date(date) : date;

  // 转换为本地时间
  return d.toLocaleDateString("zh-CN", {
    year: "numeric",
    month: "2-digit",
    day: "2-digit",
  });
}

// 日期比较时标准化到午夜
function normalizeDate(date: Date): Date {
  const normalized = new Date(date);
  normalized.setHours(0, 0, 0, 0);
  return normalized;
}
```
