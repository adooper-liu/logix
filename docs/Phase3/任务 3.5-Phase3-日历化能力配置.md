## 📋 需求概述

### 用户需求

支持通过日历设置每日能力，默认规则：
- **周六、周日**: 能力为 0
- **工作日**: 
  - 仓库：`dict_warehouses.daily_unload_capacity`
  - 车队：`dict_trucking_companies.daily_capacity`

### 扩展需求

1. **可配置的周末定义**: 支持不同国家/地区的周末定义（如中东地区周五 - 周六）
2. **节假日支持**: 通过倍率调整工作日能力
3. **手动覆盖**: 允许手动设置特定日期的能力
4. **自动初始化**: 自动创建未来 N 天的档期记录

### 方案一：数据库触发器（❌ 未采用）

- 支持手动覆盖和配置

**优点**:
- 灵活性高
- 易于维护和扩展
- 支持多种配置场景

#### 1.3 示例数据初始化

使用 PL/pgSQL 匿名块初始化未来 30 天的档期数据：

```sql
DO $$
DECLARE
  start_date DATE := CURRENT_DATE;
  day_of_week INTEGER;
  wh_record RECORD;
  capacity_value INTEGER;
BEGIN
  FOR wh_record IN 
    SELECT warehouse_code, daily_unload_capacity 
    FROM dict_warehouses WHERE status = 'ACTIVE'
  LOOP
    FOR i IN 0..29 LOOP
      day_of_week := EXTRACT(DOW FROM (start_date + i));
      
      IF day_of_week = 0 OR day_of_week = 6 THEN
        capacity_value := 0;  -- 周末
      ELSE
        capacity_value := COALESCE(wh_record.daily_unload_capacity, 10);  -- 工作日
      END IF;
      
      INSERT INTO ext_warehouse_daily_occupancy (...)
      VALUES (..., capacity_value, ...)
      ON CONFLICT (warehouse_code, date) DO UPDATE SET ...;
    END LOOP;
  END LOOP;
END $$;
```

### Step 2: 智能日历工具类 ✅

**文件**: `backend/src/utils/smartCalendarCapacity.ts`

#### 2.1 核心接口

```typescript
interface CalendarConfig {
  enabled: boolean;           // 是否启用智能日历
  weekendDays: number[];      // 周末定义 [6, 0] = 周六、周日
  weekdayMultiplier: number;  // 工作日倍率（默认 1.0）
}
```

#### 2.2 主要方法

##### 获取日历配置

```typescript
async getCalendarConfig(): Promise<CalendarConfig> {
  const [enabledConfig, weekendConfig, multiplierConfig] = await Promise.all([
    this.configRepo.findOne({ where: { configKey: 'enable_smart_calendar_capacity' } }),
    this.configRepo.findOne({ where: { configKey: 'weekend_days' } }),
    this.configRepo.findOne({ where: { configKey: 'weekday_capacity_multiplier' } })
  ]);
  
  return {
    enabled: enabledConfig?.configValue === 'true',
    weekendDays: weekendConfig?.configValue.split(',').map(d => parseInt(d.trim())),
    weekdayMultiplier: parseFloat(multiplierConfig?.configValue || '1.0')
  };
}
```

##### 判断休息日

```typescript
async isRestDay(date: Date): Promise<boolean> {
  const config = await this.getCalendarConfig();
  if (!config.enabled) return false;
  
  const dayOfWeek = date.getDay();  // 0=周日，1=周一...6=周六
  return config.weekendDays.includes(dayOfWeek);
}
```

##### 计算仓库能力

```typescript
async calculateWarehouseCapacity(
  warehouseCode: string,
  date: Date
): Promise<number> {
  const warehouse = await this.warehouseRepo.findOne({ where: { warehouseCode } });
  const config = await this.getCalendarConfig();
  
  if (!config.enabled) {
    return warehouse.dailyUnloadCapacity || 10;
  }
  
  const isRest = await this.isRestDay(date);
  if (isRest) return 0;  // 休息日
  
  // 工作日 = 基础能力 × 倍率
  const baseCapacity = warehouse.dailyUnloadCapacity || 10;
  return Math.floor(baseCapacity * config.weekdayMultiplier);
}
```

##### 确保档期记录存在

```typescript
async ensureWarehouseOccupancy(
  warehouseCode: string,
  date: Date
): Promise<ExtWarehouseDailyOccupancy> {
  let occupancy = await this.warehouseOccupancyRepo.findOne({
    where: { warehouseCode, date }
  });
  
  if (occupancy) {
    if (occupancy.capacity > 0) return occupancy;
    
    // 更新 capacity
    occupancy.capacity = await this.calculateWarehouseCapacity(warehouseCode, date);
    occupancy.remaining = Math.max(0, occupancy.capacity - occupancy.plannedCount);
    return await this.warehouseOccupancyRepo.save(occupancy);
  }
  
  // 创建新记录
  const calculatedCapacity = await this.calculateWarehouseCapacity(warehouseCode, date);
  return await this.warehouseOccupancyRepo.create({
    warehouseCode,
    date,
    plannedCount: 0,
    capacity: calculatedCapacity,
    remaining: calculatedCapacity
  });
}
```

##### 批量初始化

```typescript
async initializeFutureOccupancy(days: number = 30): Promise<void> {
  const startDate = new Date();
  const warehouses = await this.warehouseRepo.find({ where: { status: 'ACTIVE' } });
  const truckingCompanies = await this.truckingCompanyRepo.find({ where: { status: 'ACTIVE' } });
  
  for (let i = 0; i < days; i++) {
    const currentDate = new Date(startDate);
    currentDate.setDate(currentDate.getDate() + i);
    
    for (const warehouse of warehouses) {
      await this.ensureWarehouseOccupancy(warehouse.warehouseCode, currentDate);
    }
    
    for (const trucking of truckingCompanies) {
      await this.ensureTruckingOccupancy(trucking.companyCode, currentDate);
    }
  }
}
```

##### 手动设置能力

```typescript
async setManualCapacity(
  type: 'warehouse' | 'trucking',
  code: string,
  date: Date,
  capacity: number
): Promise<void> {
  if (type === 'warehouse') {
    const occupancy = await this.ensureWarehouseOccupancy(code, date);
    occupancy.capacity = capacity;
    await this.warehouseOccupancyRepo.save(occupancy);
  } else {
    const occupancy = await this.ensureTruckingOccupancy(code, date);
    occupancy.capacity = capacity;
    await this.truckingOccupancyRepo.save(occupancy);
  }
}
```

async isWarehouseAvailable(
  warehouse: Warehouse,
  date: Date
): Promise<boolean> {
  const queryDate = new Date(date);
  queryDate.setHours(0, 0, 0, 0);
  
  // 使用智能日历能力确保档期记录存在并设置正确的 capacity
  const occupancy = await smartCalendarCapacity.ensureWarehouseOccupancy(

  }
  
  // 检查剩余容量
  return occupancy.remaining > 0;
}
