## 🔧 **常见问题**

### **Q1: 组件不显示**

**原因**: 导入路径错误

**解决**:

```vue
<!-- 错误 -->
import SchedulingFilterBar from '@/components/SchedulingFilterBar.vue'

<!-- 正确 -->
import SchedulingFilterBar from './components/SchedulingFilterBar.vue'
```

---

### **Q2: Props 类型不匹配**

**原因**: 类型定义不一致

**解决**:

```typescript
// 检查父组件传递的数据类型
defineProps<{
  dateRange: [Date, Date]; // 必须是元组类型
  selectedPortCode: string; // 必须是字符串
}>();
```

---

### **Q3: Events 不触发**

**原因**: emit 定义错误

**解决**:

```typescript
// 正确写法
defineEmits<{
  'preview': []
  'refresh': []
}>()

// 模板中使用
<el-button @click="$emit('preview')" />
```

## 🎨 **样式问题**

### **样式丢失**

检查是否使用 `scoped`:

```vue
<style lang="scss" scoped>
/* ✅ 正确：scoped 隔离样式 */
.stat-bar { ... }
</style>
```

---

### **全局样式冲突**

如果多个组件使用相同类名：

```vue
<!-- 组件 A -->
<style scoped>
.stat-bar { ... }
</style>

<!-- 组件 B -->
<style scoped>
.stat-bar { ... }
</style>

<!-- 不会冲突，因为 scoped 自动添加 data-v-xxx 属性 -->
```
