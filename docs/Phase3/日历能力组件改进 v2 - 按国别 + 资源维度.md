## 🎯 改进需求

### 用户反馈

> "仓库要到具体的每个仓库，而不是所有仓库统一，按国别 + 仓库名。车队按国别 + 车队名"

### 原始问题

第一次改进后虽然支持了仓库/车队分别查看，但还存在以下问题：

❌ **问题 1**: 选择仓库时显示所有国家的仓库
```
选择仓库: 
- WH001 (美国)
- WH002 (美国)
- WH003 (英国)
- WH004 (加拿大)
...混合显示，难以查找
```

❌ **问题 2**: 无法快速定位特定国家的资源
- 美国用户只能看到美国的仓库和车队
- 加拿大用户只能看到加拿大的资源
- 跨国运营时需要频繁切换查看

### 改进目标

✅ **按国别分组**:
- 先选择国家
- 再选择该国家下的具体仓库/车队
- 实现两级筛选

✅ **显示格式优化**:
- 仓库名称：`US-洛杉矶一号仓`
- 车队名称：`CA-快速物流`
- 清晰标识所属国家

## 🔧 技术实现

### 数据结构设计

```typescript
// 国家列表
interface Country {
  code: string      // 'US', 'CA', 'GB'
  name: string      // '美国', '加拿大', '英国'
}

// 仓库数据
interface Warehouse {
  warehouseCode: string
  warehouseName: string
  country: string
  dailyUnloadCapacity: number
}

// 车队数据
interface TruckingCompany {
  truckingCompanyId: string
  truckingCompanyName: string
  country: string
  dailyCapacity: number
}
```

### 状态管理

```typescript
// 资源类型
const resourceType = ref<'warehouse' | 'trucking'>('warehouse')

// 国家列表
const countries = ref<Country[]>([])

// 所有仓库和车队（未过滤）
const allWarehouses = ref<Warehouse[]>([])
const allTruckingCompanies = ref<TruckingCompany[]>([])

// 选中的国家和资源
const selectedCountry = ref<string>('')
const selectedWarehouseCode = ref<string>('')
const selectedTruckingCode = ref<string>('')
```

### 计算属性

```typescript
// 动态过滤仓库列表
const filteredWarehouses = computed(() => {
  if (!selectedCountry.value) return []
  return allWarehouses.value.filter(
    (wh: Warehouse) => wh.country === selectedCountry.value
  )
})

// 动态过滤车队列表
const filteredTruckingCompanies = computed(() => {
  if (!selectedCountry.value) return []
  return allTruckingCompanies.value.filter(
    (tc: TruckingCompany) => tc.country === selectedCountry.value
  )
})

// 带国家的资源名称
const selectedWarehouseName = computed(() => {
  const wh = allWarehouses.value.find(
    (w: Warehouse) => w.warehouseCode === selectedWarehouseCode.value
  )
  return wh ? `${wh.country}-${wh.warehouseName}` : ''
})

const selectedTruckingName = computed(() => {
  const tc = allTruckingCompanies.value.find(
    (t: TruckingCompany) => t.truckingCompanyId === selectedTruckingCode.value
  )
  return tc ? `${tc.country}-${tc.truckingCompanyName}` : ''
})
```

### API 调用逻辑

```typescript
const loadResources = async () => {
  try {
    const countrySet = new Set<string>()
    
    // 加载仓库列表并收集国家
    const warehouseResponse = await api.get('/api/scheduling/resources/warehouses')
    if (warehouseResponse.data.success) {
      allWarehouses.value = warehouseResponse.data.data
      allWarehouses.value.forEach((wh: any) => {
        if (wh.country && !countrySet.has(wh.country)) {
          countrySet.add(wh.country)
        }
      })
    }
    
    // 加载车队列表并收集国家
    const truckingResponse = await api.get('/api/scheduling/resources/truckings')
    if (truckingResponse.data.success) {
      allTruckingCompanies.value = truckingResponse.data.data
      allTruckingCompanies.value.forEach((tc: any) => {
        if (tc.country && !countrySet.has(tc.country)) {
          countrySet.add(tc.country)
        }
      })
    }
    
    // 构建国家列表（带中文名称）
    const countryNames: Record<string, string> = {
      'US': '美国',
      'CA': '加拿大',
      'GB': '英国',
      'DE': '德国',
      'FR': '法国',
      'JP': '日本',
      'CN': '中国'
    }
    
    countries.value = Array.from(countrySet).map(code => ({
      code,
      name: countryNames[code] || code
    }))
    
    // 默认选中第一个国家
    if (countries.value.length > 0) {
      selectedCountry.value = countries.value[0].code
    }
  } catch (error: any) {
    console.error('加载资源列表失败:', error)
  }
}
```

### 查询参数增强

```typescript
const loadCapacityData = async () => {
  const startDate = dayjs().format('YYYY-MM-DD')
  const endDate = dayjs().add(2, 'month').format('YYYY-MM-DD')
  
  // 构建查询参数
  let params: any = { start: startDate, end: endDate }
  
  if (resourceType.value === 'warehouse') {
    params.resourceType = 'warehouse'
    if (selectedWarehouseCode.value) {
      params.warehouseCode = selectedWarehouseCode.value
    }
  } else {
    params.resourceType = 'trucking'
    if (selectedTruckingCode.value) {
      params.truckingCompanyId = selectedTruckingCode.value
    }
  }
  
  // 添加国家参数（新增）
  if (selectedCountry.value) {
    params.country = selectedCountry.value
  }
  
  const queryString = Object.entries(params)
    .map(([key, value]) => `${key}=${encodeURIComponent(String(value))}`)
    .join('&')
  
  const response = await api.get(`/api/capacity/range?${queryString}`)

}
```
