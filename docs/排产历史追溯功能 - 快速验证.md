## 🐛 常见问题速查

### **Q1: 看不到📋图标？**

**A:** 货柜未排产
- 解决方法：先进行智能排产并确认保存

---

### **Q2: 点击报错？**

**A:** 组件导入问题
```bash
# 检查文件是否存在
ls frontend/src/components/SchedulingHistoryCard.vue

# 重启开发服务器
npm run dev
```

---

### **Q3: 内容为空？**

**A:** API 返回数据为空
```javascript
// F12 打开控制台，查看 Network 标签
// 查找请求：/api/v1/scheduling/history/xxx
// 检查响应数据
```

---

### **Q4: TypeScript 错误？**

**A:** 类型检查警告（不影响功能）
- 可以忽略，或添加 `// @ts-ignore` 注释
