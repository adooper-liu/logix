## Docker 部署

### 启动服务

```bash
# 启动数据库
docker-compose -f docker-compose.timescaledb.yml up -d

# 启动后端
docker build -t logix-backend ./backend
docker run -d -p 3001:3001 --env-file .env logix-backend

# 启动前端
docker build -t logix-frontend ./frontend
docker run -d -p 5173:80 logix-frontend
```

### Docker Compose

```yaml
# docker-compose.yml
version: '3.8'
services:
  db:
    image: timescale/timescaledb:latest-pg14
    environment:
      POSTGRES_USER: logix_user
      POSTGRES_PASSWORD: LogiX@2024!Secure
      POSTGRES_DB: logix_db
    ports:
      - "5432:5432"
    volumes:
      - pgdata:/var/lib/postgresql/data

  backend:
    build: ./backend
    ports:
      - "3001:3001"
    environment:
      - DB_HOST=db
    depends_on:
      - db

  frontend:
    build: ./frontend
    ports:
      - "5173:80"
    depends_on:
      - backend

volumes:
  pgdata:
```

```

```

响应:

```json

  "timestamp": "2024-01-01T00:00:00Z"

```

### 日志

```bash

# 查看前端日志

```

## 备份

### 数据库备份

```bash
# 备份
pg_dump -U logix_user logix_db > backup_$(date +%Y%m%d).sql

# 恢复
psql -U logix_user logix_db < backup_20240101.sql
```

## 运维命令

```bash
# 重启服务
docker-compose restart

# 查看状态
docker-compose ps

# 查看资源使用
docker stats

# 清理
docker system prune
```
