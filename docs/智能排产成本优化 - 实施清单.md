### 1. 修复仓库代码获取错误

**问题**:

- 前端传递 `warehouseCode: 'Bedford'`（仓库名称）
- 后端查找 `WHERE warehouseCode = 'Bedford'` 失败
- 实际数据库中是 `warehouseCode = 'UK-S005'`

**根因**:

- 排产数据存储：`plannedData.warehouseId = 'UK-S005'`（编码）
- 前端提取逻辑：从 `warehouseName.split(' ')` 得到 `'Bedford'`（名称）
- 没有复用排产数据的权威源

**修复**:

- ✅ 创建 `useCostOptimization.ts` Hook
- ✅ 实现 `extractOptimizeParams()` 函数
- ✅ 优先使用 `plannedData.warehouseId`（它实际就是 warehouseCode）
- ✅ 重构 `SchedulingPreviewModal.vue` 使用 Hook
- ✅ 重构 `SchedulingVisual.vue` 使用 Hook

**代码对比**:

```typescript

// 修复后 ✅
```

```typescript
async batchSchedule(request: ScheduleRequest): Promise<BatchScheduleResponse> {
  const results: ScheduleResult[] = []

  try {
    // ... 现有排产逻辑（第 144-147 行）
    const containers = await this.getContainersToSchedule(request)
    const sortedContainers = this.sortByClearanceDate(containers)

    // 分步排产（第 151-160 行）
    const skip = Math.max(0, request.skip ?? 0)
    const limit = request.limit != null ? request.limit : sortedContainers.length
    const toProcess = sortedContainers.slice(skip, skip + limit)

    // ✅ 新增：对每个排产结果进行成本优化探索
    for (const container of toProcess) {
      const result = await this.scheduleSingleContainer(container)

      if (result.success && result.plannedData) {
        // 调用成本优化服务
        const optimization = await this.costOptimizerService.suggestOptimalUnloadDate(
          result.containerNumber,
          warehouse,
          truckingCompany,
          new Date(result.plannedData.plannedPickupDate)
        )

        // 附加优化建议
        result.optimizationSuggestions = {
          originalCost: optimization.originalCost,
          optimizedCost: optimization.optimizedCost,
          savings: optimization.savings,
          suggestedPickupDate: optimization.suggestedPickupDate.toISOString().split('T')[0],
          shouldOptimize: optimization.savings > 0
        }
      }

      results.push(result)
    }

    return {
      results,
      hasMore: skip + limit < sortedContainers.length,
      totalOptimizationSavings: results.reduce(
        (sum, r) => sum + (r.optimizationSuggestions?.savings || 0),
        0
      )
    }
  } catch (error) {
    logger.error('[IntelligentScheduling] batchSchedule error:', error)
    throw error
  }
}
```

**文件**: `frontend/src/composables/__tests__/useCostOptimization.test.ts`

```typescript
import { describe, it, expect, vi } from "vitest";
import { extractOptimizeParams } from "../useCostOptimization";

describe("extractOptimizeParams", () => {
  it("应该成功从排产结果提取参数", () => {
    const mockResults = [
      {
        containerNumber: "MSKU1234567",
        success: true,
        plannedData: {
          warehouseId: "UK-S005",
          truckingCompanyId: "TRUCK-001",
          plannedPickupDate: "2026-03-30",
        },
      },
    ];

    const params = extractOptimizeParams(mockResults);
    expect(params).toEqual({
      containerNumbers: ["MSKU1234567"],
      warehouseCode: "UK-S005",
      truckingCompanyId: "TRUCK-001",
      basePickupDate: "2026-03-30",
    });
  });

  it("应该处理缺少仓库信息的情况", () => {
    const mockResults = [
      {
        containerNumber: "MSKU1234567",
        success: true,
        plannedData: {
          truckingCompanyId: "TRUCK-001",
          plannedPickupDate: "2026-03-30",
          // 缺少 warehouseId
        },
      },
    ];

    const params = extractOptimizeParams(mockResults);
    expect(params).toBeNull();
  });
});
```
