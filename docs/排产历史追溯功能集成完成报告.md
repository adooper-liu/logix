## 🔍 故障排查

### **问题 1: 看不到📋按钮**

**原因：** 货柜未排产（`scheduleStatus !== 'issued'`）

**解决方案：**
1. 选择待排产的货柜
2. 点击"一键排产"
3. 确认保存
4. 刷新列表

---

### **问题 2: 点击后报错 "Cannot find module"**

**原因：** 组件未正确导入

**解决方案：**
```bash
# 检查组件文件是否存在
ls frontend/src/components/SchedulingHistoryCard.vue

# 重启开发服务器
npm run dev
```

---

### **问题 3: 抽屉显示但内容为空**

**原因：** API 返回数据为空

**调试步骤：**
```javascript
// 1. 打开浏览器控制台
F12 → Console

// 2. 查看网络请求
Network → 查找 /scheduling/history/xxx

// 3. 检查响应
// 如果返回 404，说明后端路由未配置
// 如果返回空数组，说明该货柜没有历史记录
```
