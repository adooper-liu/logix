# 免费天数显示问题修复

## 📋 问题描述

**现象**：排产预览页面中，最晚提柜日（LFD）和最晚还箱日（LRD）的 Tooltip 显示"免费天数：-"

**影响**：用户无法查看免费天数信息，影响对免费期的理解和决策

## 📝 相关知识点

### 免费天数计算规则

**提柜免费天数**：
- 计算公式：`MIN(滞港费免费天数，堆存费免费天数，D&D 免费天数)`
- 用途：计算最晚提柜日（LFD）
- 起算日：卸柜日（ATA/ETA/实际卸船日）

**还箱免费天数**：
- 计算公式：`D&D 免费天数 或 滞箱费免费天数`
- 用途：计算最晚还箱日（LRD）
- 起算日：实际提柜日

### 数据来源

- **滞港费标准表**：`ext_demurrage_standards`
- **字段**：`free_days`、`tiers`（阶梯费率）
- **推导规则**：从第 1 天起连续费率为 0 视为免费；首个费率>0 的日期表示前 d-1 天为免费期

详见：
- `frontend/src/utils/demurrageTiers.ts`
- `backend/src/utils/demurrageTiers.ts`
