  - 智能体规则中已有明确定义

##### ⚠️ **需要评估：01-代码规范.md**

- **大小**: 3KB
- **内容**:
  - 核心原则（数据库优先、数据完整性、日期口径统一）
  - 命名规则速查表
  - 后端/前端规范

  - ✅ **部分内容重叠**: 命名规则在 SKILL 原则中有提及
  - ✅ **独特价值**: 数据库优先原则、前后端目录结构是独特的
- **建议**: **保留**，但可以精简

##### ⚠️ **需要评估：02-命名规范.md**

- **大小**: 3.5KB
- **内容**:
  - 详细的命名对照表
  - 数据库表/字段命名规则
  - API 命名规范
  - 前端组件命名

  - ✅ **部分重叠**: SKILL 原则中有命名示例
  - ✅ **独特价值**: 更详细的命名速查表，便于快速查找
- **建议**: **保留**，作为速查手册

#### 2. 00-getting-started/ 目录

##### ✅ **建议保留：入门指南系列**

- **01-快速开始.md** (1.3KB) - 新人快速启动

**共同特点**:
- ✅ 内容不与 SKILL 重复
