### 4. 关键问题验证

#### ✅ 问题1: 同一提单货柜时间不一致（已修复）

**问题**: 提单 `NGP3069047` 下，ECMU5381817的ATA为2026/02/11，其他货柜为2026/01/01

**原因**: 代码优先使用 `latestPortOperation?.ataDestPort` 而非海运单时间

**修复**: 修改优先级为 `container.seaFreight?.ata || latestPortOperation?.ataDestPort`

**验证**: ✅ 已修复，确保同一提单所有货柜共享海运单时间

#### ✅ 问题2: LFD验证逻辑（已实施）

**规则**: LFD必须 >= ATA目的港到达日期

**代码**:
```typescript
private validateLFD(lastFreeDate: Date, ataDestPort: Date): boolean {
  return lastFreeDate.getTime() >= ataDestPort.getTime();
}
```

**示例**:
```
ATA = 2025/08/22 00:00:00
PLFD = 2025/08/21 00:00:00 (无效，拒绝更新)
```

**验证**: ✅ 正确拒绝无效LFD

## 🔧 快速排查指南

### 问题1: 导入失败

**症状**: Excel导入时报错

**排查步骤**:
```bash
# 1. 查看错误日志
grep "[FeituoImport]" logs/app.log

# 2. 检查批次记录
SELECT * FROM ext_feituo_import_batch ORDER BY id DESC LIMIT 1;

# 3. 查看原始数据
SELECT * FROM ext_feituo_import_table1 WHERE batch_id = ?;
```

**常见原因**:
- 缺少集装箱号
- 日期格式错误
- 状态码未定义

### 问题2: 数据不一致

**症状**: 同一提单不同货柜时间不同

**排查步骤**:
```sql
-- 1. 检查海运单时间
SELECT bill_of_lading_number, ata, eta FROM process_sea_freight 
WHERE bill_of_lading_number = 'NGP3069047';

-- 2. 检查港口操作时间
SELECT container_number, ata_dest_port, port_type 
FROM process_port_operations 
WHERE container_number IN ('ECMU5400183', 'ECMU5381817');

-- 3. 检查状态事件
SELECT container_number, status_code, occurred_at, is_estimated 
FROM ext_container_status_events 
WHERE container_number IN ('ECMU5400183', 'ECMU5381817')
ORDER BY occurred_at;
```

**解决方案**:
- 确保代码使用海运单时间作为第一优先级
- 检查状态事件是否正确同步

### 问题3: LFD验证失败

**症状**: LFD不更新或日志警告

**排查步骤**:
```bash
# 查看LFD验证日志
grep "LFD.*<.*ATA" logs/app.log

# 查询当前值
SELECT container_number, ata_dest_port, last_free_date, last_free_date_invalid
FROM process_port_operations 
WHERE container_number = 'TCKU1559530';
```

**常见原因**:
- LFD日期早于ATA（无效数据）
- ATA为空（无法验证）
- 时区问题导致日期比较错误
