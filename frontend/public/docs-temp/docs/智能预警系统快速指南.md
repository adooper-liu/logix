# 智能预警系统快速使用指南

## 🎯 功能简介

智能预警系统会自动识别以下两类风险货柜：

| 预警类型     | 标识颜色                 | 触发条件                   |
| ------------ | ------------------------ | -------------------------- |
| **即将超期** | 🟠 橙色边框 + 脉冲动画   | 距离最晚提柜日期 ≤ 3 天    |
| **逾期未提** | 🔴 红色边框 + 强脉冲动画 | 已超过最晚提柜日期且未提柜 |

## 🔍 预警规则说明

### 即将超期（Warning）

**触发条件**：

- 当前日期 ≤ 最晚提柜日期
- 最晚提柜日期 - 当前日期 ≤ 3 天

**示例**：

- 今天是 3 月 15 日
- 最晚提柜日期是 3 月 17 日
- 相差 2 天 → **触发预警** ✅

### 逾期未提（Danger）

**触发条件**：

- 当前日期 > 最晚提柜日期
- 物流状态 ≠ "已提柜"、"已卸柜"、"已还箱"

**示例**：

- 今天是 3 月 20 日
- 最晚提柜日期是 3 月 18 日
- 状态仍是"已到港" → **触发预警** ✅

**不触发预警的情况**：

- 虽然超期，但已经提柜 → 不预警
- 没有设置最晚提柜日期 → 不预警

## 🛠️ 如何处理预警货柜

### 步骤 1：查看详情

点击预警货柜的圆点 → 右侧弹出详细信息侧边栏

### 步骤 2：分析原因

检查以下信息：

- 为什么还没提柜？
- 是否已经安排拖车？
- 是否有其他延误因素？

### 步骤 3：采取行动

右键点击货柜圆点，可选择：

- **查看明细**：完整信息
- **编辑日期**：调整计划提柜日期
- **复制柜号**：快速分享给相关人员

### 步骤 4：更新状态

完成提柜后，及时更新物流状态，预警会自动消失。

## 💡 实用技巧

### 技巧 1：快速定位

在搜索框输入柜号或提单号，可以快速找到特定货柜。

### 技巧 2：按目的港分组

每个目的港的货柜分在一组，方便批量查看。

### 技巧 3：使用筛选

点击顶部筛选按钮，只看特定状态的货柜：

- 勾选"已到港" → 重点关注在港货柜
- 取消勾选"已还箱" → 排除已完成的货柜

### 技巧 4：导出数据

点击右上角"导出"按钮，将当前视图导出为 Excel，包含所有预警信息。

## ❓ 常见问题

### Q1: 为什么有些货柜没有最晚提柜日期？

A: 可能是因为：

- 船公司还未提供该信息
- 数据导入时遗漏
- 某些特殊运输方式不需要

### Q2: 预警多久刷新一次？

A:

- 页面加载时自动计算
- 手动刷新数据后重新计算
- 每天首次打开页面会重新计算

### Q3: 已提柜的货柜为什么还有预警？

A: 这可能是数据同步延迟，请：

1. 确认物流状态已正确更新
2. 点击"刷新"按钮重新加载数据
3. 如仍有问题，联系技术支持

### Q4: 能否同时显示多个预警？

A: 可以！如果一个货柜同时满足多个预警条件，会：

- 显示所有预警信息
- 按最高级别设置边框颜色
- Tooltip 中列出所有预警
