## 四、替换示例

### 后端

```typescript
// 替换前
const today = DateFilterBuilder.toDayStart(new Date());
const todayStr = today.toISOString().split('T')[0];

// 替换后
import { DateTimeUtils } from '../utils/dateTimeUtils';
const today = DateTimeUtils.now();
const todayStr = DateTimeUtils.format(today, 'YYYY-MM-DD');
```

### 前端

```typescript
// 替换前
params.startDate = dayjs(shipmentDateRange.value[0]).format('YYYY-MM-DD')

// 替换后
import { DateTimeUtils } from '@/utils/dateTimeUtils'
params.startDate = DateTimeUtils.format(shipmentDateRange.value[0], 'YYYY-MM-DD')
```

## 五、注意事项

1. **DateFilterBuilder.toDayStart**：若需保留「当天 00:00:00」语义，可用 `DateTimeUtils.format(date, 'YYYY-MM-DD')` 再 `DateTimeUtils.parse`，或扩展 `DateTimeUtils` 增加 `toDayStart`。
2. **Excel 导入**：`parseLocalDate` 采用「按本地时间解析」策略，与 UTC 规格机不同，建议保持独立，不强制替换。
3. **渐进迁移**：建议按 P0 → P1 → P2 顺序，每批替换后做回归测试。
