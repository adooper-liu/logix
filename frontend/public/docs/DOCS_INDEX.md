## 📝 文档维护规范

1. **新增文档**：按分类放入相应目录，使用编号命名
2. **更新文档**：直接编辑文档，保留历史记录
3. **删除文档**：删除前确认是否还有价值，重要文档归档到 `docs-temp/`
4. **临时文档**：放到 `docs-temp/` 对应分类，定期清理
5. **文档索引**：每次文档变更后更新本索引文件
