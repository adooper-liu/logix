# 快速开始

5分钟启动 LogiX 开发环境。

## 前置要求

- Node.js 18+
- PostgreSQL 14+ (或 Docker)
- Git

## 启动步骤

### 1. 克隆项目

```bash
git clone <repository-url>
cd logix
```

### 2. 启动数据库

```bash
# 方式一：Docker（推荐）
docker-compose -f docker-compose.timescaledb.yml up -d

# 方式二：本地 PostgreSQL
# 创建数据库 logix_db，用户 logix_user，密码 LogiX@2024!Secure
```

### 3. 安装依赖

```bash
npm install
cd backend && npm install
cd ../frontend && npm install
```

### 4. 初始化数据库

```bash
cd backend
npm run init-db
```

### 5. 启动服务

```bash
# 终端一：后端
cd backend && npm run dev

# 终端二：前端
cd frontend && npm run dev
```

### 6. 访问

- 前端: http://localhost:5173
- 后端: http://localhost:3001

## 常用命令

```bash
# 后端
npm run dev          # 开发模式
npm run build        # 构建
npm run init-db      # 初始化数据库

# 前端
npm run dev          # 开发模式
npm run build        # 构建
npm run lint         # 代码检查
```
