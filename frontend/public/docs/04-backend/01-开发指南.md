# 后端开发指南

LogiX 后端开发规范与实践。

## 目录结构

```
backend/src/
├── adapters/           # 外部数据适配器
│   └── feituo/        # 飞驼数据适配器
├── controllers/       # API 控制器
├── services/         # 业务逻辑层
├── entities/         # TypeORM 实体
├── routes/           # 路由定义
├── middleware/       # 中间件
├── constants/        # 常量定义
├── utils/            # 工具函数
├── schedulers/       # 定时任务
├── database/        # 数据库配置
├── interfaces/       # 类型接口
└── app.ts           # 应用入口
```

## 开发流程

### 1. 新增功能

```bash
# 1. 创建数据库迁移脚本
# 文件: migrations/add_xxx.sql

# 2. 创建/更新实体
# 文件: backend/src/entities/Xxx.ts

# 3. 创建 Service
# 文件: backend/src/services/xxx.service.ts

# 4. 创建 Controller
# 文件: backend/src/controllers/xxx.controller.ts

# 5. 注册路由
# 文件: backend/src/routes/xxx.routes.ts

# 6. 测试 API
```

### 2. 修改功能

```bash
# 1. 创建迁移脚本（如需修改表结构）
# 2. 更新实体
# 3. 修改 Service 逻辑
# 4. 修改 Controller
# 5. 测试验证
```

## Controller 层

### 规范

- 只处理请求/响应，不包含业务逻辑
- 参数验证
- 调用 Service 方法
- 统一响应格式

### 示例

```typescript
// controllers/container.controller.ts
@Controller('/containers')
export class ContainerController {
  constructor(
    private readonly containerService: ContainerService
  ) {}

  @Get('/')
  async getList(
    @Query('page') page: string,
    @Query('limit') limit: string,
    @Req() req: Request
  ) {
    const containers = await this.containerService.getContainers({
      page: parseInt(page) || 1,
      limit: parseInt(limit) || 20,
      countryCode: req.headers['x-country-code'] as string
    });

    return { success: true, data: containers };
  }

  @Get('/:id')
  async getById(@Param('id') id: string) {
    const container = await this.containerService.getById(id);
    if (!container) {
      throw new NotFoundError('货柜不存在');
    }
    return { success: true, data: container };
  }
}
```

## Service 层

### 规范

- 业务逻辑处理
- 不直接处理 HTTP
- 数据转换
- 事务管理

### 示例

```typescript
// services/container.service.ts
@Service()
export class ContainerService {
  constructor(
    @InjectRepository(Container)
    private readonly containerRepository: Repository<Container>
  ) {}

  async getContainers(query: QueryParams): Promise<ContainerList> {
    const { page, limit, countryCode } = query;

    const qb = this.containerRepository.createQueryBuilder('c');

    // 添加筛选条件
    if (countryCode) {
      qb.innerJoin('c.replenishmentOrder', 'o')
        .where('o.sellToCountry = :country', { country: countryCode });
    }

    // 分页
    qb.skip((page - 1) * limit).take(limit);

    const [data, total] = await qb.getManyAndCount();

    return { data, total, page, limit };
  }
}
```

## 常用工具

### 数据库查询

```typescript
// 使用 Repository
const containers = await containerRepository.find({
  where: { logisticsStatus: 'at_port' },
  relations: ['seaFreight'],
  order: { etaDestPort: 'ASC' }
});

// 使用 QueryBuilder
const result = await containerRepository
  .createQueryBuilder('c')
  .innerJoin('c.portOperations', 'po')
  .where('po.portType = :type', { type: 'destination' })
  .andWhere('po.ataDestPort IS NOT NULL')
  .getMany();
```

### 响应格式

```typescript
// 成功
return { success: true, data: result };

// 分页
return { success: true, data: { list, total, page, limit } };

// 错误
throw new AppError('错误信息', 'ERROR_CODE');
```
