# 系统架构

LogiX 系统架构设计。

## 技术栈

| 层级 | 技术 | 说明 |
|------|------|------|
| 前端 | Vue 3 + TypeScript | 组合式 API |
| UI 组件 | Element Plus | Vue 3 组件库 |
| 后端 | Express + TypeORM | Node.js 框架 |
| 数据库 | PostgreSQL + TimescaleDB | 主数据库 + 时序数据 |
| 缓存 | Redis | 会话/缓存 |
| 部署 | Docker | 容器化 |

## 数据流

### 货柜查询流程

```
1. 前端请求
   GET /api/v1/containers?page=1&limit=20

2. Controller 接收
   ContainerController.getList()

3. Service 处理
   ContainerService.getContainers()

4. Repository 查询
   ContainerRepository.find()

5. 返回数据
   { success: true, data: {...} }

6. 前端渲染
   展示货柜列表
```

### 数据导入流程

```
1. 用户上传 Excel
   POST /api/v1/import/excel

2. 解析文件
   ExcelImportService.parse()

3. 数据验证
   ValidationService.validate()

4. 映射字段
   MappingService.mapToEntity()

5. 写入数据库
   Repository.save()

6. 返回结果
   { success: true, imported: 100, errors: [] }
```
