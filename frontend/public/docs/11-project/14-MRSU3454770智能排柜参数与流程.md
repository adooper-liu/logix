## 四、智能排柜流程（按 MRSU3454770 实际数据）

```
1. 过滤：schedule_status = 'initial' ✓
2. 目的港操作：port_type = 'destination' ✓ (USSAV 萨凡纳)
3. 清关可放行日：ata_dest_port || eta_dest_port = 2026-02-16 23:00 ✓
4. 计划清关日 = 2026-02-16 23:00
5. 计划提柜日 = 计划清关日 + 1 天 = 2026-02-17；last_free_date 为 null，无截断
6. 候选仓库：US-S005、US016、US018（3 个）
7. 计划卸柜日 = findEarliestAvailableDay(仓库, 2026-02-17)，取各仓库日产能最早可用日
8. 计划送仓日 = 卸柜日或前一日 (Drop off)
9. 计划还箱日 = 卸柜后 + 1 天，last_return_date 为 null 则无上限
10. 车队选择：按 仓库×港口 映射，取有剩余能力的车队（从 US-S005、US016、US018 对应的 warehouse_trucking_mapping 中选）
11. 清关行：美国清关行（占位符）
12. 写入 process_trucking_transport、process_warehouse_operations、process_port_operations
```

## 五、数据缺口与影响

| 缺口 | 影响 |
|------|------|
| customer_code = null | 依赖 customer_name = sell_to_country 回退；若 customer_name 变更需维护 |
| last_free_date = null | 计划提柜日无免费期约束，可能超期产生滞港费 |
| last_return_date = null | 计划还箱日无上限，fallback 用 last_free_date + 7 天计算 |
| dict_customs_brokers 无 US | 使用「美国清关行」占位符 |
