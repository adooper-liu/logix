## 一、模块概述

货柜管理模块是 LogiX 系统的核心，负责货柜的全生命周期管理，包括查询、详情展示、状态跟踪、物流路径可视化等功能。

### 核心功能

- 货柜列表查询与筛选
- 货柜详情展示
- 物流路径可视化
- 关键日期时间线
- 流程节点管理（港口、拖车、仓库）
- 预警与风险管理

## 三、API 接口

### 3.1 API 路由定义

**路由文件**: `backend/src/routes/container.routes.ts`

```typescript
// 集装箱管理路由
router.get('/', containerController.getContainers) // 获取货柜列表
router.get('/:id', containerController.getContainerById) // 获取货柜详情
router.post('/', containerController.createContainer) // 创建货柜
router.put('/:containerNumber', containerController.updateContainer) // 更新货柜
router.delete('/:containerNumber', containerController.deleteContainer) // 删除货柜
router.get('/statistics', containerController.getStatistics) // 统计
router.get('/statistics-detailed', containerController.getStatisticsDetailed) // 详细统计
router.post('/:containerNumber/update-status', containerController.updateStatus) // 更新状态
```

## 五、前端服务

### 5.1 Service 层

**文件**: `frontend/src/services/container.ts`

```typescript
class ContainerService {
  /**
   * 获取货柜列表
   */
  async getContainers(params: any) {
    const response = await this.api.get('/containers', { params })
    return response.data
  }

  /**
   * 获取货柜详情
   */
  async getContainerById(id: string) {
    const response = await this.api.get(`/containers/${id}`)
    return response.data
  }

  /**
   * 获取物流路径
   */
  async getLogisticsPath(containerNumber: string) {
    const response = await this.api.get(`/logistics-path/container/${containerNumber}`)
    return response.data
  }
}
```
