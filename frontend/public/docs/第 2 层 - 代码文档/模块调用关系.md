## 三、前后端调用关系

### 3.1 API 调用矩阵

| 前端页面               | 调用 API                            | 后端 Controller        | 后端 Service                   |
| ---------------------- | ----------------------------------- | ---------------------- | ------------------------------ |
| Login.vue              | POST /auth/login                    | AuthController         | AuthService                    |
| Shipments.vue          | GET /containers                     | ContainerController    | ContainerService               |
| Shipments.vue          | GET /containers/:id                 | ContainerController    | ContainerService               |
| ContainerDetail        | GET /demurrage/calculate/:id        | DemurrageController    | DemurrageService               |
| ContainerDetail        | GET /containers/:id/logistics-path  | ContainerController    | LogisticsPathService           |
| SchedulingVisual       | POST /scheduling/batch-schedule     | SchedulingController   | IntelligentSchedulingService   |
| SchedulingVisual       | POST /scheduling/confirm            | SchedulingController   | IntelligentSchedulingService   |
| SchedulingVisual       | POST /scheduling/optimize-cost      | SchedulingController   | SchedulingCostOptimizerService |
| SchedulingVisual       | POST /scheduling/batch-optimize     | SchedulingController   | SchedulingCostOptimizerService |
| DemurrageTopContainers | GET /demurrage/calculate/:id        | DemurrageController    | DemurrageService               |
| DemurrageTopContainers | GET /demurrage/standards            | DemurrageController    | -                              |
| DemurrageTopContainers | GET /demurrage/summary              | DemurrageController    | DemurrageService               |
| DemurrageTopContainers | GET /demurrage/top-containers       | DemurrageController    | DemurrageService               |
| Monitoring             | GET /monitoring/status-distribution | MonitoringController   | StatusDistributionService      |
| Import                 | POST /import/excel                  | ImportController       | ImportService                  |
| Import                 | POST /import/feituo                 | ExternalDataController | FeituoImportService            |

## 六、模块边界

| 模块     | 职责                           | 边界                                                                                  |
| -------- | ------------------------------ | ------------------------------------------------------------------------------------- |
| 货柜管理 | 货柜 CRUD、查询、状态更新      | 读写 biz*containers；只读 process*\_、ext\_\_                                         |
| 排产     | 排产算法、成本优化、资源管理   | 读写 process_trucking_transport, process_warehouse_operations, dict_scheduling_config |
| 滞港费   | 费用计算、标准管理、写回免费期 | 只读 process\_\*、biz_containers；读写 ext_demurrage_records、ext_demurrage_standards |
| 统计     | 数据统计、指标计算             | 只读各类 Repository；不写操作                                                         |
| 数据导入 | 外部数据处理（Excel、飞驼）    | 写所有业务表（biz_containers、process_sea_freight、process_port_operations 等）       |
| 定时任务 | 状态同步、费用写回             | 批量读写；只更新必要字段                                                              |
| 物流路径 | 路径可视化、节点查询           | 只读；不写操作                                                                        |
| 预警     | 预警规则管理、预警生成         | 读写 ext_container_alerts；只读其他表                                                 |
| 风险     | 风险评估、评分计算             | 读写 ext_container_risk_assessments；只读其他表                                       |
