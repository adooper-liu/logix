## 七、组件通信模式

### 7.1 Props/Events 模式

```vue
<!-- 父组件 -->
<ChildComponent :prop="value" @event="handler" />

<!-- 子组件 -->
<script setup>
defineProps<{ prop: string }>()
defineEmits(['event'])
</script>
```

### 7.2 Pinia Store 模式

```typescript
// store
const useContainerStore = defineStore('container', () => { ... })

// 组件中使用
const containerStore = useContainerStore()
```

### 7.3 Provide/Inject 模式

```typescript
// 父组件
provide('theme', 'dark')

// 子组件
const theme = inject('theme')
```
