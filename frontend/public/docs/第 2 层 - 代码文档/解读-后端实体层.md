## 七、实体装饰器

```typescript
@Entity('biz_containers')
@Index(['containerNumber'], { unique: true })
export class Container {
  @PrimaryColumn({ name: 'container_number' })
  containerNumber: string;

  @Column({ name: 'order_number', nullable: true })
  orderNumber: string;

  @ManyToOne(() => ReplenishmentOrder, order => order.containers)
  @JoinColumn({ name: 'order_number' })
  replenishmentOrder: ReplenishmentOrder;

  @Column({ name: 'logistics_status', length: 50 })
  logisticsStatus: string;

  @CreateDateColumn()
  createdAt: Date;

  @UpdateDateColumn()
  updatedAt: Date;
}
```

## 八、常用装饰器

| 装饰器 | 作用 |
|--------|------|
| @Entity(name) | 定义实体对应表名 |
| @PrimaryColumn | 主键列 |
| @Column | 数据列 |
| @ManyToOne | 多对一关系 |
| @OneToMany | 一对多关系 |
| @OneToOne | 一对一关系 |
| @JoinColumn | 关联列 |
| @Index | 索引 |
| @CreateDateColumn | 创建时间 |
| @UpdateDateColumn | 更新时间 |

## 九、命名转换策略

| 数据库 (snake_case) | 实体 (camelCase) |
|---------------------|------------------|
| container_number | containerNumber |
| order_number | orderNumber |
| logistics_status | logisticsStatus |
| eta_dest_port | etaDestPort |
| ata_dest_port | ataDestPort |

**注意**: 已在所有实体中显式指定 `@Column({ name: 'snake_case' })` 避免转换错误。
