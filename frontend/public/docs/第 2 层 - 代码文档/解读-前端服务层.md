## 六、服务调用流程

```
Vue 组件
    ↓
Composables (逻辑复用)
  useContainers / useScheduling / useOptimization
    ↓
Services (API封装)
  containerService / demurrageService / schedulingService
    ↓
HTTP Client (Axios)
  httpClient.ts - 请求拦截 / 响应拦截
    ↓
后端 API
```

---

## 七、典型调用示例

```typescript
// 1. 组件中调用 composable
const { containers, loading, fetchContainers } = useContainers()

// 2. Composable 内部调用 service
import { containerService } from '@/services'
const containers = await containerService.getContainers(params)

// 3. Service 内部调用 httpClient
import { api } from './api'
const response = await api.get('/containers', { params })
```

## 八、类型安全

| 层级           | 机制                           |
| -------------- | ------------------------------ |
| **API响应**    | 使用 `ApiResponse<T>` 统一包装 |
| **Service**    | 方法签名定义明确返回类型       |
| **Composable** | 使用 TypeScript泛型            |
| **组件**       | defineProps<T>() 声明props类型 |
