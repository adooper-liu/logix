## 四、费用计算逻辑

### 4.1 滞港费计算 (Demurrage)

```
计算公式:
  chargeDays = 实际占用天数 - 免费天数
  amount = chargeDays × 日费率

参数来源:
  ├── 实际占用天数 = 计划提柜日 - 到港日 (或) 实际提柜日 - 到港日
  ├── 免费天数 = 从费用标准获取
  └── 日费率 = 从费用标准获取 (按到港日或卸船日)

特殊处理:
  ├── 免费天数基准 = 工作日/自然日
  ├── 阶梯费率 = 第一天/第二天...第N天不同费率
  └── 合并计算 = 滞港费 + 滞箱费 可合并
```

### 4.2 滞箱费计算 (Detention)

```
计算公式:
  chargeDays = 还箱日 - 提柜日 - 免费天数
  amount = chargeDays × 日费率

参数来源:
  ├── 提柜日 = 实际提柜日 或 计划提柜日
  ├── 还箱日 = 实际还箱日 或 计划还箱日
  └── 免费天数 = 从费用标准获取
```

### 4.3 存储费计算 (Storage)

```
计算公式:
  chargeDays = 出库日 - 入库日
  amount = chargeDays × 日费率

适用场景:
  ├── 港口存储 (在港期间)
  └── 堆场存储 (Drop off模式)
```

## 五、标准匹配逻辑

### 5.1 匹配条件

```
匹配优先级 (从高到低):
1. 海外公司 + 目的港 + 船公司 + 货代
2. 海外公司 + 目的港 + 船公司
3. 海外公司 + 目的港
4. 默认标准

匹配参数:
├── overseasCompanyCode (海外公司)
├── destinationPortCode (目的港)
├── shippingCompanyCode (船公司)
└── freightForwarderCode (货代)
```

### 5.2 匹配流程

```
输入: containerNumber

1. 获取货柜信息
   ├── destination_country
   ├── destination_port
   ├── shipping_company
   └── freight_forwarder

2. 查询匹配标准
   SELECT * FROM ext_demurrage_standards
   WHERE overseas_company_code = ?
     AND destination_port_code = ?
     AND shipping_company_code = ?
     AND freight_forwarder_code = ?
     AND effective_date <= CURRENT_DATE
     AND (expiry_date IS NULL OR expiry_date >= CURRENT_DATE)
   ORDER BY sequence_number ASC
   LIMIT 1

3. 如果未匹配，尝试宽松匹配
   └── 逐步减少匹配条件

4. 返回匹配的标准
```

## 六、预测模式 (Forecast)

### 6.1 预测模式条件

```
触发条件:
├── 当前日期 < 计划提柜日
└── 货柜未实际提柜

使用参数:
├── plannedPickupDate (计划提柜日)
├── plannedUnloadDate (计划卸柜日)
└── plannedReturnDate (计划还箱日)
```

### 6.2 预测 vs 实际 对比

| 维度 | Actual | Forecast |
|------|--------|----------|
| 到港日 | ATA (实际) | ETA / 修正ETA |
| 提柜日 | 实际提柜日 | 计划提柜日 |
| 还箱日 | 实际还箱日 | 计划还箱日 |
| 免费日 | 计算写入 | 预测不写库 |

## 十、常见问题

### 10.1 费用为0

| 原因 | 解决方案 |
|------|----------|
| 费用标准未配置 | 检查 ext_demurrage_standards |
| 免费天数未设置 | 检查 free_days 字段 |
| 日期未到 | 预测模式费用为0正常 |

### 10.2 费用计算异常

| 原因 | 解决方案 |
|------|----------|
| 日期格式错误 | 检查 toDateOnly 转换 |
| 时区问题 | 使用 UTC 方法 |
| 免费天数基准错误 | 检查 free_days_basis |
