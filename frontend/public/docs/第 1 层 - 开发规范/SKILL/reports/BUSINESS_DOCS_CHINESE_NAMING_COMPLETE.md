## 命名规范

### 中文命名优势 ✅

| 优势           | 说明                       |
| -------------- | -------------------------- |
| **直观易懂**   | 中文名称一目了然，无需翻译 |
| **查找高效**   | 中文关键词更易搜索和记忆   |
| **符合 SKILL** | 简洁即美，真实第一         |
| **智能体友好** | AI 能准确理解文档内容      |
| **团队友好**   | 降低理解成本，提升协作效率 |

### 编号规则

- **01-06**: 核心业务文档（状态机、统计、滞港费、货柜、甘特图、排柜）
- **07-26**: 深度业务文档（计算模式、滞港费逻辑、前端适配等）
- **无编号**: 补充文档（货柜管理模块、滞港费模块）
