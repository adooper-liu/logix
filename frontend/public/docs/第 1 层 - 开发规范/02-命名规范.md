# 命名规范

LogiX 项目命名规则速查。

## 命名对照表

| 类型 | 规则 | 示例 |
|------|------|------|
| 数据库表 | snake_case | biz_containers |
| 数据库字段 | snake_case | container_number |
| 数据库索引 | snake_case | idx_containers_number |
| 实体类 | PascalCase | Container |
| 实体属性 | camelCase | containerNumber |
| 实体文件 | PascalCase.ts | Container.ts |
| Controller 文件 | PascalCase.controller.ts | ContainerController.ts |
| Service 文件 | PascalCase.service.ts | ContainerService.ts |
| 路由文件 | kebab-case.routes.ts | container.routes.ts |
| API 路径 | kebab-case | /containers/:id |
| 前端组件 | PascalCase.vue | ContainerList.vue |
| 组件目录 | PascalCase/ | ContainerList/ |
| 组合式函数 | usePascalCase.ts | useContainerData.ts |
| CSS 类名 | kebab-case | .container-list |
| 常量 | UPPER_SNAKE_CASE | DEFAULT_PAGE_SIZE |
| 枚举 | PascalCase | LogisticStatus |
| 枚举值 | UPPER_SNAKE_CASE | NOT_SHIPPED |


## API 命名

### RESTful 规范

```
GET    /containers           # 列表
GET    /containers/:id       # 详情
POST   /containers           # 创建
PATCH  /containers/:id       # 更新
DELETE /containers/:id       # 删除
```

### 统计 API

```
GET /containers/statistics/arrival   # 按到港统计
GET /containers/statistics/eta      # 按ETA统计
GET /containers/statistics/planned  # 按计划提柜统计
```

## 颜色变量

必须使用 SCSS 变量，禁止硬编码颜色。

```scss
// 正确
color: $primary-color;
background: $success-color;

// 错误
color: #409EFF;
background: #67C23A;
```

### 常用变量

| 变量 | 用途 |
|------|------|
| $primary-color | 主色调 |
| $success-color | 成功色 |
| $warning-color | 警告色 |
| $danger-color | 危险色 |
| $text-primary | 主要文字 |
| $text-regular | 常规文字 |
