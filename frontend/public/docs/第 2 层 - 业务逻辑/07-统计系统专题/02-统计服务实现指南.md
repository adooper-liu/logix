## 📋 概述

统计服务是 LogiX 物流系统的数据分析核心，提供多维度货柜统计功能。本文档详细介绍 5 大统计服务的实现原理和使用方法。

### 核心组件

1. **ContainerStatisticsService** - 统一入口，委托子服务处理
2. **ArrivalStatisticsService** - 按到港统计
3. **EtaStatisticsService** - 按 ETA 统计
4. **PlannedPickupStatisticsService** - 按计划提柜统计
5. **LastPickupStatisticsService** - 按最晚提柜统计
6. **LastReturnStatisticsService** - 按最晚还箱统计

## 一、API 接口

### 1.1 获取统计概览

**端点**: `GET /api/v1/containers/statistics`

**请求参数**:

```typescript
interface StatisticsQuery {
  startDate?: string // 开始日期 (YYYY-MM-DD)
  endDate?: string // 结束日期 (YYYY-MM-DD)
  filterCondition?: string // 筛选条件维度
}
```

**响应格式**:

```typescript
interface StatisticsResponse {
  arrivalToday: number // 今日到港
  arrivedBeforeTodayNotPickedUp: number // 之前未提柜
  arrivedBeforeTodayPickedUp: number // 之前已提柜
  transitOverdue: number // 中转逾期
  transitWithin3Days: number // 中转 3 日内
  transitWithin7Days: number // 中转 7 日内
  transitOver7Days: number // 中转 7 日后
  overdue: number // ETA 逾期
  within3Days: number // ETA 3 日内
  within7Days: number // ETA 7 日内
  over7Days: number // ETA 7 日后
  overduePlanned: number // 计划提柜逾期
  todayPlanned: number // 今日计划提柜
  plannedWithin3Days: number // 计划提柜 3 日内
  plannedWithin7Days: number // 计划提柜 7 日内
  expired: number // 最晚提柜已超期
  urgent: number // 最晚提柜即将超期
  warning: number // 最晚提柜预警
  normal: number // 最晚提柜正常
  returnExpired: number // 最晚还箱已超期
  returnUrgent: number // 最晚还箱即将超期
  returnWarning: number // 最晚还箱预警
  returnNormal: number // 最晚还箱正常
}
```

**使用示例**:

```typescript
// TypeScript 调用
const response = await fetch(
  '/api/v1/containers/statistics?startDate=2024-01-01&endDate=2024-01-31'
)
const statistics: StatisticsResponse = await response.json()

console.log('今日到港:', statistics.arrivalToday)
console.log('即将超期:', statistics.urgent)
```

```typescript
  async getArrivalDistribution(
    startDate?: string,
    endDate?: string
  ): Promise<Record<string, number>>

  // 获取按计划提柜分布
  async getPlannedPickupDistribution(
    startDate?: string,
    endDate?: string
  ): Promise<Record<string, number>>

  // 获取按最晚提柜分布
  async getLastPickupDistribution(
    startDate?: string,
    endDate?: string
  ): Promise<Record<string, number>>

  // 获取按最晚还箱分布
  async getLastReturnDistribution(
    startDate?: string,
    endDate?: string
  ): Promise<Record<string, number>>
}
```

**实现逻辑**:

```typescript
async getArrivalDistribution(startDate?: string, endDate?: string) {
  // 1. 并行调用到港统计和 ETA 统计
  const [arrivalDist, etaDist] = await Promise.all([
    this.arrivalStatistics.getDistribution(startDate, endDate),
    this.etaStatistics.getDistribution(startDate, endDate)
  ]);

  // 2. 合并结果
  return {
    // 已到目的港
    arrivalToday: arrivalDist.today,
    arrivedBeforeTodayNotPickedUp: arrivalDist.beforeTodayNotPickedUp,
    arrivedBeforeTodayPickedUp: arrivalDist.beforeTodayPickedUp,

    // 已到中转港
    transitOverdue: arrivalDist.transitOverdue,
    transitWithin3Days: arrivalDist.transitWithin3Days,
    transitWithin7Days: arrivalDist.transitWithin7Days,
    transitOver7Days: arrivalDist.transitOver7Days,

    // 预计到港（来自 ETA 统计）
    overdue: etaDist.overdue,
    within3Days: etaDist.within3Days,
    within7Days: etaDist.within7Days,
    over7Days: etaDist.over7Days
  };
}
```

### 2.2 ArrivalStatisticsService

**职责**: 按到港统计（实际到港日 ATA）

**文件**: `backend/src/services/statistics/ArrivalStatistics.service.ts`

**核心方法**:

```typescript
class ArrivalStatisticsService {
  // 获取到港分布统计
  async getDistribution(startDate?: string, endDate?: string): Promise<Record<string, number>>

  // 今日到港
  async arrivalToday(today: Date): Promise<number>

  // 之前到港未提柜
  async arrivedBeforeTodayNotPickedUp(today: Date): Promise<number>

  // 之前到港已提柜
  async arrivedBeforeTodayPickedUp(today: Date): Promise<number>

  // 中转港统计
  async transitOverdue(today: Date): Promise<number>
  async transitWithin3Days(today: Date): Promise<number>
  async transitWithin7Days(today: Date): Promise<number>
  async transitOver7Days(today: Date): Promise<number>
}
```

**SQL 实现示例**:

```typescript
async arrivalToday(today: Date): Promise<number> {
  const sql = `
    SELECT COUNT(DISTINCT c.container_number) as count
    FROM biz_containers c
    INNER JOIN process_port_operations po
      ON c.container_number = po.container_number
    WHERE po.port_type = 'destination'
      AND po.ata_dest_port::date = $1
      AND po.port_sequence = (
        SELECT MAX(port_sequence)
        FROM process_port_operations sub
        WHERE sub.container_number = c.container_number
          AND sub.port_type = 'destination'
      )
  `;

  const result = await this.containerRepository.query(sql, [today]);
  return parseInt(result[0]?.count || '0');
}
```

**关键点**:

- ✅ 使用 `port_sequence` 确保取最新一条港口操作
- ✅ 使用 `port_type='destination'` 过滤中转港
- ✅ 使用 `DISTINCT` 避免重复计数

```typescript
async within3Days(today: Date): Promise<number> {
  const sql = `
    SELECT COUNT(DISTINCT c.container_number) as count
    FROM biz_containers c
    WHERE c.logistics_status NOT IN ('unloaded', 'returned_empty')
      AND c.eta_dest_port::date >= $1
      AND c.eta_dest_port::date <= $1 + INTERVAL '3 days'
```

```typescript
async todayPlanned(today: Date): Promise<number> {
```

**SQL 实现示例**:

```typescript
async urgent(today: Date): Promise<number> {
  const sql = `
    SELECT COUNT(DISTINCT c.container_number) as count
    FROM biz_containers c
    INNER JOIN process_port_operations po
      ON c.container_number = po.container_number
    WHERE po.port_type = 'destination'
      AND po.last_free_date >= $1
      AND po.last_free_date <= $1 + INTERVAL '3 days'
      AND po.port_sequence = (
        SELECT MAX(port_sequence)
        FROM process_port_operations sub
        WHERE sub.container_number = c.container_number
          AND sub.port_type = 'destination'
      )
      AND (tt.pickup_date IS NULL OR tt.pickup_date::date > $1)
  `;

  const result = await this.containerRepository.query(sql, [today]);
  return parseInt(result[0]?.count || '0');
```

## 五、性能优化

### 5.1 批量查询

```typescript
// ❌ 坏的做法：N+1 查询
async function getStatistics() {
  const arrivalToday = await this.arrivalToday();
  const arrivedBeforeTodayNotPickedUp = await this.arrivedBeforeTodayNotPickedUp();
  const arrivedBeforeTodayPickedUp = await this.arrivedBeforeTodayPickedUp();
  // ... 每个维度单独查询
  return { arrivalToday, arrivedBeforeTodayNotPickedUp, ... };
}

// ✅ 好的做法：批量查询
async function getStatistics() {
  const sql = `
    SELECT
      COUNT(DISTINCT FILTER (WHERE po.ata_dest_port::date = CURRENT_DATE)) as arrivalToday,
      COUNT(DISTINCT FILTER (WHERE po.ata_dest_port::date < CURRENT_DATE AND tt.pickup_date IS NULL)) as arrivedBeforeTodayNotPickedUp,
      COUNT(DISTINCT FILTER (WHERE po.ata_dest_port::date < CURRENT_DATE AND tt.pickup_date IS NOT NULL)) as arrivedBeforeTodayPickedUp,
      -- ... 其他维度
    FROM biz_containers c
    INNER JOIN process_port_operations po ON c.container_number = po.container_number
    LEFT JOIN process_trucking_transport tt ON c.container_number = tt.container_number
    WHERE po.port_type = 'destination'
      AND po.port_sequence = (SELECT MAX(port_sequence) ...)
  `;

  const result = await this.containerRepository.query(sql);
  return result[0];
}
```

### 5.3 缓存策略

```typescript
// 使用 Redis 缓存统计结果
async getStatistics(startDate?: string, endDate?: string) {
  const cacheKey = `statistics:${startDate}:${endDate}`;

  // 1. 尝试从缓存获取
  const cached = await this.redis.get(cacheKey);
  if (cached) {
    return JSON.parse(cached);
  }

  // 2. 从数据库查询
  const statistics = await this.getStatisticsFromDB(startDate, endDate);

  // 3. 写入缓存（5 分钟过期）
  await this.redis.setex(cacheKey, 300, JSON.stringify(statistics));

  return statistics;
}
```
