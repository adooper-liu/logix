## 🔍 知识体系提炼

### 一、货柜数据结构

#### 核心概念

1. **container_number 是唯一标识**
   - 所有流程表的外键
   - 一旦创建不可修改
   - 格式：4 字母 +7 数字（如 HMMU6232153）

2. **logistics_status 是状态机计算结果**
   - 7 层简化状态
   - 优先级计算（还箱 > 卸柜 > 提柜 > 到港 > 出运 > 未出运）
   - 由 ContainerStatusService.updateStatus() 自动更新

3. **gantt_derived 是甘特图快照**
   - 五阶段设计（清关→提柜→卸柜→还箱→完成）
   - JSONB 格式存储
   - 与 logistics_status 同步计算

---

#### 表结构要点

```sql
-- 主表
biz_containers (
  container_number VARCHAR(100) PRIMARY KEY,
  logistics_status VARCHAR(20),      -- 状态机结果
  gantt_derived JSONB,               -- 甘特快照
  created_at TIMESTAMPTZ
)

-- 流程表（通过 container_number 关联）
process_sea_freight
process_port_operations
process_trucking_transport
process_warehouse_operations
process_empty_return
```

### 三、Composables 模式

#### useContainerDetail

```typescript
export function useContainerDetail() {
  const containerNumber = computed(() => {
    const p = route.params.containerNumber as string;
    return p ? decodeURIComponent(p) : '';
  });
  
  async function loadContainerData() {
    loading.value = true;
    try {
      const result = await containerService.getContainerById(containerNumber.value);
      containerData.value = result.data;
    } finally {
      loading.value = false;
    }
  }
  
  return { containerNumber, containerData, loading, loadContainerData };
}
```

**核心价值**:
- 从路由参数自动解码
- 统一的加载状态管理
- 错误处理封装

#### useContainerCountdown

```typescript
export function useContainerCountdown() {
  function getCountdownInfo(container: Container): CountdownInfo {
    const lastFreeDate = container.portOperations?.[0]?.lastFreeDate;
    
    if (!lastFreeDate) return { status: 'no_data' };
    
    const diffDays = calculateDiffDays(lastFreeDate);
    
    if (diffDays < 0) return { status: 'expired' };
    if (diffDays <= 2) return { status: 'urgent' };
    return { status: 'normal' };
  }
  
  return { getCountdownInfo };
}
```

**核心价值**:
- 倒计时计算逻辑封装
- 状态判断（正常/紧急/超期）
- 可复用的工具函数

### 四、后端状态同步机制

#### ContainerStatusService

**文件**: `backend/src/services/container-status.service.ts`

```typescript
async updateStatus(containerNumbers?: string[]): Promise<void> {
  // 1. 批量查询货柜
  const containers = await this.getContainersWithDetails(containerNumbers);
  
  // 2. 并行计算每个货柜的状态
  const updates = await Promise.all(
    containers.map(async c => {
      const statusResult = calculateLogisticsStatus(c);
      const ganttDerived = buildGanttDerivedSnapshot(c);
      
      return {
        containerNumber: c.containerNumber,
        logisticsStatus: statusResult.status,
        ganttDerived
      };
    })
  );
  
  // 3. 批量写回数据库
  await this.bulkUpdate(updates);
}
```

---

#### rebuildGanttDerivedSnapshot

```typescript
async rebuildGanttDerivedSnapshot(containerNumber: string): Promise<GanttDerived> {
  // 1. 查询货柜及所有关联数据
  const container = await this.getFullContainer(containerNumber);
  
  // 2. 应用构建规则
  const snapshot = buildGanttDerivedSnapshot(container);
  
  // 3. 写回数据库
  await this.containerRepository.update(containerNumber, {
    ganttDerived: snapshot
  });
  
  return snapshot;
}
```
