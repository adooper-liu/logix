## 概述

智能排柜系统是 LogiX 物流系统的核心决策引擎，基于规则和算法自动分派仓库和车队，实现成本优化和资源高效利用。

### 核心价值

1. **自动化**: 减少人工干预，提高排产效率（50+ 柜/分钟）
2. **成本优化**: 智能选择最优方案，平均降低 15-30% 总成本
3. **资源平衡**: 合理分配仓库和车队产能，避免拥堵
4. **可追溯**: 完整的日志和档期扣减记录，支持审计

### 二、三种卸柜策略

#### Direct (Live Unload)

**特点**: 提柜 = 卸柜，当天完成  
**适用**: 无堆场的车队  
**成本**: 较低（无堆场费）  
**风险**: 可能产生滞港费

```typescript
// 成本计算
totalCost = demurrageCost + detentionCost + transportationCost
```

---

#### Drop off

**特点**: 提柜后暂存堆场，再送仓卸柜  
**适用**: 有堆场的车队  
**成本**: 堆场费 + 操作费 + 运输费×2  
**优势**: 缓解仓库压力，可能降低总成本

```typescript
// 成本计算
totalCost =
  demurrageCost + detentionCost + yardStorageCost + yardOperationFee + transportationCost * 2.0
```

---

#### Expedited

**特点**: 免费期内加急处理  
**适用**: 即将超期的货柜  
**成本**: 加急操作费  
**目标**: 避免滞港费

```typescript
// 成本计算
totalCost = handlingCost + transportationCost
// (免费期内，D&D 费用=0)
```

### 三、成本优化决策树

```
货柜到达目的港
        ↓
获取免费期截止日 (last_free_date)
        ↓
判断是否超期？
├─ 已超期 → 尽早安排（减少损失）
│         ↓
│     计算每天产生的费用
│         ↓
│     选择最早可用日期（成本最低）
│
└─ 未超期 → 是否使用 Drop off？
            ↓
        比较：堆场费 vs 滞港费
            ↓
        如果 堆场费 < 滞港费 * 0.7
            ↓
        选择 Drop off（节省 30% 以上）
            ↓
        否则选择 Direct
```

### 四、日历配置机制

#### 仓库能力模型

```typescript
interface WarehouseCapacity {
  warehouseCode: string
  date: Date
  totalCapacity: number // 总容量（如：20）
  usedCapacity: number // 已使用容量
  remainingCapacity: number // 剩余容量
  isRestDay: boolean // 是否休息日
}
```

**查询可用日期算法**:

```typescript
async findEarliestAvailableDay(
  warehouseCode: string,
  startDate: Date
): Promise<Date | null> {
  let currentDate = startDate;
  const maxDaysToSearch = 30;

  for (let i = 0; i < maxDaysToSearch; i++) {
    // 跳过周末
    if (isWeekend(currentDate)) {
      currentDate = addDays(currentDate, 1);
      continue;
    }

    // 检查剩余产能
    const capacity = await getRemainingCapacity(
      warehouseCode,
      currentDate
    );

    if (capacity > 0) {
      return currentDate; // 找到可用日期
    }

    currentDate = addDays(currentDate, 1);
  }

  return null; // 30 天内无可用日期
}
```
