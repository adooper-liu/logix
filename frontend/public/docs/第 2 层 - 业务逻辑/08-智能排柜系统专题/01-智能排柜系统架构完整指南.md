## 📋 概述

智能排柜系统是 LogiX 物流系统的核心决策引擎，基于规则和算法自动分派仓库和车队，实现成本优化和资源高效利用。

### 核心价值

1. **自动化**: 减少人工干预，提高排产效率
2. **成本优化**: 智能选择最优方案，降低总成本
3. **资源平衡**: 合理分配仓库和车队产能
4. **可追溯**: 完整的日志和档期扣减记录

### 1.2 数据流

#### 批量排产流程

```
1. 查询待排产货柜
   (schedule_status = 'initial')
        ↓
2. 按清关可放行日排序
   (sort by lastFreeDate ASC)
        ↓
3. 遍历每个货柜
   ├─ 选择仓库 → WarehouseSelectorService
   ├─ 选择车队 → TruckingSelectorService
   ├─ 计算日期 → SchedulingDateCalculator
   ├─ 扣减档期 → OccupancyCalculator
   └─ 写入数据库
        ↓
4. 成本优化建议（可选）
   → SchedulingCostOptimizerService
        ↓
5. 返回排产结果
```

---

#### 单柜排产流程（指定仓库）

```
1. 验证指定仓库
        ↓
2. 查找最早可用卸柜日
   (从提柜日 +1 开始，最多查 30 天)
        ↓
3. 选择车队
   (基于仓库和港口)
        ↓
4. 计算送仓日和还箱日
        ↓
5. 扣减档期并写入
        ↓
6. 返回排产结果
```

## 三、核心算法

### 3.1 仓库选择算法

```typescript
async selectWarehouseForPort(params: {
  countryCode: string;
  portCode: string;
  plannedDate: Date;
}): Promise<Warehouse | null> {
  const { countryCode, portCode, plannedDate } = params;
  
  // 1. 获取仓库 - 车队映射
  const mappings = await this.warehouseTruckingMappingRepo.find({
    where: {
      country: countryCode,
      isActive: true
    },
    order: { priority: 'ASC' }
  });
  
  // 2. 提取仓库代码列表
  const warehouseCodes = mappings.map(m => m.warehouseCode);
  
  // 3. 查询仓库
  const warehouses = await this.warehouseRepo.find({
    where: {
      warehouseCode: In(warehouseCodes),
      isActive: true
    }
  });
  
  // 4. 检查产能
  for (const warehouse of warehouses) {
    const available = await this.occupancyCalculator.isAvailable(
      warehouse.warehouseCode,
      plannedDate
    );
    
    if (available) {
      return warehouse; // 返回第一个有产能的仓库
    }
  }
  
  return null; // 无可用仓库
}
```

### 3.2 车队选择算法

```typescript
async selectTruckingCompany(params: {
  warehouseCode: string;
  portCode: string;
  countryCode: string;
  plannedDate: Date;
}): Promise<TruckingCompany | null> {
  const { warehouseCode, portCode, countryCode, plannedDate } = params;
  
  // 1. 获取仓库 - 车队映射
  const mapping = await this.warehouseTruckingMappingRepo.findOne({
    where: {
      warehouseCode,
      country: countryCode,
      isActive: true
    }
  });
  
  if (!mapping) {
    return null;
  }
  
  // 2. 查询车队
  const truckingCompany = await this.truckingCompanyRepo.findOne({
    where: {
      companyCode: mapping.truckingCompanyId,
      isActive: true
    }
  });
  
  if (!truckingCompany) {
    return null;
  }
  
  // 3. 验证车队 - 港口映射
  const portMapping = await this.truckingPortMappingRepo.findOne({
    where: {
      truckingCompanyId: mapping.truckingCompanyId,
      portCode,
      country: countryCode,
      isActive: true
    }
  });
  
  if (!portMapping) {
    return null;
  }
  
  // 4. 检查车队档期
  const available = await this.isTruckingAvailable(
    mapping.truckingCompanyId,
    plannedDate,
    portCode,
    warehouseCode
  );
  
  if (!available) {
    return null;
  }
  
  return truckingCompany;
}
```

### 3.3 可用日期查找算法

```typescript
async findEarliestAvailableDay(
  warehouseCode: string,
  startDate: Date
): Promise<Date | null> {
  let currentDate = new Date(startDate);
  const maxDaysToSearch = 30;
  
  for (let i = 0; i < maxDaysToSearch; i++) {
    // 1. 跳过周末
    if (this.isWeekend(currentDate)) {
      currentDate = dateTimeUtils.addDays(currentDate, 1);
      continue;
    }
    
    // 2. 检查是否为休息日（从配置读取）
    const isRestDay = await this.isConfiguredRestDay(currentDate);
    if (isRestDay) {
      currentDate = dateTimeUtils.addDays(currentDate, 1);
      continue;
    }
    
    // 3. 检查剩余产能
    const capacity = await this.occupancyCalculator.getRemainingCapacity(
      warehouseCode,
      currentDate
    );
    
    if (capacity > 0) {
      return currentDate; // 找到可用日期
    }
    
    // 4. 继续下一天
    currentDate = dateTimeUtils.addDays(currentDate, 1);
  }
  
  return null; // 30 天内无可用日期
}

private isWeekend(date: Date): boolean {
  const day = date.getDay();
  return day === 0 || day === 6; // 周日或周六
}
```

### 3.4 档期扣减算法

```typescript
async decrementAllOccupancies(params: {
  containerNumber: string;
  warehouse: Warehouse;
  truckingCompany: TruckingCompany;
  plannedPickupDate: Date;
  plannedUnloadDate: Date;
  plannedReturnDate: Date;
  portCode: string;
  unloadMode: string;
}): Promise<void> {
  const {
    warehouse,
    truckingCompany,
    plannedPickupDate,
    plannedUnloadDate,
    plannedReturnDate,
    portCode,
    unloadMode
  } = params;
  
  // 1. 扣减仓库卸柜档期
  await this.occupancyCalculator.decrementOccupancy({
    warehouseCode: warehouse.warehouseCode,
    date: plannedUnloadDate
  });
  
  logger.info(
    `[IntelligentScheduling] Decremented warehouse occupancy: ` +
    `${warehouse.warehouseCode} on ${plannedUnloadDate.toISOString().split('T')[0]}`
  );
  
  // 2. 扣减车队送柜档期
  await this.decrementTruckingOccupancy(
    truckingCompany.companyCode,
    plannedPickupDate, // 使用提柜日作为送柜档期
    portCode,
    warehouse.warehouseCode
  );
  
  // 3. 如果是 Drop off 模式，扣减还箱档期
  if (unloadMode === 'Drop off') {
    await this.decrementTruckingReturnOccupancy(
      truckingCompany.companyCode,
      plannedReturnDate,
      portCode,
      warehouse.warehouseCode
    );
  }
}
```

## 五、常见问题排查

### 5.1 问题 1: 排产失败 - 仓库无产能

**现象**:
```
排产失败：Warehouse WH001 has no available capacity on 2026-03-20
```

**排查步骤**:

```sql
-- Step 1: 检查仓库档期
SELECT 
  warehouse_code,
  date,
  total_capacity,
  used_capacity,
  remaining_capacity,
  is_rest_day
FROM ext_warehouse_daily_occupancy
WHERE warehouse_code = 'WH001'
  AND date BETWEEN '2026-03-15' AND '2026-03-25'
ORDER BY date;

-- Step 2: 检查仓库配置
SELECT * FROM dict_warehouses WHERE warehouse_code = 'WH001';

-- Step 3: 检查排柜配置
SELECT * FROM dict_scheduling_config 
WHERE config_key IN ('skip_weekends', 'planning_horizon_days');
```

**常见原因**:
- ❌ 仓库档期已满（used_capacity = total_capacity）
- ❌ 配置的休息日
- ❌ 周末（skip_weekends=true）

**解决方案**:
- ✅ 增加仓库产能：UPDATE ext_warehouse_daily_occupancy SET total_capacity = 30 WHERE ...
- ✅ 调整休息日配置
- ✅ 选择其他有产能的仓库

### 5.2 问题 2: 找不到合适车队


```sql
SELECT 
  wtm.warehouse_code,
  wtm.trucking_company_id,
  wtm.country,
  wtm.is_active,
  tc.company_name,
  tc.is_active AS trucking_active

  ON wtm.trucking_company_id = tc.company_code

  AND wtm.country = 'US';

SELECT 
  tpm.trucking_company_id,
  tpm.port_code,
```
