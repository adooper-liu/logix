# 货柜管理模块详细文档

**模块**: Shipments  
**功能**: 货柜的查询、筛选、详情查看、物流跟踪

---

## 一、模块结构

```
shipments/
├── Shipments.vue                    # 货柜列表主页面
├── ContainerDetailRefactored.vue    # 货柜详情页

    ├── ContainerInfo.vue           # 基本信息
```

## 三、核心API调用

### 3.1 货柜列表

```typescript
// GET /api/containers
interface ContainerQueryParams {
  page?: number;
  pageSize?: number;
  country?: string;
  status?: string;
  portCode?: string;
  filterCondition?: string;
  startDate?: string;
  endDate?: string;
  search?: string;
}

// Response
interface ContainerListResponse {
  data: Container[];
  total: number;
  page: number;
  pageSize: number;
}
```

### 3.2 货柜详情

```typescript
// GET /api/containers/:containerNumber
interface ContainerDetailResponse {
  container: Container;
  seaFreight?: SeaFreight;
  portOperations: PortOperation[];
  truckingTransport?: TruckingTransport;
  warehouseOperations?: WarehouseOperation[];
  emptyReturn?: EmptyReturn;
}
```

### 3.3 物流路径

```typescript
// GET /api/containers/:containerNumber/logistics-path
interface LogisticsPathResponse {
  container: Container;
  path: LogisticsNode[];
  currentStatus: string;
  etaDestPort?: Date;
  ataDestPort?: Date;
}
```

## 四、数据流

```
用户进入页面
    │
    ▼
Shipments.vue
    │
    ├── useContainers (Composable)
    │     │
    │     └── containerService.getContainers()
    │           │
    │           └── httpClient.get('/containers')
    │                 │
    │                 ▼
    │           ContainerController
    │                 │
    │                 ▼
    │           ContainerService.findAll()
    │                 │
    │                 ▼
    │           ContainerRepository.createQueryBuilder()
    │                 │
    │                 ▼
    │           PostgreSQL (biz_containers)
    │
    ▼
渲染列表/甘特图
    │
    ▼
用户点击货柜
    │
    ▼
ContainerDetailRefactored.vue
    │
    ├── 获取货柜基本信息
    ├── 获取海运信息
    ├── 获取港口操作列表
    ├── 获取拖卡运输信息
    ├── 获取仓库操作信息
    ├── 计算滞港费
    └── 渲染各Tab组件
```

## 六、关键状态

### 6.1 物流状态

| 状态值 | 说明 |
|--------|------|
| not_shipped | 未出运 |
| shipped | 已出运 |
| in_transit | 在途 |
| at_port | 已到港 |
| picked_up | 已提柜 |
| unloaded | 已卸柜 |
| returned_empty | 已还箱 |

### 6.2 港口类型

| 类型值 | 说明 |
|--------|------|
| origin | 起运港 |
| transit | 中转港 |
| destination | 目的港 |
