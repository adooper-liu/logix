## 更新概览

本次更新为甘特图系统添加了两大核心可视化功能：**路径追踪**和**呼吸动画**，并优化了视觉样式和交互体验。

### 主要功能

1. **路径追踪组件** - 曲线路径可视化货柜流转

4. **z-index层级优化** - 确保动画显示在周末背景色之上

## 一、路径追踪功能

### 1.1 功能描述

当用户右键点击货柜圆点并选择"显示路径连线"时，系统会用绿色虚线曲线连接该货柜在所有泳道（清关、提柜、卸柜、还箱）上的圆点，形成完整的路径可视化。

### 1.2 技术实现

**组件文件**: `frontend/src/components/common/gantt/GanttPathTracker.vue` (274行)

**核心特性**:
- ✅ **二次贝塞尔曲线** - 优雅的弧线效果，避免直线穿过其他圆点
- ✅ **避开圆点设计** - 连线从圆点边缘开始，保持6px间距
- ✅ **自适应弯曲** - curveOffset系数0.5，最大偏移80px
- ✅ **无箭头设计** - 简洁的曲线，视觉更清爽
- ✅ **绘制动画** - 1秒淡入动画，流畅自然

**关键代码**:
```typescript
// 生成曲线路径（避开圆点）
const generateCurvePath = (segment: PathSegment): string => {
  const { x1, y1, x2, y2 } = segment
  
  // 计算方向向量，从圆点边缘开始
  const dotRadius = 6 // 圆点半径5px + 1px间距
  const dx = x2 - x1
  const dy = y2 - y1
  const distance = Math.sqrt(dx * dx + dy * dy)
  
  const unitX = dx / distance
  const unitY = dy / distance
  
  const startX = x1 + unitX * dotRadius
  const startY = y1 + unitY * dotRadius
  const endX = x2 - unitX * dotRadius
  const endY = y2 - unitY * dotRadius
  
  // 控制点：中点上方偏移
  const midX = (startX + endX) / 2
  const midY = (startY + endY) / 2
  const horizontalDistance = Math.abs(endX - startX)
  const curveOffset = Math.min(horizontalDistance * 0.5, 80)
  
  const controlX = midX
  const controlY = midY - curveOffset
  
  return `M ${startX} ${startY} Q ${controlX} ${controlY} ${endX} ${endY}`
}
```

**样式参数**:
```typescript
strokeColor: '#67c23a'      // 绿色，与预警正常状态一致
strokeWidth: 2.5             // 稍微加粗，更清晰
isDashed: true               // 虚线样式
dashPattern: '6,4'           // 虚线间隔
opacity: 0.7                 // 70%透明度
```

### 1.3 使用方法

**在甘特图中集成**:
```vue
<GanttPathTracker
  :visible="showPathLines"
  :container="pathLineContainer"
  container-selector=".container-dot"
  :scroll-container="ganttScrollContainerRef"
  @pathCalculated="handlePathCalculated"
/>
```

**触发方式**:
1. 右键点击任意货柜圆点
2. 选择"显示路径连线"菜单项
3. 再次点击可关闭

### 1.4 视觉效果

**曲线示例**:
```
    ○ ~~~~~~~~ ○
   清关        提柜
    
    ○ ~~~~~~~~~~~~~~~~ ○
   提柜              卸柜
```

**特点**:
- 曲线向上弯曲，避开中间的其他圆点
- 从圆点边缘开始，不穿过圆心
- 绿色虚线，与整体配色一致
- 1秒绘制动画，优雅流畅

## 二、呼吸动画功能

### 2.1 功能描述

当用户将鼠标悬停在货柜圆点上超过5秒时，系统会自动高亮显示该货柜在所有泳道上的所有圆点，通过双层脉冲动画帮助用户快速识别和追踪完整路径。

### 2.2 技术实现

**组件文件**: `frontend/src/components/common/gantt/ContainerDotBreather.vue` (341行)

**核心特性**:
- ✅ **延迟触发** - 5秒后启动，避免误触发
- ✅ **全局高亮** - 同时高亮所有泳道的同一货柜
- ✅ **双层动画** - 外圈脉冲扩散 + 内圈缩放呼吸
- ✅ **Tooltip联动** - 动画启动时自动关闭Tooltip
- ✅ **响应式更新** - 滚动和窗口变化时重新定位
- ✅ **z-index优化** - 确保在周末背景色之上

**关键代码**:
```typescript
// 查找所有泳道中同一货柜的圆点
const findContainerDots = (containerNumber: string): BreathingDot[] => {
  const dots: BreathingDot[] = []
  const allDots = document.querySelectorAll(props.dotSelector)

  allDots.forEach((dot) => {
    const dataContainer = dot.getAttribute('data-container')
    
    if (dataContainer === containerNumber) {
      const rect = dot.getBoundingClientRect()
      let x = rect.left + rect.width / 2
      let y = rect.top + rect.height / 2

      if (scrollContainer) {
        const containerRect = scrollContainer.getBoundingClientRect()
        x -= containerRect.left
        y -= containerRect.top
      }

      dots.push({
        containerNumber,
        x,
        y,
        size: 12,
        color: props.breathColor,
      })
    }
  })

  return dots
}
```

**动画效果**:
```css

  0% { transform: scale(0.8); opacity: 0.8; }
  100% { transform: scale(1.5); opacity: 0; }
}

/* 内圈高亮 - 呼吸节奏 */
@keyframes highlightPulse {
  0%, 100% { transform: scale(1); opacity: 0.3; }
  50% { transform: scale(1.2); opacity: 0.6; }
}
```

**样式参数**:
```typescript

breathColor: '#67c23a'      // 绿色，与路径追踪一致
zIndex: 100                 // 确保在周末背景色之上
```

### 2.3 使用方法

**在甘特图中集成**:
```vue

  :enabled="true"

  dot-selector=".container-dot"

```

**触发方式**:
1. 鼠标悬停在任意货柜圆点上
2. 保持悬停5秒
3. 呼吸动画自动启动，Tooltip自动关闭
4. 鼠标移开，动画立即停止

### 2.4 视觉效果

**动画示例**:

**特点**:
- 双层脉冲效果，视觉冲击力强
- 绿色主题，与路径追踪统一

- 所有节点同时呼吸，易于识别

## 三、Tooltip自动关闭

### 3.1 功能描述

当呼吸动画启动时（悬停5秒后），系统会自动关闭Tooltip，避免两者同时显示造成视觉混乱。

### 3.2 技术实现

**事件通信**:
```typescript
// ContainerDotBreather.vue
const emit = defineEmits<{
  (e: 'breathStart', container: Container): void
  (e: 'breathEnd', container: Container): void
  (e: 'closeTooltip'): void // 通知关闭Tooltip
}>()

const startBreathing = (container: Container) => {
  // ...
  emit('closeTooltip') // 通知父组件关闭Tooltip
}
```

**父组件监听**:
```vue
<!-- SimpleGanttChartRefactored.vue -->
<ContainerDotBreather
  @close-tooltip="hideTooltip"
/>
```

### 3.3 交互流程

```
用户操作                    系统响应
────────                  ────────
悬停圆点 (0s)      →      显示Tooltip
继续悬停 (5s)      →      发出closeTooltip事件
                        → 调用hideTooltip()
                        → Tooltip关闭
                        → 呼吸动画启动
鼠标移开            →      呼吸动画停止
```

### 3.4 用户体验

**优化前**:
```
┌─────────────┐
│  Tooltip    │ ← 与呼吸动画重叠，信息过载
│  基本信息   │
└─────────────┘
    ○ ~~~~ ○    ← 呼吸动画
```

**优化后**:
```
0-5秒:
┌─────────────┐
│  Tooltip    │ ← 只显示基本信息
└─────────────┘

5秒后:
    ○ ~~~~ ○    ← 只显示呼吸动画，视觉清爽
```

## 四、z-index层级优化

### 4.1 问题描述

呼吸动画组件需要显示在周末背景色（`#fef0f0`）之上，否则动画可能被遮挡。

### 4.2 解决方案

**容器定位**:
```css
.container-dot-breather {
  position: absolute;
  top: 0;
  left: 0;
  width: 100%;
  height: 100%;
  pointer-events: none;
  overflow: visible;
  z-index: v-bind(zIndex); /* 动态绑定 */
}
```

**层级设置**:
```vue
<ContainerDotBreather
  :z-index="100"
/>
```

### 4.3 层级对比

| 元素 | z-index | 说明 |
|------|---------|------|
| 周末背景色 | auto/0 | 默认层级 |
| 货柜圆点 | auto/0 | 默认层级 |
| **呼吸动画** | **100** | **确保在背景色之上** |
| Tooltip | 1000 | 最高层级 |
| 拖拽指示器 | 9998-9999 | 临时最高层级 |
