## 📋 当前在用组件

### ✅ **SimpleGanttChartRefactored.vue** - 唯一推荐使用的版本

**路径**: `@/components/common/SimpleGanttChartRefactored.vue`

**使用位置**:
```typescript
// frontend/src/router/index.ts:91
{
  path: 'gantt-chart',
  name: 'GanttChart',
  component: () => import('@/components/common/SimpleGanttChartRefactored.vue'),
  meta: {
    title: '货柜甘特图',
    icon: 'Calendar',
    requiresAuth: true,
  }
}
```

**核心特性**:
- ✅ 使用 `useGanttLogic` composable 抽离业务逻辑
- ✅ 支持两种视图模式（独立表格 / 弹窗详情）
- ✅ 完整的三级分组（港口 → 节点 → 供应商）
- ✅ 拖拽改期功能
- ✅ 智能预警提示
- ✅ 资源占用统计（可选）
- ✅ 动态日期范围（7/15/30 天或自定义）

**代码示例**:
```vue
<template>
  <div class="simple-gantt-chart">
    <!-- 顶部信息栏 -->
    <GanttHeader
      :filter-label="filterLabel"
      :container-count="finalFilteredContainers.length"
      :loading="loading"
      @export="exportData"
      @back="goBack"
      @refresh="loadData"
    >
      <GanttSearchBar 
        @search="handleSearch" 
        @update:searchField="handleSearchFieldChange" 
      />
    </GanttHeader>

    <!-- 统计面板 -->
    <GanttStatisticsPanel 
      :containers="finalFilteredContainers" 
      @filter="handleStatFilter" 
    />

    <!-- 工具栏 -->
    <div class="toolbar">
      <el-radio-group v-model="viewMode">
        <el-radio-button value="independent">独立表格</el-radio-button>
        <el-radio-button value="modal">弹窗详情</el-radio-button>
      </el-radio-group>
      
      <el-radio-group v-model="rangeType" @change="onRangeChange">
        <el-radio-button :value="0">动态</el-radio-button>
        <el-radio-button :value="7">7 天</el-radio-button>
        <el-radio-button :value="15">15 天</el-radio-button>
        <el-radio-button :value="30">30 天</el-radio-button>
      </el-radio-group>
    </div>

    <!-- 甘特图主体 -->
    <div class="gantt-body" v-loading="loading">
      <!-- ... -->
    </div>
  </div>
</template>

<script setup lang="ts">
import { useGanttLogic } from '@/components/common/gantt/useGanttLogic'

const {
  containers,
  loading,
  filterLabel,
  rangeType,
  displayRange,
  viewMode,
  loadData,
  exportData,
  goBack,
  // ...
} = useGanttLogic()
</script>
```

## ⚠️ 旧版本组件（不推荐使用）

### SimpleGanttChart.vue - 旧版本

**路径**: `@/components/common/SimpleGanttChart.vue`

**状态**:
- ⚠️ **保留但不推荐使用**
- ❌ 功能已被 Refactored 版替代
- ❌ 不再维护

**为什么不推荐**:
1. **缺少 composable 抽离**: 业务逻辑和 UI 混在一起
2. **功能不完整**: 缺少视图模式切换、资源统计等
3. **性能较差**: 没有虚拟滚动优化
4. **维护成本高**: 代码重复，难以同步更新

**历史路由**（已废弃）:
```typescript
// 旧路由配置（已删除）
{
  path: 'gantt-chart-old',
  name: 'GanttChartOld',
  component: () => import('@/components/common/SimpleGanttChart.vue')
}
```
