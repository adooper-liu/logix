## 📋 概述

甘特图系统是 LogiX 物流系统的核心可视化组件，提供货柜全生命周期的时间线展示、智能预警、拖拽改期等强大功能。

### 核心价值

1. **可视化追踪**: 直观展示货柜从清关到还箱的完整流程（五节点）
2. **智能预警**: 自动识别超期、即将超期等风险货柜（红/橙预警）
3. **交互操作**: 支持拖拽改期、右键菜单、详情查看
4. **多维度分组**: 按港口 → 节点 → 供应商三级分组展示
5. **性能优化**: 虚拟滚动、懒加载、计算属性缓存

## 🎨 技术栈

- **前端框架**: Vue 3 + Composition API + `<script setup>`
- **UI 组件库**: Element Plus
- **状态管理**: Pinia (ganttFilters store)
- **工具库**: dayjs (日期处理)
- **HTTP 客户端**: Axios
- **实时通信**: Socket.IO (可选)

## 📝 关键概念

### gantt_derived 字段

**定义**: JSONB 类型，存储在 `biz_containers` 表中

**作用**:

- ✅ 预计算各节点状态（planned/actual）
- ✅ 加速甘特图渲染
- ✅ 减少数据库查询

**结构**:

```json
{
  "nodes": {
    "customs": { "status": "actual", "actualDate": "2026-03-20" },
    "pickup": { "status": "planned", "plannedDate": "2026-03-25" },
    "unload": { "status": "planned", "plannedDate": "2026-03-25" },
    "return": { "status": "planned", "plannedDate": "2026-03-26" }
  },
  "updatedAt": "2026-03-20T10:30:00Z",
  "version": "v2+"
}
```

---

### 三级分组

**层级**:

1. **第一层**: 目的港（如 USLAX、USLGB）
2. **第二层**: 五节点（清关、提柜、卸柜、还箱、查验）
3. **第三层**: 供应商（清关行、车队、仓库）

**示例**:

```
USLAX (目的港)
├── 清关 (节点)
│   └── XX 清关行 (供应商)
│       ├── HMMU6232153
│       └── HMMU6232154
├── 提柜 (节点)
│   └── XX 车队 (供应商)
│       ├── HMMU6232153
│       └── HMMU6232155
└── ...
```

---

### 圆点类型

| 类型         | 外观                    | 含义       | 触发条件                                       |
| ------------ | ----------------------- | ---------- | ---------------------------------------------- |
| **实心圆点** | 绿色填充                | 已完成     | `ganttDerived.nodes[node].status === 'actual'` |
| **空心圆点** | 白色背景 + 橙色虚线边框 | 未完成     | `status === 'planned'`                         |
| **虚节点**   | 灰色半透明              | 上游未完成 | 节点无数据且上游未完成                         |
| **销毁节点** | 红色填充                | 数据不一致 | 实际有日期但 ganttDerived 没有                 |

## 🚀 快速开始

### 1. 安装依赖

```bash
cd frontend
npm install
```

### 2. 启动开发服务器

```bash
npm run dev
```

### 3. 访问甘特图页面

浏览器打开：`http://localhost:5173/gantt-chart`

### 4. 基础配置

确保后端已运行，并且：

- ✅ 数据库中有货柜数据
- ✅ `gantt_derived` 字段已正确计算
- ✅ 预警规则已配置
