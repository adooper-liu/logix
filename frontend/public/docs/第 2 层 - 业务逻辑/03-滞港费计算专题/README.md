## 🎓 使用建议

### 第一次使用

1. **从 10-完整指南开始**
   - 花 1-2 小时通读全文
   - 理解四种费用类型
   - 掌握 actual/forecast 模式

2. **然后看 11-实战指南**
   - 重点阅读场景对比
   - 学习排查方法
   - 标记常用章节

3. **收藏 12-导航文档**
   - 作为快速查找工具
   - 使用交叉引用
   - 记录学习笔记

---

### 日常使用

1. **遇到问题时**
   - 先查 12-导航文档的快速查找表
   - 根据指引跳转到对应文档
   - 使用排查清单逐步验证

2. **深入学习时**
   - 针对特定主题查阅历史文档
   - 对比不同文档的视角
   - 形成系统认知

3. **分享知识时**
   - 推荐新人阅读 10-完整指南
   - 使用 11-实战指南的案例
   - 提供 12-导航文档作为索引
