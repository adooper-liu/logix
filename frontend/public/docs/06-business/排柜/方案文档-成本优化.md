```
├──────────┼─────────────┼──────────────┤

│ 优化后   │ 3-27 ✅     │ $900         │ ← 更早，费用更少
│ 错误方案 │ 3-30 ❌     │ $1,200       │ ← 更晚，费用更多
└────────────────────────┴──────────────┘
```

**算法流程**：
```typescript
// 超期货柜优化
let optimalDate = new Date(); // 从今天开始

while (optimalDate <= basePickupDate + 7 天) {
  const capacity = warehouseCapacities.get(optimalDate);
  
  if (capacity.availableCapacity > 0) {
    // 找到最早可用的日期，立即使用
    container.optimizedPickupDate = optimalDate;
    container.optimizedCost = calculateCost(optimalDate);
```

**关键点**：
- ✅ 从**今天**开始找（不是从免费期截止日）
- ✅ 找到**第一个**可用日期就使用
- ✅ 不追求成本为 0（已经不可能的）
- ✅ 目标是**减少损失**

```

└────────────────────────┴──────────────┴────────────┘
```

**算法流程**：
```typescript
// 免费期内的货柜优化
let optimalDate = lastFreeDate; // 从免费期截止日开始

while (optimalDate >= basePickupDate - 7 天) {
  const capacity = warehouseCapacities.get(optimalDate);
  
  if (capacity.availableCapacity > 0) {
    const cost = calculateCost(optimalDate);
    
    if (cost === 0) {
      // 找到成本为 0 且最晚的日期
      container.optimizedPickupDate = optimalDate;
      container.optimizedCost = 0;
      break; // ✅ 找到成本为 0 的就停止
    }
  }
  
  optimalDate = addDays(optimalDate, -1); // 往前找
}
```

**关键点**：
- ✅ 从**免费期截止日**开始找
- ✅ 必须**成本为 0**才使用
- ✅ 尽量**靠近免费期**（日期越大越好）
- ✅ 目标是**灵活性最高**
