# 甘特图交互增强功能

本文档说明SimpleGanttChart甘特图新增的三个交互增强功能。

---

## 一、功能概览

| 功能 | 说明 | 实现方式 |
|------|------|----------|
| **详情侧边栏** | 点击圆点后从右侧滑出详情面板，显示货柜完整信息 | ContainerDetailSidebar组件 + el-drawer |
| **拖拽调整日期** | 支持拖拽圆点到其他日期格子，快速调整货柜日期 | HTML5 Drag & Drop API |
| **右键菜单** | 右键点击圆点显示快速操作菜单 | ContainerContextMenu组件 |

---

## 二、详情侧边栏

### 2.1 使用方式

1. **左键点击**甘特图上的货柜圆点
2. 右侧滑出详情侧边栏
3. 显示货柜的完整信息：
   - 基本信息（集装箱号、柜型、物流状态、目的港）
   - 订单信息（备货单号、提单号、销往国家、客户名称）
   - 海运信息（船名、航次、起运港、预计到港）
   - 关键日期（预计到港日、实际到港日、计划提柜日、最晚提柜日、最晚还箱日、实际还箱日）
   - 备注
4. 可选择"查看完整详情"跳转到货柜详情页，或点击"关闭"关闭侧边栏

### 2.2 技术实现

**组件位置**：`frontend/src/components/common/ContainerDetailSidebar.vue`

**核心代码**：
```vue
<el-drawer
  v-model="visible"
  title="货柜详情"
  size="400px"
  direction="rtl"
>
  <!-- 详情内容 -->
</el-drawer>
```

**状态颜色映射**：
- 未出运：灰色（info）
- 已出运/在途：蓝色（primary）
- 已到港：橙色（warning）
- 已提柜/已卸柜/已还箱：绿色（success）

## 三、拖拽调整日期

### 3.1 使用方式

1. **按住左键**拖拽货柜圆点
2. 拖拽到目标日期格子时，格子会显示绿色虚线边框
3. 显示拖放指示器，提示目标日期
4. **释放鼠标**，弹出确认对话框
5. 确认后更新货柜日期

### 3.2 视觉反馈

- **拖拽时**：原圆点半透明，放大1.2倍
- **拖拽目标格**：绿色背景（#e1f3d8）+ 绿色虚线边框（2px dashed #67c23a）
- **拖放指示器**：固定定位提示框，显示"拖放至 MM-DD"

### 3.3 技术实现

**拖拽事件**：
```typescript
// 拖拽开始
@dragstart="handleDragStart(container, $event)"

// 拖拽结束
@dragend="handleDragEnd"

// 全局拖拽悬停
document.addEventListener('dragover', handleDragOver)

// 全局放置
document.addEventListener('drop', handleGlobalDrop)
```

**关键逻辑**：
```typescript
const handleDragStart = (container: Container, event: DragEvent) => {
  draggingContainer.value = container
  if (event.dataTransfer) {
    event.dataTransfer.effectAllowed = 'move'
    event.dataTransfer.setData('text/plain', container.containerNumber)
  }
}

const handleDrop = (date: Date) => {
  if (!draggingContainer.value || !dragOverDate.value) return
  const newDate = dayjs(dragOverDate.value).format('YYYY-MM-DD HH:mm:ss')
  // 调用API更新日期
  console.log('Move container to date:', newDate)
}
```

**注意事项**：
- 当前版本仅前端演示，未接入后端API
- 确认对话框使用ElMessage.confirm
- 更新成功后自动刷新甘特图数据

## 四、右键菜单

### 4.1 使用方式

1. **右键点击**货柜圆点
2. 在鼠标位置弹出右键菜单
3. 菜单包含以下操作：
   - **查看详情**：打开详情侧边栏
   - **调整日期**：打开日期编辑对话框
   - **复制柜号**：复制柜号到剪贴板
   - **删除货柜**：暂时禁用（灰色显示）
4. 点击菜单项执行操作
5. 点击菜单外任意位置或点击其他菜单项，菜单自动关闭

### 4.2 技术实现

**组件位置**：`frontend/src/components/common/ContainerContextMenu.vue`

**核心代码**：
```vue
<div class="context-menu" :style="{ left: position.x + 'px', top: position.y + 'px' }">
  <div v-for="item in menuItems" @click="handleItemClick(item)">
    <component :is="item.icon" />
    <span>{{ item.label }}</span>
  </div>
</div>
```

**菜单项配置**：
```typescript
const menuItems = computed<MenuItem[]>(() => [
  { key: 'viewDetail', label: '查看详情', icon: View },
  { key: 'editDate', label: '调整日期', icon: Edit },
  { key: 'copy', label: '复制柜号', icon: DocumentCopy },
  { key: 'delete', label: '删除货柜', icon: Delete, disabled: true },
])
```

**关闭逻辑**：
```typescript
const handleClickOutside = (event: MouseEvent) => {
  if (menuRef.value && !menuRef.value.contains(event.target as Node)) {
    emit('update:visible', false)
  }
}
```

## 五、日期编辑对话框

### 5.1 使用方式

从右键菜单选择"调整日期"后打开对话框，包含以下字段：

1. **集装箱号**：只读显示
2. **日期类型**：下拉选择
   - 预计到港日（etaDestPort）
   - 实际到港日（ataDestPort）
   - 计划提柜日（plannedPickupDate）
   - 最晚提柜日（lastFreeDate）
3. **新日期**：日期时间选择器
4. **调整原因**：必填文本域

点击"保存"提交修改，点击"取消"关闭对话框。

### 5.2 技术实现

**组件位置**：`frontend/src/components/common/ContainerDateEditDialog.vue`

**表单验证**：
```typescript
const handleSave = () => {
  if (!form.value.newDate) {
    ElMessage.warning('请选择新日期')
    return
  }
  if (!form.value.reason.trim()) {
    ElMessage.warning('请输入调整原因')
    return
  }
  // 调用API保存
}
```

**智能填充**：
- 切换日期类型时，自动填充该类型当前值
- 表单初始化时，根据默认类型设置初始值

## 八、后续改进方向

### 8.1 功能增强

1. **后端API集成**
   - 拖拽调整日期：调用`PUT /api/v1/containers/:id/date`更新
   - 对话框编辑日期：调用同一API
   - 添加操作日志记录（谁在何时修改了哪个货柜的日期）

2. **拖拽限制**
   - 只能在同一港口内拖拽
   - 不能拖到过去日期（可选规则）
   - 显示拖拽时的原始日期对比

3. **批量操作**
   - Shift+多选圆点
   - 批量拖拽调整日期
   - 批量编辑日期

### 8.2 交互优化

1. **拖拽反馈增强**
   - 显示拖拽中的圆点跟随鼠标
   - 拖放预览：显示拖放后的圆点位置
   - 拖放动画：平滑过渡到新位置

2. **右键菜单扩展**
   - 添加更多快速操作（标记为异常、添加备注等）
   - 菜单项分组（常用操作、高级操作）
   - 键盘快捷键支持（Delete删除、Ctrl+C复制）

3. **详情侧边栏增强**
   - 显示货柜的物流路径时间线
   - 显示滞港费计算结果
   - 显示关联的其他货柜（同一提单）

### 8.3 性能优化

1. **虚拟滚动**
   - 当港口或日期数量很多时，使用虚拟滚动
   - 只渲染可见区域内的圆点

2. **懒加载**
   - 详情侧边栏按需加载物流路径
   - 拖拽时延迟加载日期格子

---

## 九、已知问题

1. **拖拽边界**：拖拽到屏幕边缘时，菜单可能被遮挡
2. **移动端支持**：当前版本不支持触摸拖拽
3. **日期格式**：不同时区的用户可能看到不同日期
