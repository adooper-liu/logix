# 实体定义

TypeORM 实体与数据库表的映射。

## 命名规则

```
数据库: snake_case  →  实体: camelCase
例: container_number  →  containerNumber
```

## Container 实体

```typescript
// backend/src/entities/Container.ts
@Entity('biz_containers')
export class Container {
  @PrimaryColumn({ name: 'container_number', length: 50 })
  containerNumber: string;

  @Column({ name: 'order_number', length: 50, nullable: true })
  orderNumber: string;

  @Column({ name: 'container_type_code', length: 20, nullable: true })
  containerTypeCode: string;

  @Column({ name: 'logistics_status', length: 20, nullable: true })
  logisticsStatus: string;

  @Column({ name: 'destination_port', length: 50, nullable: true })
  destinationPort: string;

  @Column({ name: 'destination_country', length: 50, nullable: true })
  destinationCountry: string;

  @Column({ name: 'eta_dest_port', type: 'date', nullable: true })
  etaDestPort: Date;

  @Column({ name: 'ata_dest_port', type: 'date', nullable: true })
  ataDestPort: Date;

  @CreateDateColumn({ name: 'created_at' })
  createdAt: Date;

  @UpdateDateColumn({ name: 'updated_at' })
  updatedAt: Date;
}
```

## ReplenishmentOrder 实体

```typescript
// backend/src/entities/ReplenishmentOrder.ts
@Entity('biz_replenishment_orders')
export class ReplenishmentOrder {
  @PrimaryColumn({ name: 'order_number', length: 50 })
  orderNumber: string;

  @Column({ name: 'sell_to_country', length: 50, nullable: true })
  sellToCountry: string;

  @Column({ name: 'customer_name', length: 100, nullable: true })
  customerName: string;

  @Column({ length: 20, nullable: true })
  status: string;

  @Column({ name: 'total_boxes', type: 'decimal', precision: 10, scale: 2, nullable: true })
  totalBoxes: number;

  @Column({ name: 'total_volume', type: 'decimal', precision: 10, scale: 2, nullable: true })
  totalVolume: number;

  @Column({ name: 'total_weight', type: 'decimal', precision: 10, scale: 2, nullable: true })
  totalWeight: number;
}
```

## SeaFreight 实体

```typescript
// backend/src/entities/SeaFreight.ts
@Entity('process_sea_freight')
export class SeaFreight {
  @PrimaryGeneratedColumn()
  id: number;

  @Column({ name: 'container_number', length: 50 })
  containerNumber: string;

  @Column({ name: 'bill_of_lading_number', length: 50, nullable: true })
  billOfLadingNumber: string;

  @Column({ name: 'vessel_name', length: 100, nullable: true })
  vesselName: string;

  @Column({ type: 'date', nullable: true })
  eta: Date;

  @Column({ name: 'shipment_date', type: 'date', nullable: true })
```

```typescript
}
```

## TruckingTransport 实体


## 类型对应

| 数据库类型 | TypeORM 类型 |
|------------|--------------|
| VARCHAR(n) | string (length: n) |
| DECIMAL(p,s) | decimal (precision: p, scale: s) |
| DATE | date |
| TIMESTAMP | timestamp |
| INT | int |
| BOOLEAN | boolean |
| TEXT | text |
| JSONB | jsonb |

## 字段映射原则

1. 实体属性使用 camelCase
2. @Column 使用 name 参数指定数据库字段名
3. 主键使用 @PrimaryColumn 或 @PrimaryGeneratedColumn
4. 时间戳使用 @CreateDateColumn / @UpdateDateColumn
