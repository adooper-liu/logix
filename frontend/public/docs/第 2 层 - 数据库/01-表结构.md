# 表结构

LogiX 数据库表结构速查。

## 核心表

### biz_containers (货柜表)

```sql
CREATE TABLE biz_containers (
  container_number VARCHAR(50) PRIMARY KEY,
  order_number VARCHAR(50),
  container_type_code VARCHAR(20),
  logistics_status VARCHAR(20),
  destination_port VARCHAR(50),
  destination_country VARCHAR(50),
  eta_dest_port DATE,
  ata_dest_port DATE,
  created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
  updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);
```

### biz_replenishment_orders (备货单)

```sql
CREATE TABLE biz_replenishment_orders (
  order_number VARCHAR(50) PRIMARY KEY,
  sell_to_country VARCHAR(50),
  customer_name VARCHAR(100),
  status VARCHAR(20),
  total_boxes DECIMAL(10,2),
  total_volume DECIMAL(10,2),
  total_weight DECIMAL(10,2),
  created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
  updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);
```

### process_sea_freight (海运)

```sql
CREATE TABLE process_sea_freight (
  id SERIAL PRIMARY KEY,
  container_number VARCHAR(50) REFERENCES biz_containers(container_number),
  bill_of_lading_number VARCHAR(50),
  vessel_name VARCHAR(100),
  eta DATE,
  shipment_date DATE,
  created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
  updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);
```

### process_port_operations (港口操作)

```sql
CREATE TABLE process_port_operations (
  id SERIAL PRIMARY KEY,
  container_number VARCHAR(50) REFERENCES biz_containers(container_number),
  port_type VARCHAR(20),        -- origin/transit/destination
  port_sequence INT,
  port_code VARCHAR(20),
  eta_dest_port DATE,
  ata_dest_port DATE,
  last_free_date DATE,
  created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
  updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);
```

### process_trucking_transport (拖卡运输)

```sql
CREATE TABLE process_trucking_transport (
  id SERIAL PRIMARY KEY,
  container_number VARCHAR(50) REFERENCES biz_containers(container_number),
  transport_type VARCHAR(20),
  pickup_date DATE,
  delivery_date DATE,
  planned_pickup_date DATE,
  created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
  updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);
```

### process_warehouse_operations (仓库操作)

```sql
CREATE TABLE process_warehouse_operations (
  id SERIAL PRIMARY KEY,
  container_number VARCHAR(50) REFERENCES biz_containers(container_number),
  warehouse_code VARCHAR(20),
  unload_date DATE,
  return_date DATE,
  created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
  updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);
```

### process_empty_return (还空箱)

```sql
CREATE TABLE process_empty_return (
  id SERIAL PRIMARY KEY,
  container_number VARCHAR(50) REFERENCES biz_containers(container_number),
  return_date DATE,
  return_terminal VARCHAR(50),
  created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
  updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);
```

## 索引

### 常用索引

```sql
-- 货柜状态索引
CREATE INDEX idx_containers_status ON biz_containers(logistics_status);

-- 货柜目的港索引
CREATE INDEX idx_containers_dest_port ON biz_containers(destination_port);

-- 货柜备货单索引
CREATE INDEX idx_containers_order ON biz_containers(order_number);

-- 港口操作货柜索引
CREATE INDEX idx_port_ops_container ON process_port_operations(container_number);

-- 港口操作类型索引
CREATE INDEX idx_port_ops_type ON process_port_operations(port_type);

-- 港口操作到港日期索引
CREATE INDEX idx_port_ops_ata ON process_port_operations(ata_dest_port);
```

## 时间字段

| 字段 | 类型 | 说明 |
|------|------|------|
| created_at | TIMESTAMP | 创建时间 |
| updated_at | TIMESTAMP | 更新时间 |
| eta_dest_port | DATE | 预计到港 |
| ata_dest_port | DATE | 实际到港 |
| last_free_date | DATE | 最后免费日 |
