### 计算逻辑要点速查

#### 滞港费计算

```
费用 = chargeDays × ratePerDay
chargeDays = endDate - lastFreeDate - 1
lastFreeDate = startDate + freeDays - 1
```

#### 五个计划日期

```
         ←─────────── 向前推导 ───────────→        ←── 向后推导 ──→

   计划清关日  →  计划提柜日  →  计划送仓日  →  计划卸柜日  →  计划还箱日
       ↑             ↑               ↑              ↑              ↑
      ETA-1      customs+1      提+运输天       锚点           卸+在仓天
```

#### 历时/超期/倒计时

| 概念 | 方向 | 颜色 |
|------|------|------|
| 历时 | 向后看（过去→过去） | 蓝色 |
| 超期 | 向前看（当前←过去） | 红色 |
| 倒计时 | 向前看（未来→当前） | 橙色/绿色 |
