# LogiX 代码解读文档

**项目**: LogiX 物流管理系统  
**更新时间**: 2026-03-28

---

## 一、项目概述

LogiX 是一个物流管理系统，主要功能包括：
- 货柜管理（Shipments）
- 智能排柜（Intelligent Scheduling）
- 滞港费计算（Demurrage）
- 物流跟踪（Logistics Path）
- 数据导入（Import/Export）

---

## 二、技术架构

| 层级 | 技术栈 |
|------|--------|
| **前端** | Vue 3 + TypeScript + Vite + Pinia |
| **后端** | Node.js + TypeORM + Express |
| **数据库** | PostgreSQL + TimescaleDB |
| **外部API** | 飞驼物流（Feituo） |
