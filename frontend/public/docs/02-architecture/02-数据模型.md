# 数据模型

LogiX 7层流转数据模型。

## 表关系

```
备货单 ──┬──→ 货柜 ──┬──→ 海运
         │           │
         │           ├──→ 港口操作
         │           │
         │           ├──→ 拖卡运输
         │           │
         │           ├──→ 仓库操作
         │           │
         │           └──→ 还空箱
         │
         └──→ 客户
```

## 核心表

### biz_replenishment_orders (备货单)

主键: order_number

| 字段 | 类型 | 说明 |
|------|------|------|
| order_number | VARCHAR(50) | 备货单号（主键） |
| sell_to_country | VARCHAR(50) | 销往国家 |
| customer_name | VARCHAR(100) | 客户名称 |
| status | VARCHAR(20) | 备货单状态 |
| total_boxes | DECIMAL(10,2) | 箱数合计 |
| total_volume | DECIMAL(10,2) | 体积合计 |
| total_weight | DECIMAL(10,2) | 毛重合计 |

### biz_containers (货柜)

主键: container_number

| 字段 | 类型 | 说明 |
|------|------|------|
| container_number | VARCHAR(50) | 集装箱号（主键） |
| order_number | VARCHAR(50) | 备货单号（外键） |
| container_type_code | VARCHAR(20) | 柜型 |
| logistics_status | VARCHAR(20) | 物流状态 |
| destination_port | VARCHAR(50) | 目的港 |
| destination_country | VARCHAR(50) | 目的国 |

### process_sea_freight (海运)

主键: id (复合主键: container_number)

| 字段 | 类型 | 说明 |
|------|------|------|
| container_number | VARCHAR(50) | 货柜号（主键） |
| bill_of_lading_number | VARCHAR(50) | 提单号 |
| vessel_name | VARCHAR(100) | 船名 |
| eta | DATE | 预计到港日 |
| shipment_date | DATE | 开船日 |

### process_port_operations (港口操作)

主键: id

| 字段 | 类型 | 说明 |
|------|------|------|
| container_number | VARCHAR(50) | 货柜号 |
| port_type | VARCHAR(20) | 港口类型 |
| port_sequence | INT | 港口顺序 |
| port_code | VARCHAR(20) | 港口代码 |
| eta_dest_port | DATE | 预计到港日 |
| ata_dest_port | DATE | 实际到港日 |
| last_free_date | DATE | 最后免费日 |

### process_trucking_transport (拖卡运输)

主键: id

| 字段 | 类型 | 说明 |
|------|------|------|
| container_number | VARCHAR(50) | 货柜号 |
| transport_type | VARCHAR(20) | 运输类型 |
| pickup_date | DATE | 提柜日 |
| delivery_date | DATE | 送仓日 |
| planned_pickup_date | DATE | 计划提柜日 |

### process_warehouse_operations (仓库操作)

主键: id

| 字段 | 类型 | 说明 |
|------|------|------|
| container_number | VARCHAR(50) | 货柜号 |
| warehouse_code | VARCHAR(20) | 仓库编码 |
| unload_date | DATE | 卸柜日 |
| return_date | DATE | 还箱日 |

### process_empty_return (还空箱)

主键: id

| 字段 | 类型 | 说明 |
|------|------|------|
| container_number | VARCHAR(50) | 货柜号 |
| return_date | DATE | 还箱日期 |
| return_terminal | VARCHAR(50) | 还箱堆场 |

## 字典表

```
dict_countries         # 国家
dict_ports             # 港口
dict_container_types   # 柜型
dict_shipping_companies # 船公司
dict_freight_forwarders # 货代公司
dict_customs_brokers   # 清关公司
dict_trucking_companies # 拖车公司
dict_warehouses        # 仓库
```

## 物流状态

```
not_shipped    # 未出运
shipped        # 已出运
in_transit     # 在途
at_port        # 已到港
picked_up      # 已提柜
unloaded       # 已卸柜
returned_empty # 已还箱
```

## 端口类型

```
origin       # 起运港
transit      # 中转港
destination  # 目的港
```
