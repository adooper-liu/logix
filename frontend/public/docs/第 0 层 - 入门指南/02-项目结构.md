# 项目结构

LogiX 项目目录结构及代码组织。

## 整体结构

```
logix/
├── backend/                 # 后端服务
│   ├── src/
│   │   ├── adapters/       # 外部数据适配器（飞驼）
│   │   ├── controllers/    # API 控制器
│   │   ├── services/       # 业务逻辑层
│   │   ├── entities/       # 数据库实体
│   │   ├── routes/         # 路由定义
│   │   ├── middleware/      # 中间件
│   │   ├── constants/      # 常量定义
│   │   └── scripts/        # 工具脚本
│   └── sql/                # SQL 脚本
├── frontend/               # 前端应用
│   ├── src/
│   │   ├── views/          # 页面组件
│   │   ├── components/     # 通用组件
│   │   ├── services/       # API 服务
│   │   ├── stores/         # Pinia 状态管理
│   │   ├── composables/    # 组合式函数
│   │   └── types/          # TypeScript 类型
│   └── public/docs/        # 文档
└── docs/                   # 项目文档
```

## 数据库表分类

### 表前缀含义

| 前缀 | 含义 | 示例 |
|------|------|------|
| dict_ | 字典表 | ports, countries, container_types |
| biz_ | 业务表 | containers, replenishment_orders |
| process_ | 流程表 | sea_freight, port_operations |
| ext_ | 扩展表 | status_events, loading_records |
| sys_ | 系统表 | data_change_logs |
| hist_ | 历史表 | scheduling_records |
| flow_ | 流程定义表 | definitions, instances |

### 7层流转架构

```
备货单 → 货柜 → 海运 → 港口 → 拖卡 → 仓库 → 还空箱
```
