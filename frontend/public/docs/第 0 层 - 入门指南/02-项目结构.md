# 项目结构

LogiX 项目目录结构及代码组织。

## 整体结构

```
logix/
├── backend/                 # 后端服务
│   ├── src/
│   │   ├── adapters/       # 外部数据适配器（飞驼）
│   │   ├── controllers/    # API 控制器
│   │   ├── services/       # 业务逻辑层
│   │   ├── entities/       # 数据库实体
│   │   ├── routes/         # 路由定义
│   │   ├── middleware/      # 中间件
│   │   ├── constants/      # 常量定义
│   │   ├── utils/          # 工具函数
│   │   ├── schedulers/      # 定时任务
│   │   └── database/       # 数据库配置
│   ├── migrations/         # 数据库迁移脚本
│   └── package.json
│
├── frontend/                # 前端应用
│   ├── src/
│   │   ├── components/     # 通用组件
│   │   ├── views/          # 页面组件
│   │   ├── composables/     # 组合式函数
│   │   ├── services/       # API 服务
│   │   ├── store/          # 状态管理
│   │   ├── router/         # 路由配置
│   │   └── assets/         # 静态资源
│   ├── public/
│   │   └── docs/           # 文档
│   └── package.json
│
├── migrations/             # 数据库迁移脚本（根目录）
├── scripts/               # 辅助脚本
├── docs/                  # 项目文档（归档）
└── docker-compose.*.yml   # Docker 配置
```

## 后端代码组织

### 实体 (entities/)

| 实体 | 数据库表 | 说明 |
|------|----------|------|
| Container | biz_containers | 货柜主表 |
| ReplenishmentOrder | biz_replenishment_orders | 备货单 |
| SeaFreight | process_sea_freight | 海运信息 |
| PortOperation | process_port_operations | 港口操作 |
| TruckingTransport | process_trucking_transport | 拖卡运输 |
| WarehouseOperation | process_warehouse_operations | 仓库操作 |
| EmptyReturn | process_empty_return | 还空箱 |

### 服务 (services/)

| 服务 | 职责 |
|------|------|
| container.service.ts | 货柜 CRUD、列表查询 |
| containerStatistics.service.ts | 统计服务（到港/ETA/计划提柜） |
| demurrage.service.ts | 滞港费计算 |
| intelligentScheduling.service.ts | 智能排柜 |
| externalDataService.ts | 飞驼数据同步 |

### 控制器 (controllers/)

| 控制器 | 路由前缀 |
|--------|----------|
| container.controller.ts | /api/v1/containers |
| replenishment-order.controller.ts | /api/v1/replenishment-orders |
| dict-manage.controller.ts | /api/v1/dict-manage |
| statistics.controller.ts | /api/v1/statistics |

## 前端代码组织

### 页面 (views/)

```
views/

│   ├── Shipments.vue     # 主页面

```

### 组件 (components/)

```
components/

```

## 数据库表分类

### 表前缀含义

```
dict_;      # 字典表：ports, countries, container_types
biz_;       # 业务表：containers, replenishment_orders
process_;   # 流程表：sea_freight, port_operations
ext_;       # 扩展表：status_events, loading_records
```

### 7层流转架构

```
备货单 → 货柜 → 海运 → 港口 → 拖卡 → 仓库 → 还空箱
```
