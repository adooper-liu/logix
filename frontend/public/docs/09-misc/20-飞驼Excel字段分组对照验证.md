# 飞驼 Excel 字段分组对照验证

> 以飞驼 Excel 实际导出结构为准，与 LogiX `FeituoFieldGroupMapping` 对齐。**未知分组字段不导入**。

---

## 一、分组对应关系（表一）

| Excel 分组          | 组号 | 说明                                                         |
| ------------------- | ---- | ------------------------------------------------------------ |
| 未知                | 0    | **不导入**（编码、名称、创建人、修改人、创建/修改时间等）    |
| 基本信息            | 1    | MBL、订单状态、更新时间、是否3PL、是否海铁联运、首次获取时间 |
| 船公司信息          | 2    | 船公司代码、SCAC、网站url等                                  |
| 订舱信息            | 3    | 订舱状态（暂不落库）                                         |
| 接货地信息          | 4    | 起运港                                                       |
| 交货地信息          | 5    | 目的港                                                       |
| 头程船信息          | 6    | 船名、航次、航线                                             |
| 当前状态信息        | 7    | **不导入**（与十二组重复，仅十二组落库）                     |
| 发生地信息          | 8    | 实际装船/卸船、时间节点、免堆存等                            |
| 路径信息            | 9    | 起始地、目的地、运输方式                                     |
| 港区船舶计划        | 10   | 港口代码、开截港、船代理                                     |
| 集装箱物流信息      | 11   | 箱型、当前状态描述                                           |
| 集装箱物流信息-状态 | 12   | 状态事件序列                                                 |
| 船舶信息            | 13   | imo、mmsi、船籍                                              |
| 港区货运单证-装箱单 | 14   | 箱皮重、总重等                                               |
| 港区货运单-VGM      | 15   | VGM（暂不落库）                                              |

## 二、未知分组（不导入）

以下字段**未**在 `FeituoFieldGroupMapping` 中配置，导入时归 group 0，**不写入 raw_data、不落库**：

| 字段名                                             |
| -------------------------------------------------- |
| 编码                                               |
| 名称.简体中文、名称.English、名称.Français         |
| 数据状态                                           |
| 创建人.工号、创建人.姓名、修改人.工号、修改人.姓名 |
| 使用状态                                           |
| 创建时间、修改时间                                 |

## 四、影响 ETA 的字段

以下字段会写入 `process_sea_freight.eta`、`process_port_operations.etaDestPort`（目的港预计到港）：

| 飞驼字段 | 分组 | 说明 |
|----------|------|------|
| 交货地预计到达时间 | 5 | 目的港预计到港 |
| 预计到达时间 | 5（交货地） | 无前缀时，第 2 次出现为交货地 |
| 预计到港日期 | 5 | 别名 |
| 目的港到达日期 | 5 | 别名 |

**注意**：仅 **group 5（交货地）** 的预计到达时间会写入目的港 ETA；接货地(4)、港区船舶计划(10)、发生地(8) 的预计到达时间不会影响目的港 ETA。
