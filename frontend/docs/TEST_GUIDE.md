## 🐛 常见问题排查

### 问题 1: 跳转后新窗口打开

**原因**: 代码未正确修改  
**检查**: 
```typescript
// ❌ 错误
window.open(url.href, '_blank')

// ✅ 正确
router.push({ path: '/gantt-chart', query })
```

---

### 问题 2: 返回后状态丢失

**原因**: Pinia Store 未正确配置  
**检查**:
1. `ganttFilters.ts` 是否正确引入
2. `setFilters()` 是否调用
3. localStorage 是否有数据

---

### 问题 3: Keep-Alive 未生效

**原因**: 组件 name 未设置  
**检查**:
```vue
<script lang="ts">
export default {
  name: 'Shipments'  // 或 'GanttChart'
}
</script>
```
