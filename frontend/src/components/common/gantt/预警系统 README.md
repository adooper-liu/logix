# 甘特图智能预警系统

## 📖 概述

为 LogiX 物流管理系统的简单甘特图组件添加了智能预警功能，帮助用户及时识别和处理有风险的货柜。

---

## ✨ 核心功能

### 1. 自动风险识别

系统会自动扫描所有货柜，识别两类风险：

| 风险类型 | 标识 | 说明                       |
| -------- | ---- | -------------------------- |
| 即将超期 | 🟠   | 距离最晚提柜日期 ≤ 3 天    |
| 逾期未提 | 🔴   | 已超过最晚提柜日期且未处理 |

### 2. 多维度视觉呈现

- **甘特图**：彩色边框 + 脉冲动画
- **统计面板**：独立卡片展示数量
- **Tooltip**：详细信息 + 预警列表

### 3. 可扩展规则引擎

采用配置化的预警规则设计，支持快速添加新的预警类型。

## 🎯 技术亮点

### 1. 预警规则引擎

```typescript
interface AlertRule {
  id: string // 规则唯一标识
  name: string // 规则名称
  condition: (container: Container) => boolean // 判断条件
  level: 'info' | 'warning' | 'danger' // 预警级别
  message: string // 预警消息
}
```

### 2. 智能样式生成

根据预警级别自动生成对应的视觉样式：

```typescript
getContainerBorderStyle(container)
// → 'none' | '2px solid #e6a23c' | '3px solid #f56c6c'
```

### 3. 性能优化

- 使用 `computed` 缓存计算结果
- 避免重复计算和渲染
- CSS 动画使用 GPU 加速

## 🔧 使用方法

### 基础使用

预警系统已集成到甘特图中，**无需额外配置**即可使用：

```vue
<template>
  <SimpleGanttChartRefactored />
</template>
```

### 查看预警信息

1. **甘特图**：查找带彩色边框的圆点
2. **统计面板**：查看"即将超期"和"逾期未提"卡片
3. **Tooltip**：鼠标悬停查看详情

### 处理预警货柜

1. 点击预警货柜圆点
2. 查看详细信息
3. 右键菜单编辑日期或执行其他操作

## 🛠️ 扩展开发

### 添加新的预警规则

在 `useGanttLogic.ts` 中的 `alertRules` 数组添加新规则：

```typescript
const alertRules: AlertRule[] = [
  // 现有规则...

  // 新增规则示例：滞期费预估
  {
    id: 'high_demurrage',
    name: '高额滞期费',
    condition: container => {
      const estimatedDemurrage = calculateDemurrage(container)
      return estimatedDemurrage > 1000
    },
    level: 'danger',
    message: '预估滞期费超过 $1000',
  },
]
```

### 自定义预警阈值

修改 `useGanttLogic.ts` 第 97 行：

```typescript
// 将 3 天改为 5 天
return daysUntilDeadline >= 0 && daysUntilDeadline <= 5
```

## 📊 数据流

```
容器数据加载
    ↓
预警规则引擎
    ↓
遍历所有容器 → 匹配预警规则
    ↓
生成预警列表
    ↓
计算预警级别
    ↓
生成样式配置
    ↓
渲染到界面
    ├─→ 甘特图（边框 + 动画）
    ├─→ Tooltip（详细信息）
    └─→ 统计面板（数量汇总）
```
