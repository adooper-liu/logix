# 路径追踪功能优化总结

## 优化日期
2026-04-04

## 优化内容

### 1. 样式参数优化

#### 颜色调整
- **连线颜色**: `#409eff` (蓝色) -> `#67c23a` (绿色)
  - 理由: 与预警正常状态保持一致，视觉更统一
- **节点颜色**: `#409eff` (蓝色) -> `#67c23a` (绿色)
  - 理由: 与连线颜色一致，形成完整的视觉体系

#### 尺寸调整
- **连线宽度**: 2px -> 2.5px
  - 理由: 稍微加粗，在复杂背景下更清晰可见
- **节点半径**: 4px -> 5px
  - 理由: 增大节点标记，更容易点击和识别
- **节点边框宽度**: 2px -> 2.5px
  - 理由: 与连线宽度保持一致

#### 透明度调整
- **连线透明度**: 0.6 -> 0.7
  - 理由: 提高可见度，同时保持半透明效果
- **节点透明度**: 0.8 -> 0.9
  - 理由: 节点更加突出，悬停时可达1.0

#### 虚线样式调整
- **虚线模式**: `5,5` -> `6,4`
  - 理由: 调整间隔比例，视觉效果更美观

#### 层级调整
- **z-index**: 5 -> 10
  - 理由: 确保路径连线在其他UI元素之上显示

### 2. 动画效果增强

#### 连线绘制动画
```css
.path-line-animated {
  stroke-dasharray: 1000;
  stroke-dashoffset: 1000;
  animation: drawLine 0.8s ease-out forwards;
}
```
- **效果**: 路径显示时从起点到终点逐渐绘制
- **时长**: 0.8秒
- **缓动**: ease-out (先快后慢)
- **用户体验**: 给用户明确的视觉反馈，路径正在建立

#### 节点脉冲动画
```css
.node-pulse {
  animation: pulse 2s ease-in-out infinite;
}

@keyframes pulse {
  0%, 100% { opacity: 0.9; r: 5; }
  50% { opacity: 0.6; r: 7; }
}
```
- **效果**: 节点像呼吸一样周期性缩放和改变透明度
- **周期**: 2秒
- **缓动**: ease-in-out (平滑过渡)
- **用户体验**: 吸引用户注意，提示这些是可交互的节点

#### 箭头淡入动画
```css
.arrow-fade-in {
  animation: fadeIn 0.5s ease-out;
}
```
- **效果**: 高亮圆环和标签淡入显示
- **时长**: 0.5秒
- **应用场景**: 鼠标悬停时的反馈元素

### 3. 交互功能增强

#### 鼠标悬停交互
**实现方式**:
- 监听节点的 `mouseenter` 和 `mouseleave` 事件
- 维护 `hoveredNodeIndex` 状态
- 悬停时显示:
  1. 高亮圆环 (半径 + 4px, 半透明)
  2. 节点信息标签 (节点名称 + 日期)
  3. 节点完全不透明 (opacity: 1.0)

**触发事件**:
```typescript
emit('nodeHover', node) // 悬停时
emit('nodeHover', null) // 离开时
```

**用户体验**:
- 即时反馈，用户知道当前悬停的是哪个节点
- 显示详细信息，无需额外操作
- 视觉上突出当前关注的节点

#### 节点点击交互
**实现方式**:
- 监听节点的 `click` 事件
- 触发 `nodeClick` 事件，传递节点数据

**触发事件**:
```typescript
emit('nodeClick', node)
```

**扩展性**:
- 父组件可以自定义点击行为
- 例如: 显示节点详情、跳转到相关页面、高亮相关货柜等

#### 节点信息标签
**显示内容**:
- 节点名称 (清关/提柜/卸柜/还箱)
- 节点日期 (如果有)

**样式**:
- 字体大小: 12px
- 颜色: #303133 (深灰色)
- 字重: bold
- 位置: 节点右上方 (x + 10, y - 10)

### 4. 代码结构优化

#### 新增状态管理
```typescript
const hoveredNodeIndex = ref<number | null>(null)
```
- 跟踪当前悬停的节点索引
- null 表示没有节点被悬停

#### 新增事件处理函数
```typescript

const handleNodeClick = (index: number) => { ... }
const formatDate = (date: Date): string => { ... }
```

#### 模板结构优化
使用 `<g>` 元素分组节点相关的所有SVG元素:

## 性能考虑

### 动画性能
- 使用CSS动画而非JavaScript动画
- 利用GPU加速 (transform 和 opacity)
- 动画时长适中，不会造成视觉疲劳

### 交互性能
- 事件委托，避免大量事件监听器
- 使用 `ref` 响应式更新，Vue自动优化
- 悬停状态只更新必要的DOM元素

### 渲染性能
- SVG原生支持，浏览器优化良好
- 只在需要时显示标签和高亮
- z-index 提升避免重绘其他元素
