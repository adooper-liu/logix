## 发现的问题

### 问题1: "基础信息"分组未定义 (严重)

**问题描述**: 
Excel文件中有3个字段使用"基础信息"作为分组前缀，但`FeituoFieldGroupMapping.ts`中的`GROUP_NAME_TO_ID`未包含此分组。

**受影响的字段**:
- `基础信息_是否海铁联运`
- `基础信息_首次获取时间`
- `基础信息_是否3PL`

**影响**: 
这3个字段会被归到group 0，在`buildRawDataByGroup`函数中会被跳过（第95行：`if (gid === 0 || gid === 7) continue`），**不会导入到数据库**。

**建议修复**:
在`FeituoFieldGroupMapping.ts`的`GROUP_NAME_TO_ID`中添加：
```typescript
'基础信息': 1,  // 与"基本信息"相同，分组ID=1
```

### 问题2: 铅封号字段全为空 (轻微)

**问题描述**: 
`集装箱物流信息_铅封号`字段在所有49行数据中均为空。

**影响**: 
不影响导入，但货柜的`seal_number`字段将无法从飞驼数据填充。

**建议**: 
这是数据源问题，非映射问题。
