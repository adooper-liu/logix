## 经验总结

### 成功经验

1. **分阶段执行**: 将大任务拆分为多个小阶段，每阶段独立验证
2. **脚本自动化**: 使用 PowerShell 脚本批量操作，减少人工错误
3. **ARCHIVE 标记**: 归档文件添加 `_ARCHIVE_` 标记，便于识别
4. **索引先行**: 先整理目录结构，再更新索引，最后验证链接

### 踩坑记录

1. **中文路径问题**: Windows 下读取中文路径文件需使用绝对路径
2. **重复文档识别**: 部分文档在多个目录重复存在，需手动确认唯一版本
3. **链接失效**: 旧文档使用编号引用，整理后编号规则变化导致链接失效
