# 🤖 AI 自动 Skill 调用 - 快速参考卡

## 📌 核心原则

**你不需要记住任何 Skill 名称！**

只需要用自然语言描述问题，AI 会自动识别并调用相应的 Skill。

---

## 🎯 你只需要说

| 你说什么 | AI 自动做什么 |
|---------|-------------|
| "这个字段名是什么？" | 🔍 自动验证数据库字段 |
| "表结构是怎样的？" | 🔍 自动查询数据库 |
| "前端代码怎么改？" | 💻 自动检查 Vue 组件 |
| "后端 API 怎么写？" | ⚙️ 自动检查后端代码 |
| "这样对吗？" | ✅ 自动验证 |
| "帮我检查一下" | ✅ 自动验证 |
| "保存失败" | 🚨 自动验证 + 排查 |

---

## 🚨 监督 AI

**如果 AI 没有自动调用 Skill**，你可以：

1. **提醒**："请自动调用相应的 Skill"
2. **检查**：AI 是否读取了真实代码？
3. **验证**：AI 是否基于权威源？

---

## ✅ 正确示范

**你**："unload_mode_plan 这个字段对吗？"

**AI 应该**：
1. [自动调用 fix-verification skill]
2. [自动调用 database-query skill]
3. [读取 03_create_tables.sql 验证]
4. [读取 TruckingTransport.ts 验证]
5. 输出验证结果

**❌ 错误**：只说"会调用"但没有实际调用
