# 甘特图按钮添加记录

## 修改时间

2026-04-02

## 修改内容

### 文件

`frontend/src/views/scheduling/SchedulingVisual.vue`

### 变更

在顶部操作区（第 84-93 行）添加了"甘特图"按钮。

```vue
<!-- ✅ 新增：甘特图按钮 -->
<el-button type="success" size="default" @click="router.push('/gantt-chart')" title="查看甘特图">
  <el-icon><View /></el-icon>
  甘特图
</el-button>
```

### 位置

按钮位于顶部操作区的最右侧，在"批量保存优化"按钮之前。

### 功能

点击后跳转到甘特图页面（路由：`/gantt-chart`）

### 技术细节

- 使用 `type="success"` 绿色主题
- 使用 `View` 图标（已在第 931 行导入）
- 使用 `router.push()` 进行路由跳转
- 添加 `title` 属性提供悬停提示

## 现有甘特图按钮对比

页面中已有两个甘特图按钮：

1. **顶部操作区按钮**（新增，第 84-93 行）
   - 位置：顶部操作区
   - 样式：success 主题，默认大小
   - 始终可见

2. **排产结果卡片头部按钮**（既有，第 297-305 行）
   - 位置：排产结果卡片头部右侧
   - 样式：success 主题，plain 样式，小大小
   - 仅在正式模式（非预览模式）下显示

## 路由配置

甘特图路由已在 `frontend/src/router/index.ts` 第 89-95 行配置：

```typescript
{
  path: 'gantt-chart',
  name: 'GanttChart',
  component: () => import('@/components/common/SimpleGanttChartRefactored.vue'),
  meta: {
    title: '货柜甘特图',
    icon: 'Calendar',
    requiresAuth: true,
  },
}
```
