### 2. 核心逻辑缺失

#### 问题1: 没有火车/无火车的判断
**需求**: 几乎所有关键字段都需要判断目的港是否有火车
**当前**: 无任何判断逻辑

**解决方案**:
```typescript
// 新增判断方法
private async hasTrainService(portCode: string): Promise<boolean> {
  // 判断逻辑：
  // 1. 检查港口代码是否在欧洲内陆点列表
  // 2. 或检查历史数据中是否有火车状态码
  // 3. 或配置表维护
  const trainPorts = ['DEHAM', 'NLRTM', 'BEANR']; // 示例
  return trainPorts.includes(portCode);
}
```

---

#### 问题2: 没有状态事件优先级处理
**需求**: BDAR/IRAR/FETA取最大值
**当前**: 只取最后一个（覆盖）

**解决方案**:
```typescript
// 新增优先级处理方法
private getMaxDateByStatusCodes(
  events: ContainerStatusEvent[],
  statusCodes: string[],
  locationCode: string
): Date | null {
  const matchingEvents = events.filter(e => 
    statusCodes.includes(e.statusCode) && 
    e.location === locationCode &&
    e.isEstimated === false  // 只取实际值
  );
  
  if (matchingEvents.length === 0) return null;
  
  return new Date(Math.max(...
    matchingEvents.map(e => e.occurredAt.getTime())
  ));
}
```

---

#### 问题3: 没有LFD验证逻辑
**需求**: LFD < ATA时不更新
**当前**: 直接保存，无验证

**业务影响**: 可能导致负免费期（货柜未到达就已过期）

**解决方案**:
```typescript
// 在mergeTable1ToCore中
const lastFreeDate = parseDate(getVal(row, '免费提箱截止日'));
const ataDestPort = destPo.ataDestPort;

if (lastFreeDate && ataDestPort && lastFreeDate >= ataDestPort) {
  destPo.lastFreeDate = lastFreeDate;
} else if (lastFreeDate && ataDestPort && lastFreeDate < ataDestPort) {
  logger.warn(`[FeituoImport] LFD (${lastFreeDate}) < ATA (${ataDestPort}), skipping update`);
  // 不更新LFD
}
```
