## 一、统计逻辑（后端）

### 1. 统计服务

### 2. 统计维度（5 个子维度）

| 维度           | 字段名             | 统计条件                                | 后端方法                     |
| -------------- | ------------------ | --------------------------------------- | ---------------------------- |
| 已逾期         | `expired`          | `last_return_date < today`              | `getReturnExpiredCount()`    |
| 紧急           | `urgent`           | `today <= last_return_date <= today+3`  | `getReturnUrgentCount()`     |
| 警告           | `warning`          | `today+3 < last_return_date <= today+7` | `getReturnWarningCount()`    |
| 正常           | `normal`           | `last_return_date > today+7`            | `getReturnNormalCount()`     |
| 最后还箱日为空 | `noLastReturnDate` | `last_return_date IS NULL`              | `getNoLastReturnDateCount()` |

### 3. 目标集定义（统计范围）

```typescript
// 条件 1：有拖车记录且已提柜
EXISTS (
  SELECT 1
  FROM process_trucking_transport tt
  WHERE tt.container_number = container.container_number
  AND tt.pickup_date IS NOT NULL
)

// 条件 2：未还箱
NOT EXISTS (
  SELECT 1
  FROM process_empty_return er2
  WHERE er2.container_number = container.container_number
  AND er2.return_time IS NOT NULL
)
```

**统计范围**：已有实际提柜记录且未还箱的货柜
